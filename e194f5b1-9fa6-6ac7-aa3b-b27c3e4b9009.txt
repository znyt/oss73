        &lt;h3&gt;第三卷第二百二十五章  保镖守中&lt;/h3&gt;
<br/>
<br/>        陈子昂上班后就投入到了紧张的工作中，公司走到这一步实属不易，现在不是说只要她陈子昂能吃饱穿暖就好，而是全国上上下下几十个分公司的全体员工能不能吃饱穿暖的问题。
<br/>
<br/>        陈子昂奋斗了这么多年，除了北京的房产和厂房及西城的一套房子是她的，其余也没有什么是她的，她已经将她手中的股份分出去了，留在她手中的股份少得可怜。
<br/>
<br/>        她任重道远，还有很多事情都没有干，她无暇顾及身边的事情，一头扎进工作中。
<br/>
<br/>        潘禺是她的外甥，从小一起长大，她就将潘禺带回了家，俩个孩子倒是对潘禺喜欢的紧，天天的黏着潘禺，她很好笑潘禺能和俩个小孩子玩到一起，真的是玩心不改，还是一点都没有变。
<br/>
<br/>        北京的天气有点阴沉沉的，那种雾蒙蒙让人有一种悲凉的感觉，冷风一吹，呼啦啦的掀起街上行道树上的黄叶纷纷起舞，街面上卷起了一阵龙卷风，将一颗大树枝“咔嚓”一声折成两半，重重的砸在陈子昂前面行驶的车上。
<br/>
<br/>        陈子昂紧急刹车，车头稍微斜了一斜，无奈的停下来，紧接着后面是不停歇的刺耳的刹车声。
<br/>
<br/>        陈子昂被旁边广告牌上的大屏播放的新闻吸引住了，她下车跑到大屏前，仔细的看着上面的字幕，嘴上说着：“怎么可能怎么可能！”
<br/>
<br/>        就那样站在寒风中，任泪水成冰，这是她这么多年来第一次流泪，在最困难的时候她都不曾滴落一滴泪水，在张函离开之后，在陆宇霆分手之后，在给江俞轩送礼物之后，可是今天她好想放声大哭，哭她的命运，哭落雪的遭遇。
<br/>
<br/>        新闻已经过了，路障已经清理了，可是陈子昂还站在大屏幕前泪眼婆娑，直到交警过来：“女士，你好，那个是你的车吗？”
<br/>
<br/>        交警连问了三遍，虽然眼前的女人泪流满面，但是车阻挡了大家正常的交通，悲伤是可以，但是你不能影响别人。
<br/>
<br/>        “啊，是，是我的，对不起。”陈子昂回过神来，才发现前面的车已经被托走，她的车停在路中央。
<br/>
<br/>        陈子昂擦了擦眼泪，上车，交警礼貌的问：“你要不要我帮忙？”
<br/>
<br/>        陈子昂摇摇头，微微一笑：“谢谢，我能行！”
<br/>
<br/>        她已经没有心情回公司了，落雪在美洲遭遇袭击，现场一片惨烈，她开车回家带着俩个孩子直奔落家，她要去看看落叔叔。
<br/>
<br/>        陈虞和落妍因为放了寒假，一直都乖乖的呆在家里，妈妈决定要送他们出国，那他们也只能等着被妈妈送走了。
<br/>
<br/>        这几天潘禺哥哥在给他们联系保镖的事情。
<br/>
<br/>        潘禺哥哥答应陈虞，给陈虞找一个很厉害的保镖，既能保护陈虞和落妍，还能教他们一些东西。
<br/>
<br/>        潘禺哥哥说：“这个世界上只有自己才能保护自己，别人都是辅佐的，因为你自己对危险的感知比任何人都强，在你感知到有危险的时候，你要有能力来应对一切，而不是被动的等人救援。”
<br/>
<br/>        陈虞和落妍虽然还不到十岁，但是他们太早熟了，身高和智商都已经达到了成人的要求，而智商有可能比有些成人都还高，他们能明白潘禺说的话。
<br/>
<br/>        自从新年爸爸一家来了之后，他们就一直处于一种惶恐的状态，他们害怕分离，害怕和妈妈分开。他们兄妹俩经常偷偷的唱着“世上只有妈妈好。”唱着唱着俩个人就会相对哽咽，又相互安慰。
<br/>
<br/>        爸爸在很小的时候就会隔三差五的来陪着他们，虽然他们也很快乐，但是总是偷偷摸摸的，和妈妈在一起就不一样了，妈妈就是他们的爱他们的精神支柱，妈妈在哪里他们的安全感就在哪里，没有爸爸他们会难过一阵子，没有妈妈他们该怎么办？
<br/>
<br/>        曾经他们多么希望他们的爸爸和妈妈能够在一起，曾经他们偷偷摸摸的帮助爸爸和妈妈见面，可是当爸爸光明正大的出现在他们和妈妈的面前时，却是他们要和妈妈分别之时，早知道这样，哪怕爸爸一辈子不露面都好啊。
<br/>
<br/>        潘禺哥哥今天带回来了一个帅气的大哥哥。
<br/>
<br/>        潘禺哥哥说：“陈虞，落妍，这个是你们的守中哥哥，也是你们以后的师傅，你们要跟着守中师傅努力的强身健体，以后就能保护妈妈了。”
<br/>
<br/>        陈虞和落妍的体能不错，很小就被陈子昂带着每天五公里的跑步，而且他们在体育上年年都是拿第一的人，酷爱运动，学习好。
<br/>
<br/>        守中看着俩个小朋友，如果不说他们的年龄，他会当他们俩个都是少年了，俩个人长着一样水灵灵的丹凤眼，明眸皓齿，顾盼生辉，落妍有一点点的婴儿肥，那胖乎乎的瓜子脸上一笑就露出俩个甜甜的酒窝。而陈虞却是嘴唇有点厚，那双睿智的丹凤眼里露出了与他年龄不相匹配的老成。
<br/>
<br/>        陈虞和落妍同时对着守中喊着：“守中老师好。”
<br/>
<br/>        守中一下就喜欢上了这俩个可爱的孩子：“从今天起，我就是你们的师傅，你们可以叫我老师，也可以喊我哥哥，我会对你们俩个很严格。希望你们能坚持下来。”
<br/>
<br/>        “守中哥哥，书上的你就不用教我们了，书上的我们都会，你就教我们一些书上没有的，我们不想成为书呆子。”落妍认真的对着守中说。
<br/>
<br/>        他们不缺知识，他们缺的是经验。
<br/>
<br/>        “是啊，守中哥哥，体能方面你放心，我和妹妹每天五公里20分钟，除非我们生病了。”陈虞骄傲的说，他们是小孩子，但是妈妈对他们很严厉，很小的时候就拽着他们跑步，他们上学体育成绩都是他们包揽了。
<br/>
<br/>        守中看了一眼潘禺，潘禺点点头。
<br/>
<br/>        陈子昂回来的时候就看到了潘禺和另外的一个年轻人在和陈虞落妍称兄道弟。
<br/>
<br/>        “小姨，你回来了。”潘禺看见陈子昂回来了，站起来打招呼。
<br/>
<br/>        “嗯，来朋友了？”陈子昂的心情不好，她回来是带陈虞和落妍去落家的。
<br/>
<br/>        “嗯，这个是我朋友守中，我找他来照顾陈虞和落妍，将陈虞和落妍送出去由他陪着。”潘禺解释着。
<br/>
<br/>        “小姨好，我叫守中，是潘禺哥的弟弟！”守中站起来向陈子昂问好。
<br/>
<br/>        陈子昂知道这就是潘禺给陈虞落妍找的保镖，她心里满满的感动，潘禺是行动派，往往都是事情做出来了才说。
<br/>
<br/>        看着守中那英俊而一脸正气的脸，陈子昂微微心安，有个年轻人跟在陈虞和落妍的身边她也能安心，况且这个人是潘禺找到的人，其身手肯定不简单。
<br/>
<br/>        落雪遇难，一时半会怕是不能送孩子出去了，得重新寻找机会，这一段时间就让守中陪在陈虞和落妍的身边。
