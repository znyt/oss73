　　很显然，山谷这一块地盘，直接成了禁地，在一股股超强恐怖的气息下，就算是三阶妖灵也要先掂量一下自己的实力。
<br/>　　无数生物都不明白，它们在做什么？看样子到像是在守护什么即将要出世的绝世宝物一样。
<br/>　　可在必死天劫毁灭性的疯狂轰炸下，成为荒芜废墟的山谷恐怕连一只植物不复存在，能有什么宝物。
<br/>　　可很快，它们的疑问得到了解答。
<br/>　　只见那颗苍天大树下，一股光芒愈演愈烈的笼罩出去，光芒内散发出一股无比阴冷的蛇类气息，宛如寒冬腊月，冷到骨髓，冷到灵魂深处。
<br/>　　光芒内，一条紫金蛇影若隐若现，仿佛九天神龙要一飞冲天，翱翔天际。
<br/>　　紫金蛇影一出现，让关注这场天劫风暴的精怪妖灵们都无比的熟悉，瞬间想到了那条巍峨巨蟒对抗天劫的画面。
<br/>　　“什么，那条蛇精还活着？”
<br/>　　无数暗中的精怪妖灵都惊讶的如五雷击顶，难以置信，精神处于半痴半呆的状态之中。
<br/>　　必死天劫下，蛇精还活着，在那么毁天灭地的攻击下，活了过来。
<br/>　　天哪，简直太不可思议了，简直太不科学了。
<br/>　　这绝对是创造历史的一刻，必死天劫下出奇迹，比死而复生更加神奇。
<br/>　　山谷中，那股光芒下的蛇类气息愈演愈烈，蛇影慢慢的变大，然后实质化。
<br/>　　“蛇精证道渡劫成功，他现在分明是在蜕皮进化啊”
<br/>　　妖灵山脉的修炼生物对于这一幕是非常熟悉的，精怪三阶证道成功，步入妖灵层次，都会有一番脱胎换骨的机遇，血脉提升，实力暴涨。
<br/>　　顿时间，很多区域的生物暗中讨论了起来：“这条蛇精证道成功，恐怕要朝着化龙而去了”
<br/>　　“蛇类一脉，一般成功证道，都是化龙”
<br/>　　“哼，化龙岂是那么简单的吗，普天之下，也不是没有蛇类证道成功，但你什么时候见过龙的影子了，都是进化一个四不像”
<br/>　　“不过这条蛇精还真的有台面，他证道成功，你看看出来守卫他的都是什么级别的妖精”
<br/>　　“一只三阶妖灵，三只超凡脱俗的超等生物，还有一株觉醒树精，这阵仗也太大了点吧”
<br/>　　“尤其是那三只超等生物，未入三阶，但是散发出来的气息却可以媲美三阶，血脉血统简直太可怕了”
<br/>　　修炼界，都说超等生物是老天的宠儿，血脉高贵，实力强大，步入三阶几乎是铁板钉钉的事情。
<br/>　　当然，在必死天劫不出的情况下。
<br/>　　猴儿山的两位猴大王表情一阵古怪，目光远视，望着苍天大树下光芒笼罩的那条紫金蛇影目瞪口呆。
<br/>　　真是打不死的小强啊，生命力是何等的顽强，如此天劫的毁灭下，还能苟延残喘的活下来，不佩服都不行。
<br/>　　“大哥，看来我们报仇的机会来了，这蛇精现在蜕皮，可谓任人宰割，就算他身边有不少妖精守护，但这些都算不了什么”赤阳巨猿看到紫金蛇影反而一喜，它们猴儿山成了这鸟样，无数猴子猴孙死在那条蛇的手上，这仇不能不报。
<br/>　　猿猴王脸色俨然，却摇了摇头，傲然道：“就算找那条蛇精算账，也不是这个时候，我堂堂猿猴王可不屑做这等落井下石的勾当，本王到想看看这条蛇蜕皮后会有多大的变化跟提升”
<br/>　　自信，这就是自信，猿猴王可是在妖灵山脉号称妖王之下无敌手的存在，岂会怕一条刚刚证道的蛇精。
<br/>　　不，貌似不能叫蛇精了，而是蛇妖。
<br/>　　另外，这蛇在必死天劫下都能不死，气运斐然，不是池中之物，要嘛出手直接杀死，如果杀不死，那么以后就是天大的祸害。
<br/>　　中间的利于弊，不断在猿猴王脑海中思忖，能成为一脉之王，可都不是泛泛之辈，智商都很高。
<br/>　　但不管怎样，现在猿猴王是绝对不会出手的，趁人之危，可不是他一代猴王能做出来的事情。
<br/>　　在则，山谷内一道道残暴的，恐怖的气息可不是假的，看样子是准备誓死守卫那条证道后蜕皮的蛇妖。
<br/>　　猿猴王还算厚道，没有落井下石，但是别人就没它那么厚道了。
<br/>　　山巅之上，白妖王妖气冲天，眉头紧促，淡淡的道：“还好本王逗留一下，否则还真被这条蛇给蒙混过去了”
<br/>　　“必死天劫之下都能不死，那么在本王的手中，你就只能必死无疑”白妖王一句话落下，看了看身后两位妖灵一眼。
<br/>　　两位妖灵那里还能不明白，残忍一笑道：“大王，您稍歇，属下两人去去就来，一条在蜕皮的蛇，简直跟捏死蚂蚁一样轻松”
<br/>　　另一妖灵道“到是牛妖王怎么也在下面，好像还帮着那条蛇守卫，有点棘手”
<br/>　　它们显然是认识牛妖王，至于那些气息强大的精怪，根本没放在眼中，三阶之下皆蝼蚁。
<br/>　　白妖王一眼千里，看着残暴气息，手握铁棍的牛妖王，道：“这头牛不是去了人类社会混吃等死吗，没想到回到妖灵山脉也不来拜见本王，看来眼里是没有本王啊，他若敢出来阻拦，你们两个也不用留手”
<br/>　　“属下明白了”
<br/>　　两位三阶妖灵骤然消失在山巅之上。
<br/>　　白妖王位高权重，身份高贵，不屑出手击杀一条还在蜕皮的蛇，换句话来说，那蛇还没这个资格。
<br/>　　派出两只三阶妖灵足够了。
<br/>　　......
<br/>　　山谷苍天大树下，光芒笼罩中的一条通体紫金色的蛇真在蜕皮，他浑身的蛇鳞片不断的蜕落，每脱落一片，他的蛇类气息就强盛一分。
<br/>　　没错，涂小安活了过来，他从必死天劫的究极大招下抗了下来，都说大难不死必有后福。
<br/>　　他证道渡劫成功，现在正是鲤鱼跃龙门的时刻。
<br/>　　只要涂小安蜕皮成功，那么他就是真正的三阶蛇王，从此超凡脱俗，天高任鸟飞。
<br/>　　他要让天底下所有的生物都要臣服在他这条证道的蛇王之下，让万物臣服。
<br/>　　这也是他第三次蜕皮了，但这一次的蜕皮最是无比至关重要，是三阶蜕皮，蜕去的是肉体凡胎，然后一步登天。
<br/>　　紫金条纹鳞片在龟裂，在蜕落，涂小安沉寂其中，感觉身体慢慢带来的变化跟改变。
