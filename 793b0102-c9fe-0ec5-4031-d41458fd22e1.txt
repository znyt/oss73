　　“来了？”元始站在玉虚宫门口，看着天空，道。
<br/>　　“见过大兄！”太清与通天两人走到元始面前行礼道。
<br/>　　“既然来了，那便走吧。”元始在两人的身上留意了一下，他一样便看出，老子的实力已经到了混元第四重天，通天的却还是在第三重天巅峰，这是初期与中期的差距。不过，他也没有说什么。
<br/>　　“是，大兄。”
<br/>　　于是，三人便这样踏空而去，前往混沌天外的紫霄宫。
<br/>　　此次前去紫霄宫的并不是元始的本尊，而是他的超我之身原始圣人，超我一直在参悟天道，期间鸿钧道祖也有好几次发现他的存在，此次前去并无不妥。
<br/>　　原始圣人的进步比较快，如今已经是混元后期的修为，他只需不断炼化天道法则，便可以提升修为，这就是以力破开天道空间证道的好处，虽然这也在很大的程度上抑制了他的未来发展，但因为有了本尊的存在，这其中限制可以说是没有，反正他也不想让超我之身证得天道境。
<br/>　　紫霄宫依然庄严、古朴，带着一种厚重的威势，就这样静静地立在那里，好似要存在到亘古，而且身上的天道气息也越发的浓郁了，更有一种说不出来的神性。
<br/>　　“见到诸位师兄！”
<br/>　　不久，女娲到来，看到原始三人，过来行礼道。女娲自上古时期妖族落寞之后，她便几乎没有出过娲皇宫，在洪荒的存在感实在是太低了。而且她也是一个人，不立教，反正人族自有她的一分气运存在，可谓是洪荒过得最悠闲的一位。态度之上既不偏向于玄门三教，也不偏向西方教，站在两岸观风，一副谁也不帮谁的样子。
<br/>　　三界水很深，女娲一人却从不涉水，只要不侵犯道她的利益，她就谁也不得罪。
<br/>　　“女娲师妹这日子过得挺悠闲的。”原始不咸不淡地说了一句。
<br/>　　“哪有师兄们过得悠闲啊。”女娲将此话反弹回去。
<br/>　　“哦，是吗？”这时，原始也不再说话了。
<br/>　　又一会儿，接引与准提两位圣人也来了，一番虚伪的见礼，各自便都站在了紫霄宫中，等待着鸿钧道祖的到来。
<br/>　　“嗡！”
<br/>　　一股无形地波动散开，鸿钧的那愈发缥缈的身影出现在前方的道台之上，他身上没有任何气势，平平淡淡，在诸圣眼中却是显得那么深不可测。
<br/>　　“吾等拜见老师！”
<br/>　　诸圣心中一凛，急忙躬身拜道。原始双眼一眯，看着上首的鸿钧，发觉他此时并无丝毫的感情波动，就好似是一块石头，他知道，此时的鸿钧代表的是天道，而不是鸿钧。虽然鸿钧为天道，但天道却不为鸿钧。至少现在天道还不是鸿钧，至于以后，那可不一定了。
<br/>　　“嗯。”鸿钧面无表情的点了点头，语气平淡道：“不必多礼。”
<br/>　　“敢问老师，此次召我等前来，所为何事？”原始身为众人的大师兄，自然是第一个开口。
<br/>　　“天地无量劫不兴，仙神无量劫不衰，一物兴，一物败，巫妖量劫之后，人族大兴，此乃定数。然天地无常，风云无相，兴衰更替，天数使然。天发杀机，斗转星移，地发杀机，龙蛇起陆，人发杀机，天翻地覆！”鸿钧眼皮都没抬下，便接着说道：“此次召尔等前来，乃是因为量劫将至，每次量劫，都是死伤无数，此次亦不例外。然，此次量劫，是应在诸圣门下。”
<br/>　　说着，鸿钧将目光看向诸圣，不知有意还是无意，原始总感觉他的目光多次看向自己与通天。
<br/>　　不过，他面色淡然，装作什么也没发现，任由鸿钧打量。
<br/>　　“敢问老师，此次量劫因何缘故，何人为应劫之人，吾等又该如何应对？”老子沉吟片刻，出声问道。
<br/>　　“洪荒天界，统领三界，但却是人手不够，此次量劫，名曰封神，是以三界众生除却功德深厚者，或证得圣道者，其余都在量劫之中。众仙参与量劫，评定功过，或生，或死。生者以身封神，死者真灵如封神榜，上榜封神，以成就天地诸神之位，完善天地秩序。”鸿钧淡淡道。
<br/>　　“什么？众生都在量劫之中？”诸圣听后，无不大吃一惊，封神可不是好玩的，此乃天地封神，只有封神之后的神，才能够得到天地承认，之后天地气运加身，还可以在人间传承香火，若是天地不承认的神，那只是野神，如无根浮萍，没有气运。
<br/>　　但封神之后，却是能够让天庭气运大增，而诸教却会气运大损，看来是鸿钧看不过三教把持洪荒大势了。原始若有所思。
<br/>　　“老师，这所封之神却是如何划分，可有等级之说？”通天亦开口问道，封神虽说不是一件好事，但若是利用得到，却也可以谋划一二。
<br/>　　鸿钧点头道：“众仙分为四等，功德福缘深厚者，可肉身封神，或者躲过量劫；根性薄者，死后真灵入封神榜，可成其神道，根性次者，不入封神，直接入其轮回，转世归来。”
<br/>　　“其余，此类之外，无功德，亦无福缘、气运者，若身入劫中，自当化为飞灰，就此了去。”鸿钧道。
<br/>　　诸圣听后，脸色都不好看，准提与接引亦是脸色阴沉，他西方教本无多少门人，却都还在此劫之中，让他们愁苦不已。女娲脸色平常，她不立教，无甚门人，量劫自然与她无关，她只要看好几个妖族之人即可。
<br/>　　“嗡！”
<br/>　　鸿钧因为不管诸圣的脸色是否好看，他大袖一甩，便有两道金光从虚空之中飞来，化为一块神榜与一根神鞭。两宝物熠熠生辉，宝光流露出来，有煌煌之威，更有天道的气息弥漫其中。原始从其中感受到了神道的气息。
<br/>　　神道乃是一种与仙道截然不同的大道，所谓仙道追求逍遥，追求天道、大道，而神道却是修香火，成神位，与日月同庚，天地同寿，亦能借助一丝天道的力量大战，只不过分多寡的问题。
<br/>　　“此乃封神榜与打神鞭，原始为诸圣大师兄，此二物便暂且交于你保管？待应劫之人入教，便交于他代天封神。”
<br/>　　“是！”原始接过封神榜与打神鞭，仔细打量了一下，想着回去好好研究一番，以后是否可以做出封妖榜、封仙榜什么的。
<br/>　　“尔等可商议封神之人，签订其中即可。”
<br/>　　说完，鸿钧便消失不见，将主动权交于诸圣。
<br/>　　原始脸色却是不好看，因为鸿钧这看似将主动权交于他们，但实际上却是暗藏祸心，诸教之中，只有阐教与截教之中弟子众多，鸿钧这一举动，却是要将原始与通天推到对立面上，因为谁也不想自己教下门人上榜。
<br/>　　不过，封神一事，谁又说得准呢？
