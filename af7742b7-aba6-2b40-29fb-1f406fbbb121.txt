　　时间才刚刚到达12点，便已是万人空巷。
<br/>　　现场十万名观众此刻都以落座，没有出现任何空位，无数的网友这个时候也都是在家中打开了电脑。
<br/>　　而这个时候距离决赛开幕式还有1个小时，距离决赛正式开始还有3个小时。
<br/>　　“哇，现场好热闹啊，不过好在终于不是荣耀蓝白的铁粉霸占全场的局面了。”
<br/>　　“期待我芷澜女神的出场，一定要多唱几首歌啊。”
<br/>　　“我荣耀白这次肯定会取得胜利的，上中野三路优势，怎么有不赢的道理。”
<br/>　　“得了吧，我蓝队也不是吃素的，上路南神并不输给香蕉，中路暗夜也可以和大魔王对位，至于下路那就更不用多说了，这可是世界公认的第一下路组合，怎么可能不赢。”
<br/>　　“就是，我蓝队能够拿第一次冠军，就能够拿第二次冠军，现在只能心疼大魔王他们了，谁让你们运气总是差那么一丢丢呢。”
<br/>　　“哥哥们，你们别闹了好么，我大夜雪你们怎么不说，而且不要忘了我们荣耀白可是有大招的，希望一会你们还可以这么淡定。”
<br/>　　比赛还没开始，直播机那里的众多蓝白战队的粉丝便是开始了‘掐架’，谁也不服谁。
<br/>　　现场此刻无数认识的还有不认识的粉丝，一个个也是聊的热火朝天，那个样子就像是他们已经认识了好多年一样。
<br/>　　不得不说，有的时候电竞就是有这样的一种美丽，能够让两个素不相识的人，只要聊上去，便像是多年的伙伴一般。
<br/>　　而在众人期待的目光下，时间终于是到达了一点钟，随着现场灯光一暗，十万名观众的声音顿时停了下来。
<br/>　　到了这一刻大家都知道正戏要来了，当看到聚光灯下一个身穿白色裙子仿若仙女一般的叶芷澜出现后，现场顿时爆发出了一阵山呼海啸般的欢呼声。
<br/>　　与此同时直播间里也是疯狂的出现了666的弹幕，仅仅因为叶芷澜的出现，所有人便是都已经到达了高*潮。
<br/>　　这也说明了叶芷澜如今的人气，随着叶帝给她创作的那些歌曲，这其中不仅仅有汉语的其他语言的也有很多，要不然的话叶芷澜恐怕也不会如此早的就打进了世界乐坛的领域。
<br/>　　听着那完全不熟悉的伴奏，一瞬间众多观众更加激动了起来。
<br/>　　“新歌，是新歌，没想到开幕式芷澜女神居然要唱新歌。”
<br/>　　“我的天，这次现场真的没有白来，还有比这更让人激动的开幕式么。”
<br/>　　“哇，这个排场真心有些大了啊，芷澜女神的新歌居然在这里发布出来了。”
<br/>　　“无敌，这个真的无敌了。”
<br/>　　听着这个伴奏，众人几乎第一时间便是想到了新歌，对于叶芷澜的歌曲，这些人早就已经熟悉的不能在熟悉了。
<br/>　　而直播间里的观众，在听到这个伴奏的时候，一个个也都是激动万分，激动过后则是遗憾，遗憾自己没有去现场。
<br/>　　“666666666666666，现在世界火神组织很悲伤，一个华夏的决赛居然能够请动芷澜女神，而且还在开幕式的时候唱新歌，这个可真的太牛逼了。”
<br/>　　“现在可是有着无数的平台抢着希望芷澜女神去他们那里发布新歌，并且给出的报酬都不菲，而今天女神居然在这里发布，这个真的没想到。”
<br/>　　“世界火神官方感觉要慌了，马上就是世界赛了，如果他们在世界赛的开幕式没有国内弄的好的话，那可就真的要丢人了。”
<br/>　　无数的观众在那里议论着，而叶芷澜唱的这首歌曲，正式之前叶帝给她创作的童话镇，就是连叶帝也没想到她会将这首歌曲在这里唱出来。
<br/>　　“听说白雪公主在逃跑，
<br/>　　小红帽在担心大灰狼。”
<br/>　　.......
<br/>　　叶芷澜那干净的声音，配合上这首歌曲，一瞬间所有人的心灵仿佛受到了洗涤一般，就那么静悄悄的看着叶芷澜。
<br/>　　一曲歌曲结束良久好，众人才反应过来，刹那间现场的欢呼声更加热烈了起来。
<br/>　　“我去，这首歌曲太好听了。”
<br/>　　“肯定又是出自大魔王之手，我也好想有个这样的哥哥啊。”
<br/>　　“是啊，有个这样的哥哥我感觉我也可以成明星。”
<br/>　　“说实话，芷澜女神之所以能够有今天的名气，大魔王至少占了8成的功劳，她的所有歌曲全都是大魔王包办的。”
<br/>　　“好听，真的好听，清澈的声音，配上这干净的曲子，真的完美。”
<br/>　　接着叶芷澜又唱了两首歌曲，才结束了她今天的表演，临下台前她还特意对着叶帝那边为叶帝加油。
<br/>　　看着叶芷澜做出了一个fighting的手势，那些宅男们更加的醉了。
<br/>　　场下的、直播间的众人这个时候已经是纷纷喊起了大舅哥，弄的在场下观看表演的叶帝，也是有些无语。
<br/>　　接着其他明星也是陆续出来表演，开幕式么，自然不可能只请了叶芷澜一人，只不过叶芷澜是其中名气最大的一个而已。
<br/>　　虽说珠玉在前，不过大家对于接下来的演出也都是非常满意的，看的出来这次华夏火神准备的非常充分，而且也非常的走心。
<br/>　　“好的，开幕式表演已经结束，现在距离比赛还有30分钟，让我们有请双方选手登台。”表演结束，米粒来到舞台上大声喊道。
<br/>　　随后在无数粉丝的欢呼声中，叶帝还有蒋念他们从两个方向向着舞台上走去，一边是黑色队伍，一边是红色队伍，依旧是帅气逼人。
<br/>　　“采访一下大魔王，对于今天的比赛你有什么想说的么？”林雅楠笑着开口问道。
<br/>　　“我们会是冠军！”叶帝淡淡的开口说道。
<br/>　　“那我们蓝队的选手怎么看？”米粒连忙朝着身边的蓝队众人问道。
<br/>　　“我们今天也准备了大招，冠军依旧会在我们手中。”李景禹同样淡淡的回到，话语中满是自信。
<br/>　　听到双方这番话，众多粉丝更加的激动了起来。
<br/>　　接着众人又被采访了一会后，才得以走进比赛室，看到双方开始调试设备，这一刻所有的观众都期待起了接下来的对战。
<br/>　　对蓝白两队的大招，他们更是好奇不已。
