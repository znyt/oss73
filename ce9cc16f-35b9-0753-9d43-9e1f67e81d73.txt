        她的师父，顶级大神。
<br/>
<br/>        可是……
<br/>
<br/>        连星儿郁郁寡欢地咬了咬唇。
<br/>
<br/>        要说她完全不在意自己被师父“除名”，那是不可能的。
<br/>
<br/>        但是，一想到师父可能是有女朋友了，才会把她删了，她又稍稍有些安心了。
<br/>
<br/>        毕竟由此可以证明，他的师父是个“好男友”。
<br/>
<br/>        明司寒坐在台式电脑前，目光直直地盯着荧幕上连星儿的游戏头像。
<br/>
<br/>        她一直在线，也未去打游戏，也未看直播视频。
<br/>
<br/>        但是不到十分钟的时间，她便下线了。
<br/>
<br/>        连星儿的头像灰了后，明司寒起身离开了网咖。
<br/>
<br/>        傅氏集团大厦，贵宾室。
<br/>
<br/>        傅夜沉亲自招待了从北城过来的明护国，也就是明司寒的爷爷。
<br/>
<br/>        北城明家，世代从军从政，在国内可谓是有权有势。
<br/>
<br/>        明护国的强势，远远在傅夜沉的爷爷傅炎廷之上。
<br/>
<br/>        明明两家有世仇，但此刻明护国却心平气和地跟傅夜沉提出了两家联姻的要求。
<br/>
<br/>        “我听说，冷先生有个掌上明珠。正巧，我也有六个优秀又未婚未恋的孙子。”明护国抬了抬手。
<br/>
<br/>        他身边的助理立即将一份资料，双手呈给了坐在对面沙发上的傅夜沉。
<br/>
<br/>        傅夜沉接过资料翻看了起来，资料里是他明护国的六个未婚未恋的孙子的个人信息。
<br/>
<br/>        其中一个，让傅夜沉印象格外深刻。
<br/>
<br/>        那就是明司寒。
<br/>
<br/>        上午的时候，他还会见过他，只不过拒绝了他的企划案。
<br/>
<br/>        明护国的六个孙子一个个确实都很优秀，有从军从政的，也有经商的，还有在演艺圈里混的，最小的这个明司寒，虽然还是个在校大学生，但是已经自己创业有自己的小公司。
<br/>
<br/>        傅夜沉看完资料后，将资料还给了明护国。
<br/>
<br/>        “明伯的孙子们一个个真的很优秀，我们傅家的女儿只怕高攀不起。”傅夜沉的脸上带着淡定的笑容，十分客气，又委婉地拒绝了明护国提出的联姻要求。
<br/>
<br/>        “你知道你们傅家人最后为什么会没落到只剩你这一脉了吗？因为，你们傅家人太喜欢内斗了。我们明家就不一样，上下都很齐心。”明护国冷嘲热讽道。
<br/>
<br/>        傅炎廷当年是怎么继承傅氏集团的，明护国可谓是一清二楚。
<br/>
<br/>        傅氏家族的继承权确实很残忍，而且还继续沿用旧时代里的嫡庶有别。
<br/>
<br/>        傅夜沉也不否认自己现在很偏心，因为他有了自己的儿子冷晏琛，就不会让弟弟傅昼景的儿子冷尚轩来分傅氏集团的一杯羹。
<br/>
<br/>        当初设想让冷尚轩来继承傅氏集团，是他以为筱颖无法生育。
<br/>
<br/>        “你们傅家的人，现在在军政界已经没什么影响了，更何况，你送你儿子去参军也只是为了磨炼他的毅力，回来后有能力继承傅氏集团，并非是为了让他留在军界出人头地吧？。此外，你们还有夏家可以联手。虽然，你夫人是夏家的外孙女，但终究也是个外姓人。更何况，夏家的人现在在军政界也已经渐渐没落了。俗话说得好，民不与官斗。你若是为了傅氏集团今后能发展得更好，我们明家，是你最好的选择。”明护国条条是道地说道。
<br/>
<br/>        他忽然改变主意，必定有他心里的老谋深算。
<br/>
<br/>        傅夜沉猜不透，但被人威胁的滋味也确实不好受。
<br/>
<br/>        明护国要是想搞傅氏集团的事情，也确实很简单。搬出一些条条框框，对傅氏集团的产业进行打压即可。
<br/>
<br/>        他是官，他是民，自然是斗不过他。
<br/>
<br/>        傅夜沉思忖了一会儿，终于不再像刚刚那么态度强硬，而是放软了口气，缓缓地说道：“我会把资料带回去，供我女儿选择。”
<br/>
<br/>        明护国也算是给足了他傅夜沉的面子了，拿六个孙子出来，让他女儿随便挑。
<br/>
<br/>        “希望，今后，我们能一笑泯恩仇，从此是一家人。”明护国起身，笑道。
<br/>
<br/>        傅夜沉却只是皮笑肉不笑。
<br/>
<br/>        晚上，傅夜沉将资料带回到家后，童筱颖看着资料上的六个明家之孙，不禁感慨道：“明家还真是人才辈出啊！”
<br/>
<br/>        “你真希望星儿嫁到明家去？”傅夜沉一脸不悦地皱起了眉头。
<br/>
<br/>        童筱颖瞥了傅夜沉一眼，忍俊不禁道：“你知道吗？认识我的好几个贵太太，都有意想跟我们攀亲。只是，我觉得星儿还小，就以星儿的年龄小为由委婉地拒绝了。”
<br/>
<br/>        “明护国的父亲，跟我爷爷是死对头。这一次，他们突然提出联姻，我总觉得是个陷阱。”傅夜沉随之在童筱颖的身旁坐下，抚着额头，愁眉不展。
