<br/>　　林峰道。

<br/>　　“大英雄。”
<br/>　　苏云笑呵呵的看着林峰。
<br/>　　“老师，我刚才帅不帅。”
<br/>　　林峰道。

<br/>　　林峰眨巴着眼睛道。
<br/>　　“你怎么就一天到晚想着这个。”苏云哭笑不得。
<br/>　　“那好吧，那我不睡大腿了，老师，我想抱着你睡可以吗？”
<br/>　　林峰认真道。
<br/>　　“还没完了事吧，小屁孩一个，脑袋里都想着什么呢，赶快出发吧，虽然这群怪物都死了，但难免会吸引更多怪物过来。”苏云道。

<br/>　　大汉心头的摸了摸孩子的脸。
