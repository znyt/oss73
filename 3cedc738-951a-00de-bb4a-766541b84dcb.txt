        七点半集合，还有十五分钟。
<br/>
<br/>        连星儿顾不上去把头发弄干，而是直接换了鞋，拿起手机和钥匙，关上门离开了寝室。
<br/>
<br/>        临海大学很大，连星儿脱离了大部队后，没找到指路牌，只能求助路过的学长学姐。
<br/>
<br/>        “这位学姐，请问第五教在哪？”
<br/>
<br/>        “不知道。”
<br/>
<br/>        “学长，请问第五教在哪？”
<br/>
<br/>        “……”
<br/>
<br/>        “请问……”
<br/>
<br/>        连星儿不下问了十多个路过的学长学姐，他们要么不回答，要么说不知道，要么看到她就加快步伐或者调头走。
<br/>
<br/>        就算她再傻，也明白，他们是有意躲着她。
<br/>
<br/>        只是令她不明白的是，他们为什么要躲着她？
<br/>
<br/>        连星儿此刻真后悔，当时报到的时候就应该拿一份校内导图。
<br/>
<br/>        无奈之下，她给顾千凝打了通电话，但是顾千凝未接。
<br/>
<br/>        此时，她又想到了明司寒。
<br/>
<br/>        但是一想到他白天对她的那种态度，她又打消了打电话找他问路的念头。
<br/>
<br/>        最后，连星儿特意去了一趟校门口，在保安大叔那里要了一张校内导图，才赶到了第五教。
<br/>
<br/>        结果因为迟到，被班主任当堂点名批评了。
<br/>
<br/>        “看样子，你们这届的文科状元，竟然是个没时间观念的女生。这说明，人家能拿到状元，靠的是天赋。”班主任这么一调侃，台下哄堂大笑。
<br/>
<br/>        连星儿真不知道这有什么好笑的，默默地找座位坐下时，才发现顾千凝和另外两个室友坐在了一块。
<br/>
<br/>        晚上的集合，班主任除了点名以外，还交代了一下接下来半个月时间的军训内容，以及介绍军训教官。
<br/>
<br/>        连星儿忍不住在桌底下发了条简讯给顾千凝，问她为什么不等她，为什么不接她电话。
<br/>
<br/>        结果，她刚把信息发出去，又被班主任给当堂点名了。
<br/>
<br/>        “连星儿，就算是班级集合，老师和教官在台上讲话，作为学生，你在桌底下玩手机，合适吗？”班主任是个女老师，说起话来，声音有点儿尖。
<br/>
<br/>        连星儿立马收起手机，尴尬地冲着讲台上的班主任咧嘴一笑。
<br/>
<br/>        虽然在桌底下玩手机的不止她一个人，但是一个晚上被单独点名两次，连星儿觉得自己的颜面都快丢没了。
<br/>
<br/>        集合解散后，连星儿主动朝顾千凝走去，谁知顾千凝却对她视而不见，直接挽着另外两个室友的臂弯，一起离开了教室。
<br/>
<br/>        而且，顾千凝并也未回她的信息。
<br/>
<br/>        回到寝室后，她们三个人倒是有说有笑，但如果她连星儿一插嘴，她们三就立即安静下来，各忙各的事情。
<br/>
<br/>        都是新来的，大家不可能无缘无故就这样对待自己。
<br/>
<br/>        连星儿趁着顾千凝去卫生间的时候，也尾随了进去，一脚踢关了卫生间的门。
<br/>
<br/>        “说说吧！你们到底是怎么回事？”连星儿双手抱臂，很没好气地问。
<br/>
<br/>        明明还跟她有说有笑还有共同兴趣爱好的女孩子，怎么就突然对她不理不睬了呢？
<br/>
<br/>        顾千凝被连星儿的气势也吓到，下意识地缩着脖子，弱弱地回答道：“你洗澡的时候，有个学姐跑来寝室里说，要我们孤立你。说是这里的‘校霸’吩咐的。如果我们不按她所说的做，下一个被‘欺负’的就是我们了。”
<br/>
<br/>        什么？校霸？孤立？欺负？
<br/>
<br/>        “星儿，你在这学校里有得罪谁吗？”顾千凝咽了咽口水，问道。
<br/>
<br/>        连星儿皱了皱眉头，百思不得其解。
<br/>
<br/>        她一个大一新生，刚来学校报到，还没认识几个人，怎么可能会得罪人嘛？
<br/>
<br/>        “对不起啊！星儿……我这也是……”顾千凝欲言又止。
<br/>
<br/>        连星儿倒是落落大方地摆摆手：“算了！没事！”
<br/>
<br/>        看样子，解决问题的根本是要找到那个“校霸”。
<br/>
<br/>        事情问完了后，顾千凝从卫生间里出来时，另外两个室友用异样的眼神打量着她。
<br/>
<br/>        连星儿见状，索性跟另外两个室友摊牌，嘴角微扬地说：“你们若是想继续孤立我，我没意见，只不过了，我这人性格很极端的噢！有恩报恩，有仇报仇，你们孤立我就孤立我，但不要招惹我，因为……我还很会打架，对女生也不会手下留情的。”
<br/>
<br/>        三人听后，同时沉默了。
<br/>
<br/>        顾千凝率先打破僵局：“星儿，我不管了那些了。我们依旧是好朋友。”
<br/>
<br/>        另外两个室友面面相觑后，也跟着向连星儿问好：
<br/>
<br/>        “星儿，你好，我叫路娅玟。”
<br/>
<br/>        “我叫赵安彤。”
<br/>
<br/>        看到另外两个室友对自己转态，连星儿顿时高兴不已，连忙从床下衣柜里拿出一个小包，往寝室中央的大桌上一放。
