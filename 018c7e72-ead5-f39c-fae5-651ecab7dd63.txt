　　帝乙在位三十年，驾崩，传位于三子寿王。寿王乃帝辛，成王之后，为纣王。
<br/>　　帝辛为王，文有太师闻仲，武有镇国武成王黄飞虎，文足以安邦，武足以定国。纣王坐享太平，天下八百诸侯尽归商。又有四路大诸侯统领八百小诸侯，东伯候姜桓楚，南伯候鄂崇禹，西伯候姬昌，北伯候侯崇虎。
<br/>　　一日，天子早登大殿，下方文武大臣各立两边，纣王身旁驾官问道：“有事请奏，无事退朝！”
<br/>　　话还没有说完，便有一人出列，俯首拜道：“臣商容有事请奏！”
<br/>　　上首纣王却好奇了，商容为丞相，更是前朝元老，素来德高望重，见他说话，纣王非常好奇，便答道：“爱卿但说无妨！”
<br/>　　“臣启禀陛下，半月之后三月十五日，乃是女娲娘娘的圣诞之辰，请陛下驾临女娲宫降香。”
<br/>　　纣王好奇问道：“女娲娘娘有何功德，要孤王亲自前去上香？”
<br/>　　商容连忙说道：“女娲娘娘乃是远古大神，亲自创造了人类，为人族圣母。人族感恩故祭祀以报之。今朝歌福泰民安，国运绵长，正是有正神庇佑，陛下当前往上香。”
<br/>　　纣王想了想，觉得没什么问题，便道：“准奏！”
<br/>　　半月之后，纣王带领朝中文武百官驾车前往女娲庙进香。
<br/>　　至女娲庙，纣王开始观看宫殿华丽，只见殿前五彩金装。金童对对执幡幢，玉女双双捧如意。玉钩斜挂，半轮新月悬空；宝帐婆娑，万对彩鸾朝斗。碧落床边，俱是舞鹤翔鸾；沉香宝座，造就走龙飞凤。飘飘奇彩异寻常，金炉瑞霭；袅袅祯祥腾紫雾，银烛辉煌。
<br/>　　纣王在一众大臣的主导下，完成了祭祀活动，焚香入炉，祭拜女娲娘娘。
<br/>　　拜后，纣王仔细打量着大殿中的一切，他看向女娲神像静静地矗立在那里，神像前好似有一层薄纱阻隔，让人望了，不能尽兴。只觉得神女朦朦胧胧间，仙子妙曼，容貌瑞丽，瑞彩翩翩，国色天姿，宛若蕊宫仙下凡，月殿嫦娥下世。
<br/>　　就在此时，一阵阴风吹过，吹入纣王脑海深处，让纣王心中的欲念，一瞬间放大千百倍。与此同时，商朝的国运真龙突然对着女娲庙方向一声长吼，吼声惊人，一般人却是不能听见。
<br/>　　这时纣王再向女娲神像看去，却觉得神魂飘忽，陡起色心，自思道：“孤王贵为天子，富有四海，纵有六院叁宫，却并无如此艳色。”
<br/>　　他立刻命人取来文房四宝，天子深润紫毫，在行宫粉壁之上，作诗一首：
<br/>　　“凤鸾宝帐景非常，尽是泥金巧样妆。
<br/>　　曲曲远山飞翠色，翩翩舞袖映霞裳。
<br/>　　梨花带雨争娇艳，芍药笼烟骋媚妆。
<br/>　　但得娇娆能举动，取回长乐侍君王。”
<br/>　　丞相商容见状，大惊，连忙劝谏道：“女娲是上古大神，天下之福主，老臣是让陛下来祈福求德的，如今陛下作诗亵渎圣明，毫无虔诚，必定获罪于神圣，非天子之礼，希望陛下将此诗用水洗去。恐怕天下百姓见到此诗，会传言陛下无德啊！”
<br/>　　“哼！”
<br/>　　纣王冷哼一声，不悦道：“孤王看女娲之姿绝色无双，以此作诗称赞，岂有他意？卿家无需多言。孤王乃万乘之尊，留下此诗让百姓见之，可以见到女娲的绝世美貌，亦可见到孤王之遗笔！”
<br/>　　“陛下……”
<br/>　　商容还想说，可是纣王打断他的话，道：“好了，不必多言，回宫！”
<br/>　　文武百官见此，皆默默无言，不敢顶撞大王，遂随着纣王回去了。
<br/>　　在纣王走后，女娲庙的虚空之上，一灰袍道人出现，他的嘴角还隐隐有丝丝血迹。看着纣王走后，他低语道：“不愧是人族国君，气运深厚，贫道只是略施手段，便让国运反噬，受了轻伤。不过，一切都值了，为了东方动乱，量劫开启，这一切都是值得的。”
<br/>　　道人默默无言，良久，施展手段，掩去这一切痕迹，便转身离开。
<br/>　　在道人离开的瞬间，便有数道圣念扫视过来，扫了多遍，见无任何人，便都散了开来。
<br/>　　娲皇宫中，女娲若有所感，掐手一算，圣念一扫，便知此事经过，在看到那庙宇粉壁上的诗句之后，她立刻火了，大骂道：
<br/>　　“好胆！无道昏君，不想修身立德以保天下，却竟敢写淫诗侮辱本宫。甚是可恶！成汤享国运六百余年，如今气数将尽，还有二十八年，若本宫不给他个报应，他却是不知本宫之怒！”
<br/>　　女娲心中极度不悦，她取来当初妖族天庭留下的招妖幡，召集各路妖魔前来。不一会儿，阴风吹过，迷雾笼罩，天下群妖俱到行宫听后女娲调遣。
<br/>　　女娲吩咐彩云童子：“让各路妖魔退避，只留下轩辕坟三妖伺候。”
<br/>　　三妖分别是一千年狐狸精，一九头雉鸡精，一玉石琵琶精。三妖入宫，拜见女娲娘娘。娘娘却说：“商汤气数已尽，凤鸣西岐，西周已生圣主，你三妖可隐其妖形，托入朝歌，祸乱君心，助武王伐纣，事成之后，可助尔等修成正果。”
<br/>　　三妖自然不敢违背女娲圣人的旨意，一番拜谢后，便去了人间，想办法拜入朝歌。
<br/>　　同时，女娲觉得如此还不够，想了想，亦将自己座下灵珠子投放下界，通过转世去祸乱成汤的天下。
<br/>　　同时，玉虚宫中，元始若有所思，纣王这态度前后反差实在是太大了，之前还是一个治国明君，而自从进香之后，便成了无道昏君，天地间哪有这样奇怪的事情，这定是有人在背后搞鬼。
<br/>　　“是谁？能够瞒过诸圣，也只有圣人了。女娲自然不会自污，当然也不排除这个可能，那么，还有西方二圣，是他们么？”元始默默思索道。
<br/>　　“还是，这是天地大劫来临的征兆，量劫的气息在影响着纣王？”
<br/>　　元始不能确定，只能走一步算一步，封神量劫，未尝不是一个机会，三界毕竟安逸太久了。
<br/>　　“希望此次量劫出现点有意思的事情。”元始默默道。
<br/>　　朝歌之中，自从纣王进香之后，看见女娲美貌，便朝思墓想，废寝忘食，每每看见三宫六院，便觉得没有兴味。
<br/>　　一日，纣王唤来中谏大夫费仲。此人是纣王的幸臣，经常向纣王进献谗言，纣王无有不从。
<br/>　　见到费仲，纣王也不隐瞒，直接说出自己的目的。费仲听后，直接谏言道：“陛下乃九五至尊，天下四海，皆为陛下所有。陛下此事，又有何难？陛下只需明日传下一旨，颁布四路诸侯，选取天下美女，以充王庭后宫，又何必担忧天下绝色不入大王之法眼。”
<br/>　　纣王听后，大喜，即刻命人颁布指令，筛选天下美女，进贡皇朝，以充后宫。
<br/>　　纣王这一道命令的颁布，直接导致了商朝的危机，加快了王朝灭亡的步法。
