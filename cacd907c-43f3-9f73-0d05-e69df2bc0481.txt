　　“十四公子，可要小心一点。”
<br/>　　沉沦海王看向楚风眠，出言提醒道。
<br/>　　楚风眠现在得到了一整条的灵石矿脉，这可是月海商会的大主顾了，他也不愿意楚风眠发生什么意外。
<br/>　　“无妨，区区海十三而已，敢来找麻烦，我随便就可以杀了他。”
<br/>　　楚风眠随意的说道，语气之中透露这无边自信。
<br/>　　那沉沦海王跟文供奉都是对视了一眼，眼神之中露出了几分吃惊。
<br/>　　海十三跟海十四的争斗，在镇南海城之中也是被人津津乐道的事情，不过二者的实力，向来都是在伯仲之间，海十三还稍强几分。
<br/>　　可听着现在楚风眠的话，居然是如此的自信，丝毫不惧海十四。
<br/>　　“难道这海十四，还另有奇遇？”
<br/>　　沉沦海王跟文供奉心中都不由的想到，要是另有奇遇的话，未来也并非是没有一飞冲天的可能。
<br/>　　曾经海剑宗的一位宗门，在海姓的核心弟子之中，只是位列第十九，可最终得到了惊天的奇遇，一飞冲天，直接夺得了海剑宗的宗主之位。
<br/>　　得到奇遇，一飞冲天的事，可并非是没有。
<br/>　　“沉沦海王，文供奉，我们还是谈谈交易的事情吧。”
<br/>　　沉沦海王跟文供奉心中的想法，楚风眠看了一眼便是就看穿的一清二楚，他淡然的说道。
<br/>　　反正这一次海十四离开的一段时间内，除了楚风眠，外人根本不知道发生了什么事，楚风眠想要怎么说都成。
<br/>　　让人以为海十四得到惊天奇遇，未来楚风眠出手施展出更强的实力，也足以解释的通。
<br/>　　“好，先谈交易，我们月海商会之中，一共还有二百一十枚仙石，现在就全部归十四公子了。”
<br/>　　沉沦海王也是话锋一转，不再去继续谈论刚刚的事。
<br/>　　他拿出了一枚空戒，递给了楚风眠。
<br/>　　楚风眠接过空戒，一股灵力注入其中，就感觉到了一股精粹的仙力，一共二百一十枚仙石，一枚不差。
<br/>　　楚风眠接过仙石，便是大手一挥，那四万枚神品灵石化为的山峰，直接向着沉沦海王飞了过去，随后准备转身离开了。
<br/>　　“十四公子稍等。”
<br/>　　楚风眠就在即将离开的时候，沉沦海王突然追了上来，他只拿走了区中一半的神品灵石，而将另外一半，又给了楚风眠。
<br/>　　“刚刚的事，只是说辞罢了，谈好的价格，不用更改，这些灵石还请十四公子收回。”
<br/>　　沉沦海王看向楚风眠开口道。
<br/>　　“哦？月海商会倒是讲诚信，那我就不客气了。”
<br/>　　楚风眠嘴角露出了几分笑容，也不客气，手一挥，这些神品灵石又落入到了楚风眠的空戒之中去。
<br/>　　虽然楚风眠拥有神脉，可以凝聚神品灵石，不过能省一点，楚风眠自然也要省一点。
<br/>　　买下了仙石，楚风眠也没有从这月海商会之中在停留的打算了，化为了一道遁光，身形瞬间消失了。
<br/>　　这月海商会之中，布下了层层的禁制阵法，常人一旦是进入其中，都难以离开。
<br/>　　可楚风眠的突然离开，却是无声无息之间，那沉沦海王，文供奉二人，都没有看清楚风眠离开的遁光，到底是何时离开的。
<br/>　　就是一个眨眼之间，楚风眠的身形便是消失在了这房间之中，消失的无影无踪。
<br/>　　“这，海十四的实力，什么时候如此恐怖了，我都看不清他的身形？”
<br/>　　文供奉呆呆的打量了一眼四周，不可置信的说道。
<br/>　　他可是一位半步天人境强者，如今都没有看清楚风眠离开时候的遁光，楚风眠究竟是如何离开的，他都一概不知。
<br/>　　“这海十四，果然是有着惊天的奇遇，神品灵石矿脉啊，这可都不是如今的时代之中可以诞生出的东西了，你可记得，数百年前，海祭宗得到的那一条神品灵石矿脉，可是从古海遗迹之中发现的。”
<br/>　　沉沦海王也是惊叹一声，脸色凝重的说道。
<br/>　　“看来此人，也是得到了古海遗迹的宝藏，不然不可能实力会进展的如此之快。”
<br/>　　“古海的遗迹啊，这可是天大的机缘，那可是我们海族巅峰之时的遗迹。”
<br/>　　“我们海族，曾经在数十万年前，也是称霸一时的存在，当时沧海星域，甚至都不在三大圣域之下了，堪称第四大圣域，当时据说沧海星域中，天人境，甚至是玄天境的强者，都有不少。”
<br/>　　“这海十四能够得到古海遗迹，那真的是大机缘，一步登天了。”
<br/>　　文供奉的声音之中，也无比的羡慕。
<br/>　　“这种机缘，可遇而不可求，羡慕也羡慕不来，古海那种地方，我可没有勇气去踏足。”
<br/>　　沉沦海王感叹了一声说道。
<br/>　　“本来我看着海十四未来也就能成为海祭宗的长老，没想到居然得到了如此机缘，看来这一次的海祭宗宗主之位，又多了一个争夺的人物了。”
<br/>　　“古海……年轻之时，我倒是踏入过其中一次过，不过那其中的危险，一旦是踏入其中，真的是十死无生，我可不愿意踏入第二次了。”
<br/>　　提到古海的时候，文供奉都是不由的一哆嗦。
<br/>　　哪怕他身为一位半步天人境的强者，在提到古海的时候，心中已经是无比恐惧。
<br/>　　这古海，乃是沧海星域的禁地，提到人人都会为之色变。

<br/>　　沉沦海王突然话锋一转的说道。
<br/>　　“九域？你是说曾经身为圣域，后来没落的九域？”
<br/>　　文供奉眼神一亮，开口道。
<br/>　　“看来你也知道这消息了。”
<br/>　　“不错，九域消失了这么久，没想到居然是藏在了一个小小的大罗天星域之中，九域虽然没落了，可毕竟是曾经圣域，遗留下来的无数宝藏，比起古海还要多得多。”
<br/>　　沉沦海王眼神之中露出了几分炙热之色开口道。
<br/>　　“据说不少星域的强者，已经是去了九域，得到了惊天的宝藏，如果有机会，我们也去九域，若是得到什么机缘，我们未必没有去渡天人大劫的机会。”
