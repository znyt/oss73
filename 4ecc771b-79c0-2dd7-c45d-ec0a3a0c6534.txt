<br/>    陈明将背后的卓青瑶放了下来，看向周围的无尽水珠，如同一场水幕天华将所有的一切笼罩，这一刻，陈明踏步行走在水珠之中。
<br/>    
<br/>        紫霞仙子看向周围，他们可以从水珠之中看到陈明已经动身，同样看得到血龙正在急速飞行，如同在靠近他们，又如同在远离他们，莫名的恐惧开始升起。
<br/>    
<br/>        即便是悟道榜上的天才，在此刻也是产生了一丝惧怕之意，他们虽然是悟道榜上的天才，但是毕竟只是悟道境，而不是超脱。
<br/>    
<br/>        超脱代表着的，是大陆之上的最强战力。
<br/>    
<br/>        超脱，不仅仅是超脱于世，他们更在世间，而当他们入世，代表的就是灾难。
<br/>    
<br/>        超脱，名为超脱，便是应该超脱于世外的存在，只是可惜，他们，依旧在人间。
<br/>    
<br/>        雪自悟看向身旁的三人，问道：“现在怎么办？”
<br/>    
<br/>        刘芒提着手中的长枪，道：“宗师做到他应该做到的，我们做到我们应该做到的！除此之外，还能怎么样？”
<br/>    
<br/>        刘芒话里的意思很简单，即便是他们想怎么样，但是他们又能怎么样，还是做好宗师安排给他们的事吧。
<br/>    
<br/>        就让悟道境的归悟道境，大君的归大君。
<br/>    
<br/>        陈明的危险指数是1750，开启了和之后，未必不能和大君一战，龙鲲虽然有着2500的危险指数，但是这里毕竟是它的心脏，在它的心脏之中，它并不能完全发挥自己的实力。
<br/>    
<br/>        行走在水珠之中，陈明可以感受到自己的天人合一受到了阻碍，那是血龙侵入了他的世界之中，他在大阵之中不再是掌控一切。
<br/>    
<br/>        说实话，他很讨厌这种感觉。
<br/>    
<br/>        现在，总有一些地方让他感应不到。
<br/>    
<br/>        就像是，自己的构筑的大阵开始支离破碎。
<br/>    
<br/>        他还是第一次遇到了这样的难关，一个如此强大的对手。
<br/>    
<br/>        陈明伸手，剑龙的口中吐出一剑，飞向他的手中，那是秋霜，代表着四时之中肃杀的秋。
<br/>    
<br/>        陈明握剑，一股秋意盎然，秋天，霜杀百草，何为草，弱小而多便如草，一如这些水珠，虽然很多，但是每一颗水珠都不强。
<br/>    
<br/>        这并不是幻术，所以陈明也没有动用云系功法，他只是握剑，人剑再次合一，这一剑，叫做霜杀百草！
<br/>    
<br/>        一剑祭出，陈明跨步，双手握剑，于潇潇水珠之间便是一剑劈去！
<br/>    
<br/>        杀意顿时弥漫四周，一颗水珠接着一颗水珠破碎，无边的肃杀剑势笼罩一切，而后，漫天都是水花飞舞。
<br/>    
<br/>        无尽的水花之中，一条血龙纵横咆哮，朝着紫霞仙子四人冲去！
<br/>    
<br/>        陈明抬头，挥手之间，剑龙已经穿梭而来，但是此刻，剑龙的速度已经有所不及。
<br/>    
<br/>        陈明一跃而起，双手紧握秋霜，朝着血龙的头便是一剑劈去！
<br/>    
<br/>        血龙狂舞，陈明被一击撞在心脏的墙壁上，口中当时就是一口血雾喷出！
<br/>    
<br/>        而就在这时，剑龙已经追上血龙，两条长龙，瞬间交织在一起！
<br/>    
<br/>        苏青阳道：“我等将这个豁口变大，尽快削弱龙鲲的力量！”
<br/>    
<br/>        其他三人也是点头，外面战的天昏地暗，他们也不能干愣着。
<br/>    
<br/>        刘芒一边快速挥枪，一边道：“我从来不知道，这大陆上居然还有着宗师这般强者！”
<br/>    
<br/>        紫霞仙子道：“大君的战力，大陆之上，几乎都是人尽皆知的，毕竟他们的一举一动，都是一场灾难，可是就是我群星仙门，也是从来没有宗师的半点消息。我一直以为，宗师只会炼丹，并没有任何战斗的手段，看来，是我看错了。”
<br/>    
<br/>        苏青阳道：“不是吧，你从来没有见过宗师出手吗？宗师脾气这么好的吗？”
<br/>    
<br/>        紫霞仙子道：“不是他脾气好，而是一般不知道他的身份的，都是一些普通人王，连宗师门下的卓青瑶都打不过，宗师怎么会出手？而知道他的身份的，比如我们这些古仙门，谁敢去得罪一个月产百万枚丹药的宗门？”
<br/>    
<br/>        刘芒忽然道：“不用看宗师的战斗了，他的战斗方式，我们学不会的。”
<br/>    
<br/>        雪自悟也是道：“我们做好自己的事就好了。不要给宗师添麻烦了，我居然也有成为别人的麻烦的一天！”
<br/>    
<br/>        这一边，血龙和剑龙已经纠缠在了一起，陈明全心全意的操控着剑龙，到了这个时候，这龙鲲应该没有什么别的手段了，毕竟不像是修行者，手段花样很多。
<br/>    
<br/>        随着龙鲲的鲜血不住的流淌而出，血龙也是变得越发虚弱，渐渐的，竟然是剑龙占了上风。
<br/>    
<br/>        陈明也是开始变得轻松起来，足足一刻钟之后，血龙终于是消散，而心脏之中的鲜血，也是流淌得差不多了，紫霞仙子忽然道：“这些血肉，变得柔软了，龙鲲撑不住了！”
<br/>    
<br/>        陈明此刻，也是松了一口气，“现在不用急了，这龙鲲必死无疑，不过只是时间问题，龙鲲死了，这些凶兽想必也会撤退了。”
<br/>    
<br/>        陈明伸手，单手抱住卓青瑶，头顶之上千剑开路，直接将龙鲲的血肉割开一个大口子，从龙鲲的身体之中出来。
<br/>    
<br/>        陈明看到紫霞仙子四人也已经出来了，便道：“你们四人对这里的人王比较熟，你们善后。”
<br/>    
<br/>        带着卓青瑶，陈明回到头骨之中，陈明在头骨之中的唯一的座椅上坐下，喝道：“跪下！”
<br/>    
<br/>        卓青瑶有些心虚的跪下，陈明笑道：“长能耐了啊？敢一个人去挑战大君都无法战胜的龙鲲了？”
<br/>    
<br/>        卓青瑶抬头，丝毫没有后悔之色，“师父那个时候在炼丹！”
<br/>    
<br/>        陈明道：“既然你还没有想明白你错在哪里了，就去外面跪着去！”
<br/>    
<br/>        三个时辰之后，陈明走到头骨之外，看向跪在地面上的卓青瑶，小脸之上，写满了倔强，“你知道错在哪儿了吗？”
<br/>    
<br/>        卓青瑶道：“师父那个时候在炼丹，要我自己一个人逃走，我做不到！”
<br/>    
<br/>        陈明笑道：“这一点，你没错，你错在两点，第一，你不相信我可以安全离开，这是怀疑我的实力，第二，你错在你太弱。弱小，便是你的错，是谁口口声声说要做那天下第一的？现在，你明白了吗？”
<br/>    
<br/>        卓青瑶抬头，满眼都是坚毅，双眼之中已经有着泪花泛出，“我知道错了，师父。”
<br/>    
<br/>        陈明将卓青瑶扶起来，问了句，“跪着不疼吧？”
<br/>    
<br/>        从卓青瑶哪里传来蚊子哼哼的声音，“嗯，一点点难受。”
