　　天涯城，坐落在加玛帝国万里之遥一座名‘天擎’的广阔山脉中。
<br/>　　作为方圆千里内，唯一一座拥有空间虫洞能够通往中州的城市，天涯城在千里内的繁华程度可谓首屈一指。
<br/>　　不出所料，以小医仙御空飞行的速度，直到中午的时候才抵达天涯城最外围。
<br/>　　在赶路的过程中，慕清霄也将中州的信息叙述了一些。
<br/>　　中州大陆，面积极其辽阔，因此在中州陆地上的一些大型城市里，都会设有空间虫洞，以此来缩短路程。
<br/>　　这些空间虫洞的建立者，都是一些属于大城市巅峰强者，或者是他们的先辈，所以空间虫洞是属于私人所有。
<br/>　　当然，空间虫洞这种东西，就算是某些强者家族也不敢独自占用，也极少有这样家族如此愚蠢。
<br/>　　若是如此，暂且不说引起群愤，更会损失空间虫洞每年所带来的利润，如此庞大的利润，足以养活一个大型家族。
<br/>　　因此，中州每一座空间虫洞的出现，都会引起无数势力眼红，但除了本身强大的家族或者宗派，其他人只能眼馋，不敢染指。
<br/>　　虽然斗皇和斗宗多如狗，但斗尊已经是一方霸主，拥有空间虫洞的势力，基本都有斗尊级别的存在。
<br/>　　如此雄厚力量的势力，若是谁敢染指手中的资源，必将遭到雷霆一般的报复。
<br/>　　除了空间虫洞外，中州势力的划分也是非常明确。
<br/>　　在中州内部，人族势力分为：一塔一殿，二宗三谷，四方阁。
<br/>　　塔指的是丹塔，殿指的则是魂殿，魂殿是中州的极强势力，曾经与丹塔在暗中发生过摩擦，至于结果未曾可知。
<br/>　　以慕清霄对魂殿的了解，以及诡异的作风，吃亏的极有可能是丹塔。
<br/>　　而且，除了某些远古族，显有人知道，在魂殿的背后还有整个魂族，这可是一个庞然大物。
<br/>　　得知魂殿的背后乃是远古八族最强最诡异的魂族，并且蛰伏在中州，彩鳞与小医仙心中都莫名的一寒。
<br/>　　只是，听闻慕清霄平淡的语气后，心中顿时涌上无尽的安全感。
<br/>　　语气平淡，但其中却充斥着无尽的自信，仿佛魂族都不被他放在眼中。
<br/>　　除了魂殿与丹塔，后面的就是二宗，三谷，四方阁。
<br/>　　二宗所指的是天冥宗与花宗，这两个宗派在中州也有着极强的地位，宗内强者如云，虽然比不上丹塔与魂殿，但也是不可小觑的存在。
<br/>　　三谷则是冰河谷，音谷和焚炎谷。
<br/>　　焚炎谷中正巧有慕清霄需要的异火‘九龙天罡火’，前往中州后，有必要前往一趟焚炎谷。
<br/>　　三谷势力肯定是比不了丹塔和魂殿的，更是略逊二宗一筹，却也是一股极强的势力，都拥有斗尊坐镇。
<br/>　　或许在其他人眼中，焚炎谷极其强大，可对慕清霄来说，区区斗尊而已，挥手可杀的蝼蚁，哪怕他将焚炎谷当作后花园，未必不可。
<br/>　　排在最后的便是四方阁了，星陨阁，万剑阁，黄泉阁，风雷阁四阁。
<br/>　　因为四个方位刚好成四角方向屹立在中州，所以也被称为四方阁。
<br/>　　星陨阁有风尊者，万剑阁有剑尊者，黄泉阁有黄泉尊者，风雷阁有雷尊者，四位斗尊坐镇，在中州也是一股不弱的势力。
<br/>　　除了一塔一殿，二宗三谷，四方阁外，中州还有许多强大势力，甚至非人族势力……
<br/>　　对于慕清霄如此了解中州的信息，小医仙与彩鳞都有些诧异，若非后者否认，她们都认为离开的五年里，慕清霄是去了中州。
<br/>　　不多时，慕清霄的视线中已经看到一处自地面上凸出来一截的山峦，正是通往中州的天擎山脉。
<br/>　　御空飞行穿过平原，化作流光跃入山脉，旋即山脉中一条条宽敞的道路，便四通八达的出现在慕清霄等人的视线中。
<br/>　　来此的并非只有慕他们三人，毕竟这里是唯一可以通往中州的地方。
<br/>　　立于虚空中，慕清霄就能听见周围响起一阵阵破风声，旋即就见到许多奇形怪状的飞行魔兽展开双翼向山脉中掠去。
<br/>　　相比加玛帝国，哪怕黑角域都不可能呈现如此繁华的景象。
<br/>　　慕清霄与小医仙立于虚空，由于没有斗气化翼，瞬间吸引住了一片片目光，目光中多了一丝向往与敬畏。
<br/>　　因为，不用斗气化翼，想要立于虚空中，唯有斗宗才行。
<br/>　　望着天空上的飞行魔兽，小医仙美眸中闪烁着兴奋，道：“不愧是方圆千里内最繁华的城市。”
<br/>　　“我们也进去吧。”
<br/>　　微微点头，斗气大陆确实独到之处，最起码人族居住的城市，比他见过的任何城市都繁华。
<br/>　　好比面前的天涯城，虽然比不了某些仙门福地，却比世俗中的京城要繁华许多倍。
<br/>　　小医仙闻言，便调动灵气跟在慕清霄身边，化作一抹流光向山脉中掠去。
<br/>　　约莫十分钟后，一座巨大的城市轮廓就隐隐出现在葱郁的山峰中，随着距离拉近，城市的规模也彻底映入眼帘中。
<br/>　　就在慕清霄与小医仙要掠过城门口时，一道光影便从城门口掠至上空，旋即对面就出现一名黄袍老者。
<br/>　　老者面色严肃，身后斗气双翼微微振动，诧异的瞥了眼小医仙，他能感觉到，面前的小女娃根本就是斗灵，为何能够立于虚空？
<br/>　　当目光扫向慕清霄的时候，眼底更是困惑，青年除了容貌与气质不凡，根本察觉不到任何的气息，仿佛是个普通人。
<br/>　　随后，老者的目光被慕清霄肩上的绚丽的七彩小蛇给吸引住。
<br/>　　似乎是察觉到了老者的目光，彩鳞微微睁开眸子，金色蛇瞳瞥了一眼后者，旋即便微微闭目。
<br/>　　老者浑身一震，骤然色变。
<br/>　　仅仅只是被这条小蛇瞥了一眼，他就有种汗毛炸立的感觉，在这种感觉下，连体内的斗气流转都变得缓慢起来。
<br/>　　“难道是宠物？将斗宗级别的魔兽当宠物！”
<br/>　　黄袍老者心中惊骇不已，但七彩小蛇卷缩在慕清霄肩膀上的模样，似乎就是宠物啊！
