　　这一剑之下，将这青衣男子的所有虚影，都给尽数毁灭，甚至与是他的本体，都被逼了出来。
<br/>　　青衣男子后退连连，目光变幻这，随后冷哼一声道。
<br/>　　“好，小子，这一次算是我认栽了！走！”
<br/>　　一边说着，一边这青衣男子便是身形一动，直接转身，化为一道遁光，打算离开。
<br/>　　“逃？”
<br/>　　楚风眠嘴角冷笑一声，猛然之间大手一挥，一道巨大无比的战龙之爪，便是从天而降，冲着那青衣男子，笼罩下来。
<br/>　　“小子，你什么意思？难道你还真的敢杀我？”
<br/>　　青衣男子脸上尽是怒意的开口道。
<br/>　　“我是天剑宗弟子，你敢杀我？”
<br/>　　“天剑宗？七大宗门？在这个九域，还没有我楚风眠不敢杀的人！”
<br/>　　楚风眠冷笑一声，毫不犹豫，他的战龙之爪顿时落下。
<br/>　　这青衣男子的身躯，在战龙之爪下，顿时开始扭曲起来，只剩下一声声的惨叫。
<br/>　　“轰！”
<br/>　　只见他的身形，便是直接化为了一团血雾，他的空戒，还有他的灵剑，全部都被楚风眠直接收走。
<br/>　　这青衣男子的实力，虽然不如冷面魔君等人，却也是比起北域百绝榜上的大部分人，要强大的多了。
<br/>　　这种实力，放在北域去，足有争夺前三，也怪不得他敢如此狂妄，打算要抢楚风眠手中的无上剑气。
<br/>　　不过如今他的狂妄，也是令他付出了代价，在楚风眠的手中，这青衣男子几乎是没有任何挣扎的能力，如今便是被楚风眠当场轰杀，甚至连一个渣，都没有留下。
<br/>　　“七大宗门，果然富有，这一把灵剑，已经是地级灵器中最为极品的了！”
<br/>　　楚风眠一手抓过那青衣男子的灵剑，打量一眼，便是心神一动。
<br/>　　黑玄剑阵，被直接呼唤出来，楚风眠手一扔，这灵剑便是落入到黑玄剑阵中去，被一股龙炎所灼烧，化为了无数微粒，全部融入到了黑玄剑阵之中去。
<br/>　　同时这青衣男子的空戒之中，也是有着不少的灵药宝物，显然也是他从归墟天中，收敛到的，如今也全部都被楚风眠收取走。
<br/>　　“这杀人，果然是比起自己辛苦收取宝藏，要快得多。”
<br/>　　看着空戒中的无数宝藏，楚风眠不由的感叹一声。
<br/>　　在这归墟天中，人杀人，才是最好的选择。
<br/>　　不过楚风眠心中有着自己的规矩，他从来不是滥杀无辜之人，这青衣男子，要不是想要出手来抢夺楚风眠手中的无上剑气的，他也不至于会落到一个被楚风眠斩杀的下场。
<br/>　　“这周围星球，已经都被收取了不少了，要去远一点的地方了。”
<br/>　　楚风眠打量这周围的星球，突然他的目光，看向了远处，在这远方，有着无数岩石所在，这一块块巨大无比的岩石，正是漂浮在了空中。
<br/>　　在这无数岩石之中，楚风眠突然有着一种熟悉的感觉，这一种感觉，就跟那无上剑气，极为相似。
<br/>　　在这归墟天中，果然还有着其他无上剑气的存在。
<br/>　　“走！”
<br/>　　楚风眠身形一动，直接化为一道遁光，向着那无数岩石的方向飞去。
<br/>　　而在另一边，就在这归墟天另外一处区域上，三尊武者，正联合起来，围杀一只巨大无比的远古异兽。
<br/>　　随着无数武技打出，一声悲鸣声响起，这一只远古异兽，终于倒地。
<br/>　　“剑师兄，这只太皇蛇终于死了，他守护的，居然是一枚圣药，看来我们真是运气不错。”
<br/>　　一尊武者上千，就在这太皇蛇的身边，取走了一枚雪莲。
<br/>　　这雪莲，看似平常，但是上面的药力，却是浓厚到了吓人的地步，已经是超脱了灵药的范畴，是一枚圣药。
<br/>　　“这归墟天，真是到处都是宝物。”
<br/>　　又一尊武者，不由的开口道。
<br/>　　圣药，放在九域中，都足以称得上是难得一见的东西了，听说过的人不少，但是真正见到过圣药的武者，却是寥寥无几。
<br/>　　而在这归墟天，他们第一天便是碰到了这样一条太皇蛇，斩杀之后便是得到了一枚圣药。
<br/>　　“圣药，在这归墟天中，算不得什么，这归墟天的真正宝藏，还是那些远古时代宗门的遗址，在里面很有一些宝库，这些宝库，才是真正的宝藏。”
<br/>　　那名为首被称为剑师兄的男子，冷冷的开口道。
<br/>　　“将这古皇蛇有用的东西带走，我们直接前去归墟大陆，不要在这外围纠缠了，对了吕师弟如今在哪里？怎么还不过来？是不是遭遇了什么不测？”
<br/>　　“剑师兄不要这么担心，在这归墟天中，谁敢跟我们天剑宗作对，吕师弟传送进来的地方，应该离这里有点远，所以还没有赶过来，我这就问问。”
<br/>　　一尊武者开口道，便是从空戒中，拿出了一道玉符，一股灵力注入其中。

<br/>　　咔嚓一声。
<br/>　　出现了无数裂痕，随后下一刻，直接破碎了。

<br/>　　另外三人的目光，也是一下子看了过来，看到这玉符破碎的一刻，都是满眼不可置信。

<br/>　　那另外两名武者，都是不可置信的开口道。

<br/>　　看到那青衣男子，被楚风眠直接斩杀，剩下的几人，都是面露怒色。
<br/>　　那为首的剑师兄，更是眼神中杀意到了极点。
<br/>　　“好一个小子，不将我们天剑宗放在眼里？还敢杀我们天剑宗的弟子！我要他死！”
<br/>　　突然之间。
<br/>　　这剑师兄猛然拔剑，他的一把剑，直接向着虚空之中刺了过去。
<br/>　　看似的无端端的一剑。
