　　也就是说，小九现在的所作所为，完全毁掉了她一身的修为，最最关键的是，她身上受的两处伤都是致命的，如果没有内丹护体，她必死无疑……
<br/>　　“她怎么敢这样！”黎山老母大叫起来，“为了叶少阳，值得吗！！”
<br/>　　摇光仙子望着小九，突然懂了她心中的感觉，喃喃道：“或许，是值得的吧。”
<br/>　　碎裂的内丹，瞬间迸发出了强大的灵力，挡住了李浩然这一剑。
<br/>　　一向宠辱不惊的李浩然，也是被震惊了，傻傻地看着小九，道：“为何如此？”
<br/>　　“没有为什么。我喜欢他。”小九虚弱地说道。
<br/>　　“我喜欢他”四个字，仿佛一记记重锤，敲打在叶少阳的心头。但是被小九提着的他，什么都做不了，只能默默流泪，这一刻，他深切地感到，自己是那么的无能……
<br/>　　“转世鬼童，不是他心上人吗？”李浩然自然也知道叶少阳和芮冷玉的关系。
<br/>　　“那有怎么样，我喜欢他，只是喜欢他。”
<br/>　　我喜欢他，跟他有什么关系，跟他喜欢什么人有什么关系？
<br/>　　李浩然不动，怔怔地看着她，“我为证道，轮回十世，三世为僧，七世为道，万事万物了然于胸，也生有一女，却对男女之事不甚了解，无非是修道路上的业障。你业障蒙眼，也怪不得别人。”
<br/>　　“就算是业障，我也认了。”
<br/>　　“那便是好，我成全你。”
<br/>　　李浩然目光冰冷起来，手中剑不断施压，对着小九内丹所化形成的结界挤压过去。
<br/>　　小九努力维持着结界，体内的血，更加快速地从伤口流出来，顺着铁塔的表面滑下去，湿润了风尘千年的铸铁。
<br/>　　“少阳，我走了……”
<br/>　　小九看了叶少阳一眼，将叶少阳丢在了地上。
<br/>　　情到深处人孤独，缘来缘去总伶仃，魂飞魄散无处去，千年等待一场空……
<br/>　　小九脑海中浮现出了智深法师给出的这四句箴言。
<br/>　　原来……是应在眼下这个时候了。
<br/>　　终归是做出选择了吧。
<br/>　　小九心中没有难过，没有挣扎，反而只有轻松——在智深禅师给她说过那四句箴言之后，她一直惴惴不安，不知道抉择什么时候到来。
<br/>　　她预感到，一切会在眼下这次行动中发生，但没有想过，一切来的这么快。
<br/>　　“小九！”
<br/>　　叶少阳刚要起身去救她，李浩然的剑堪破了内丹形成的结界，刺进了小九的心口，轻轻的一下，然后就拔出来。
<br/>　　剑已经刺破了心脏，妖跟人一样，心脏破碎，肉身必死。他只是觉得小九很执着也很悲情，他想成全她，但不想让她就这么元神俱灭，说到底，还是跟他真没什么仇怨瓜葛。
<br/>　　剑拔出后，噗的一声，又喷出了一股血。
<br/>　　小九身子晃了两晃，差一点坠落，低头望着叶少阳。叶少阳飞身而起，爬上宝塔顶端，将她抱起来，抱在怀里，傻傻地凝视着她，泪水落在她脸上，又顺着她的脸颊流下去。
<br/>　　“告诉我，怎么样才能救你，要怎么样？”
<br/>　　小九摇摇头，抓住叶少阳一只手，气喘吁吁地说道：“你快走。”
<br/>　　“我带你一起走！”叶少阳想要抱起她，突然发现一丝异常，低头看去，顿时变眼前的一幕震惊了：
<br/>　　铁塔正在融化！
<br/>　　小九的血流在铁塔上，仿佛是某种强酸，一点点将铁塔融化，先是表层，然后一点点融化下去。
<br/>　　所有人再次震惊了，一个个睁大眼睛看着这不可思议的一幕。
<br/>　　当铁塔融化到一定程度之后，刻在四个面上的那些奇怪的符文，开始明亮起来，然后，因为融化了太多的缘故，整个铁塔最后不堪重负，崩塌下去。叶少阳抱着小九，也坠落在地上，好在叶少阳有所准备，用法力减缓了坠落的压力，两人平安降落。
<br/>　　叶少阳先查看小九的伤势，发现已经不流血了，心头感到一丝宽慰个，跟着小九一起抬头看去：
<br/>　　铁塔融化之后，留下了一个根基，比铁塔原本小了几号，有两个人左右高，形状还是像一座塔，通体漆黑，在最上面一圈，闪烁着一些金色的符号。正是之前塔上的那些没人能看懂的文字。
<br/>　　“妖族文字……”小九望着这些符号，喃喃说道。
<br/>　　“什么？”
<br/>　　“这是失落的妖族文字……”
<br/>　　叶少阳怔住，问道：“不会吧，不是所有人都看过，都不知道这是什么，怎么你一眼就看出来了？”
<br/>　　“这是刻在我记忆里的……只有传承的妖王才会知道，我一出生，就懂这种文字，除了我，没人知道的。”
<br/>　　原来如此……叶少阳猛然想到，小九之前一直被困在人间，有上千年，这座宝塔被发现的时候，她并没有前来看过，青丘山还因为没有她这个主人，失去了这个与东皇钟有关的奇怪宝塔。
<br/>　　世上只有她一个人能看懂，但她从没来过……
<br/>　　“这上面写的什么？”
<br/>　　“你抱我起来，我去看看另外几个符号。”
<br/>　　叶少阳听她这么说，感觉她对这东西还有好奇，似乎也没什么事，心里松了一口气，将她从地上抱起来，看了一眼芮冷玉，芮冷玉已经被小白和雪琪抱了起来，因为贴了三张定魂符，直挺挺地躺在两人怀里，一动不动，叶少阳这才放心。
<br/>　　一时间所有战斗都停止了，所有人都在看着这个奇怪的东西，没有人来管叶少阳做什么，甚至还带着期待。
<br/>　　大家都想知道这是什么东西。
<br/>　　就连李浩然也暂时放弃了进攻叶少阳的打算，静静地站在一边等待着。
<br/>　　叶少阳抱着小九，绕着这个奇怪的东西走了一圈，将四周的妖族文字看了个遍，见小九怔怔发呆，叶少阳也上下打量这个奇怪的东西，吸了口气说道：“我怎么觉得这东西的造型像一口钟呢？”
<br/>　　“东皇钟。”
<br/>　　“什么？”叶少阳没听清。
<br/>　　“东皇钟……”小九的神色也很激动，因为失血过多而发白的脸庞变得红润起来，“我也没想到，东皇钟居然在这里……”
