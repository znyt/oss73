　　元始如今所有的大道法则相加，法力暴增，实力暴增，长枪轰下，简直拥有破灭天地，粉碎苍穹的无上伟力！
<br/>　　黑暗魔神倒吸一口气，他知道自己遇上狠茬子了，这是闻所未见的大敌，要是他没有受伤之前，遇到此人，他还不至于放在心上，但如今，他有可能会饮恨在次。
<br/>　　他挥动手中的黑暗天幕，化作一道惊天利刃，连带着一道道法则秩序，轰然斩下，恐怖无边。
<br/>　　这绝对是可以割碎虚空的恐怖威力，但此时遇到元始诸多法则汇聚而成的紫金长枪，瞬间就被湮灭，好似一个小泡似的，一戳就破！
<br/>　　“这怎么可能？！”
<br/>　　一道充满不可置信的惊恐之声响起，却见那黑暗魔神眼中充满不可置信。但紧接着，他却冲天而起，想要逃离此地。
<br/>　　他如今实力十不足三，最强大的法宝黑暗天幕化剑居然对敌人无用，面前这人不好对付，越打越让人吃惊！
<br/>　　他可是混沌魔神，好不容易躲过了混沌劫，以后要证大道不朽的存在，怎么能死在这里？至于眼前此人，就先留着，等以后实力恢复了，再来报仇。
<br/>　　然而，元始却不会给他再来一次的机会，诸多大道汇聚而成的诛神长枪继续轰下，神威凛凛。
<br/>　　黑暗魔神大急，大手一撕，虚空划开，就要逃走！
<br/>　　“哪有那么容易的事！”
<br/>　　元始冷笑一声，长枪继续轰下，扭曲时空，洞破虚空，黑暗魔神只感觉汗毛颤栗，心脏抖动，一股巨大的危机笼罩其身，仿佛下一刻就会被灭亡！
<br/>　　轰隆隆！
<br/>　　一声巨响，长枪毫不留情地洞穿黑暗魔神真身，力量在瞬间爆发，道光流转，魔神惨叫连连，真身瞬间破碎。
<br/>　　黑暗魔神，陨落于元始之手！
<br/>　　他全身被诛神印洞穿爆炸，化作一片黑暗。
<br/>　　生于黑暗，也葬于黑暗！
<br/>　　下一刻，元始巨手向虚空一抓，顿时，手掌之中，出现一道散发着黑暗道光的本源光团，道韵流转，玄之又玄！
<br/>　　正是黑暗魔神的黑暗大道法则本源！
<br/>　　“这可是好东西啊，有了这团黑暗本源，领悟黑暗法则就容易多了！”元始淡淡的声音响起。
<br/>　　“嗯？还有这个！”元始大手向虚空一抓，顿时，一道叫做黑暗天幕的黑布出现在元始手中。
<br/>　　“黑暗天幕吗？先天至宝级别，还算不错。”一道淡淡的声音响起。
<br/>　　元始收起战利品，抬头看向上空，他能感受到，此行的最终目的，不周之巅，就快要到了。
<br/>　　不再多话，元始再次踏步走向不周之巅的路，至于黑暗法则，黑暗本源都到手了，还怕不能领悟吗？只是时间的多少罢了。与其在此地浪费时间，不如先登上不周之巅。
<br/>　　……
<br/>　　越是往上，威压越是沉重，好几次元始都被盘古这惊人的意志震慑，不得不停下来稍作休整，待适应了此处威压后，才继续前进。面对如此威压，就算元始意志再如何惊人，也逐渐开始变得不耐烦了。
<br/>　　好在，就在此时，元始竟然在仰望之中，看到了不周山顶峰的存在！
<br/>　　霎时间，那一股原本充斥天地之间的盘古威压猛增百倍，压迫元始的身心，进入一个玄之又玄，难以名状的怪异境界。
<br/>　　不周之巅，并非常规模样的锥形，而是隐隐呈平台状，像是一个人的两只手掌高举叠合在一起，支撑着整片天幕！
<br/>　　元始虽然还未登顶，但是从此处向上看去，已不由自主地生出一种整片天地都向自己碾压下来的感觉，一时不由心神震撼！
<br/>　　沉浸在不可名状的玄妙境界之中，元始双眼紧紧盯住不周天柱的顶部，脑海之中掀起无边波澜，浩瀚气息奔腾如浪。但是，他并没有感到不妥，反而感受到一股亲切的气息。那是盘古大神的气息，在冲击他的心神。
<br/>　　时间，就在无声无息之中流逝，不知过了多少春秋，终于，元始的意识清醒了过来。随后，他一步一步，缓缓走上不周之顶，霎时间，元始从身心到灵魂感受到一种神清气爽，顿时，整个人变得清爽无比，神透意明。
<br/>　　恍惚之间，他来到一个浩瀚无垠的混沌世界，看到一个身高千万丈的大汉，提着一柄巨斧，劈向混沌。混沌剧烈抖动，轰然作响，化作地水火风肆掠，三千混沌魔神大战，盘古霸道开天，清者上升，为天，浊者下降，为地，万物造化，天地始成。
<br/>　　正在此时，天道偷袭，盘古大笑，挥斧斩天，之后洪荒满目疮痍。盘古无奈，以手撑天，天日高一丈，地日厚一丈，十二万九千六百年，即一元会之后，盘古身化万物，造化乾坤……
<br/>　　元始心中震撼，恍惚明白过来，这是开天印记。他本以为这开天印记只有盘古三清与十二祖巫有，却没想到，此地竟然还有开天印记。他身为盘古元神所化，本身就有开天印记，现在再次得到一份，不过是让他大道感悟更深罢了。
<br/>　　不过，这一份开天印记也与之前他所得到的不同，它多了天道偷袭，盘古斩天的那一部分，现在这份开天印记更为完整。而且，这一份开天印记之中，是以盘古的角度看开天，所有领悟，都为盘古的领悟，若是元始能够完全吸收，他将更进一步。盘古最后虽然失败了，但毕竟道行、肉身也突破到了不朽境界，只差最后一步罢了。
<br/>　　“盘古，没想到你最后竟然到了那个境界！你比我们三千魔神都强，不仅仅只是法力境界强大，而且还有其他。比如，身化万物，这是何等大爱！你完全不必如此，但你终究还是做了。
<br/>　　不过，你最终还是失败了。我元始虽然之前不如你，但，我毕竟是还活着，而你却已经死了。我将有无限可能，我就不信不能超越你。不过，盘古，你真的死了吗？”
<br/>　　元始看到这份开天印记后，就知道，自己当年能够与盘古拼杀如此久已是非常幸运了，但他并没有气馁，盘古是强，但并非不可超越，元始对自己非常有信心，将来超越盘古。
<br/>　　“天道，天道，天道么？”
<br/>　　元始抬头看了看天，便不再多言。沉默了片刻，便开始在这不周之巅观察起来了。毕竟，不周之顶才是他此行的目的，其他只是顺带的。
<br/>　　四下张望，不周山之顶一片寂静，并没有任何生灵存在，想找那传说中的十二祖巫，可是找了半天，什么也没有找到，不仅如此，连一件灵宝都没有见到。
<br/>　　元始并不满意，当下，在不周之顶一番寻找，别说，还真让他找到两株初生的幼苗，他心念一动，便已知道，这应该是后世鼎鼎有名的不周山两大灵根——先天葫芦藤与先天芭蕉树了！
