        而诺筱颖的单子也接连不断，短短的一个月时间里，她的提成加底薪一起，她拿到了一万块钱。
<br/>
<br/>        这大概是她实习期以来，拿得最多的一份薪水。
<br/>
<br/>        有了钱，诺筱颖想的第一件事情就是还钱给傅昼景。下班后回到公寓里，她便坐电梯直接去了顶楼傅昼景的家。
<br/>
<br/>        幸好同住在一栋公寓里，不然她都不知道要上哪里去找他。
<br/>
<br/>        诺筱颖按响了傅昼景家的大门，却无人开门，她只好坐在一旁的阶梯上等傅昼景回来。
<br/>
<br/>        等了半会儿，傅昼景还未回来，诺筱颖觉得是时候该回去做饭了，于是起身朝电梯那走去。
<br/>
<br/>        她刚按电梯，电梯门便缩进了两边的门框中，傅昼景一袭黑色西装革履，出现在电梯里。
<br/>
<br/>        两人打了个正面，诺筱颖微微一笑：“你终于回来了。”
<br/>
<br/>        “你在等我？”傅昼景看着诺筱颖先是一怔，然后从电梯里走了出来。
<br/>
<br/>        诺筱颖点了点头，忙从包包里掏出一个鼓得饱满的牛皮纸信封，双手握着朝傅昼景递了过去：“这里面有五千块，我先还你一半。”
<br/>
<br/>        “这么快就能还上一半了？”
<br/>
<br/>        “是地利置业的业绩不错，我每天都有单子可以做，而每个单子的提成也比较高。”诺筱颖的脸上露出充实的微笑。
<br/>
<br/>        傅昼景会心一笑，接过信封，直接塞进了自己的公文包。
<br/>
<br/>        “你不数数看吗？”诺筱颖反问。
<br/>
<br/>        傅昼景笑着回答：“若是差钱了，你不怕我来找你麻烦吗？”
<br/>
<br/>        “那也是。”诺筱颖噗嗤一笑。
<br/>
<br/>        “今晚到我家吃晚饭。”
<br/>
<br/>        “不了，我还得回家去给我哥哥嫂嫂做晚饭。”诺筱颖摇摇手，然后按了下电梯。
<br/>
<br/>        趁着电梯还未被其他人按走，诺筱颖见电梯门开了后，连忙走进电梯。
<br/>
<br/>        傅昼景看着诺筱颖，微笑着说：“以后，多上来坐坐。”
<br/>
<br/>        “嗯，有时间，我就上来。拜拜——”诺筱颖挥着手回答。
<br/>
<br/>        电梯门关上后，傅昼景才想起一件事情想问问诺筱颖，可惜电梯已经下去了。
<br/>
<br/>        他本来想问问她和他的助理赵斌是怎么一回事，为什么姚明娜会说他两在谈恋爱。
<br/>
<br/>        或许下次再问也不迟，反正都在一个公司里。再或许，他可以直接去问赵斌。
<br/>
<br/>        傅昼景无奈地笑了笑，他也不知道自己怎么突然会想到要问诺筱颖这个问题。起初在公司里听姚明娜说的时候，就不在意，这会儿怎么又在意了？
<br/>
<br/>        算了，还是不问了，应该只是一些流言蜚语而已。
<br/>
<br/>        傅昼景想了想，于是转身回到了自己的家中。
<br/>
<br/>        这边，诺筱颖回到家后便开始淘米煮饭，哥哥嫂嫂也差不多在这个点要下班回来了。
<br/>
<br/>        饭菜都端上了桌，大门传来开锁声。
<br/>
<br/>        诺筱颖解开身上的围裙，将碗筷摆好。
<br/>
<br/>        诺司峻和唐倩兮有说有笑地走了进来。
<br/>
<br/>        诺筱颖站在餐桌前盛饭，见哥哥和嫂嫂两人脸上笑容灿烂，便微笑着问道：“哥哥、嫂嫂，今天，你们是有什么喜事吗？”
<br/>
<br/>        “当然有！你哥哥升职了！从小编辑升到了主编的位置，工资也跟着翻了一倍！”唐倩兮笑不拢嘴地说。
<br/>
<br/>        诺司峻也高兴不已道：“今晚，我要喝酒！不醉不归！”
<br/>
<br/>        诺筱颖跟着非常高兴地祝贺道：“哥哥，恭喜恭喜！”
<br/>
<br/>        随后，唐倩兮便去酒柜上取来了红酒。
<br/>
<br/>        三人一起吃饭，一起碰杯。
<br/>
<br/>        唐倩兮看了诺筱颖一眼，忽然意味深长地问道：“筱颍，你真的不打算去夜魅娱乐城兼职了吗？”
<br/>
<br/>        自从那晚，诺筱颖以艺名“小谢”的身份在她那儿驻唱了后，很多客人都对她印象深刻，都想再听到她的歌声。
<br/>
<br/>        唐倩兮一点都不想否认，诺筱颖的歌声真的非常好听，说她的声音是天籁之声也一点都不为过。
<br/>
<br/>        其实，上次之后，诺筱颖跟她说不唱了，她也没觉得什么，只是这两天，她的上司费尔南多次开口，让她去求她的这小姑子诺筱颖回来继续驻唱，费尔南还提出要给诺筱颖加驻唱的提成费。
<br/>
<br/>        如果，诺筱颖同意回来驻唱的话，她唐倩兮也能从中拿到分红，毕竟，诺筱颖是她带入夜魅娱乐城的。
<br/>
<br/>        “我不去了，那种地方，真不适合我，而且，公司里给我安排的工作，暂时足够我开销了。”诺筱颖心满意足地说。
<br/>
<br/>        唐倩兮眼珠子一转，微笑着劝说道：“钱是永远都赚不够的啊！筱颍，你以后就不想留在我们滨江城发展吗？还是说，你只是打算混个实习期，毕业后索性回你们那个西镇去种田？水往低处流，但人总得往高处走。再说了，你若是在夜魅娱乐城里还有我罩着，你怕什么？”
<br/>
<br/>        “嗯，我觉得倩兮说得对，既然，你在夜魅娱乐城里驻唱一晚能赚一千块钱，为何不继续驻唱呢？反正，你只是卖艺不卖身，有你嫂嫂罩着你。你不用担心。”诺司峻也赞同唐倩兮的话。
<br/>
<br/>        他俩一人一句，说得诺筱颖心里有所动容了。
<br/>
<br/>        她，诺筱颖，农村出身，没家世没背景，又是大专文凭。
<br/>
<br/>        想要在滨江城混下去，就得赚钱买套房，总不能一辈子都和哥哥嫂嫂住一块吧！
<br/>
<br/>        “那……我可不可以每周只驻唱两次？就星期五的晚上和星期六的晚上？公司里每周星期六星和期天放假，所以，我不想影响自己白天的工作。”诺筱颖看着唐倩兮，弱弱地问道，毕竟，在那儿驻唱，要驻唱到凌晨两三点。
<br/>
<br/>        唐倩兮见她都已经答应了，立即做出了让步，并允诺道：“这绝对没问题！”
<br/>
<br/>        “那倩兮，我就把我妹妹交给你了啊！你一定要保护好我妹妹！别让她在你那夜魅娱乐城里被坏男人给糟蹋了！”诺司峻一脸严肃地说。
<br/>
<br/>        唐倩兮微微醉，笑道：“筱颍难道就不是我妹妹呢？我自家的妹妹，我不保护着她，那我还保护谁去？”
<br/>
<br/>        诺筱颖欣慰地微微一笑，有哥哥和嫂嫂护着真好。
<br/>
<br/>        晚饭后，诺司峻和唐倩兮虽然喝多了，但是也没有完全醉得不省人事。
