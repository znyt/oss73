　　寺庙内的空间并不大，除了一座面容怪异的佛像还算完整之外，其他的物件都是残破不堪。
<br/>　　怎么看，都没有什么地方可以藏人，这个突然出现的老僧，出现的着实过于诡异。
<br/>　　警惕的看着老僧，刘哔将体内的灵气聚集在手中，他不太清楚这僧人的意图。
<br/>　　“施主，不要动这盏烛台，万一弄灭了，可没办法让它再燃烧起来。”双手合十，老僧扫去脸上的阴沉，一脸祥和的说道。
<br/>　　对于刘哔身后不断忽闪的一对翅膀，这僧人似乎并不感到意外。
<br/>　　“你，是人是鬼？”平复了一下心情，刘哔试探的问道。
<br/>　　“人又如何，鬼又如何，都是一物罢了，施主既能看到此庙，大概也是已死之人吧。”老僧的手并没有放下，依旧合在胸前，说出的话，却让刘哔头皮一麻。
<br/>　　这个僧人，知道自己不是活人！
<br/>　　而且这个寺庙，似乎也有些诡异，从老僧的话中理解，似乎活人，是看不到这座庙的。
<br/>　　那这样说来，这是一座鬼庙，能进来这里的，都是死人，包括，这名老僧！
<br/>　　“阿弥陀佛，老衲虽然已经坐化，却从未有过害人之心，施主可放心在此躲避风雨。”念了一声佛号，老僧慈眉善目，倒有些佛教高僧的风范。
<br/>　　“只是不害人吗，但如果是别的东西呢。”挑了挑眉，刘哔不置可否的一笑，继续道：“麻烦你下次和别人说话的时候，先把牙齿清理干净，上面还带着肉丝呢。”
<br/>　　从老僧刚开口的时候，他就闻到了一股淡淡的血腥味，和猪马羊狗不同，这股血腥味中，带着一股专属于人肉的香气。
<br/>　　“哦，这是我的晚饭，不是人。”顿了一下，老僧解释道：“这是一个赶路的官兵，被一个修炼了百年的女鬼追赶，阴差阳错的进入了庙里，我看他可怜，就把他吃了。”
<br/>　　抽动了一下嘴角，刘哔没多说什么。
<br/>　　似乎是知道他心中的想法，老僧解释道：“活人进了这里，就会一直被禁锢在庙中，再也无法走出，失去自由的人，你说，还能算人吗？”
<br/>　　不可置否的撇了撇嘴，刘哔莫名有些烦躁，从戒指中取出一根龙头拐杖，看着老僧的目光有些冰冷。
<br/>　　“你留不下我。”话落音，一股强横的气势从他体内爆发了出来。
<br/>　　淡然一笑，老僧念了一句佛号，身上闪起一片金光，一股比刘哔强横了不知多少倍的气势，向他反压了过来。
<br/>　　置身于这股重逾万斤的气场中，刘哔心中有些骇然，这个老僧实力之强，超乎了他的想象。
<br/>　　甚至他有一种感觉，如果对方想要对他动手，他根本没有半点胜算。
<br/>　　“阿弥陀佛，施主脾气不要这么暴躁，老僧，从未有过害人之心，施主若不相信，大可离去。”一反开始时的慈眉善目，僧人脸色变的有些狰狞。
<br/>　　借着身上璀璨的金光，刘哔依稀看到，其身上僧袍破损的地方，露出一片怪异的图案。
<br/>　　沉默了片刻，刘哔收起能量，对着僧人叹息一声，没有言语，转身走出了庙外。
<br/>　　他想起了一个传说，是很早之前看过的，没想到，在这个世界真的存在。
<br/>　　大概是十七年前的时候，他去庙里许愿，听一位和尚讲起过，但当时他并没有当真，认为只是一个凭空编造出来的故事。
<br/>　　据说，在佛教刚开始起源的时候，很多人都把它当做一个邪教，尤其是极北之地，所有僧人都被明令禁止踏入。
<br/>　　当时有一个年轻的僧人，怀揣着热血，想要将教义传到这个被他们称为‘蛮夷’的地方，却仿若鱼沉大海一般，彻底失去了踪迹。
<br/>　　所有人都认为他死了，毕竟从来没有僧人能从那种地方活着回来。
<br/>　　但就在其消失的三年后，这个年轻僧人，突然又出现在了自己先前所在的寺庙里。
<br/>　　这个消息，无疑引起了很多佛教信徒的兴趣，皆是络绎不绝的来拜访年轻僧人，想要知道他在那里经历了什么。
<br/>　　年轻僧人什么也没说，事实上从回来之后，他就再也没开过口，只是有同寺庙的僧人发现，他身上多出了很多伤痕，细看下去，是一个怪异的图案。
<br/>　　这个图案，有人在极北之地见过，是那里人信奉的神灵。
<br/>　　后来，逐渐有流言传开，说年轻僧人背弃了信仰，信奉了邪灵，一些信徒要求佛教将他除名，认为他这样的人根本不配信奉佛教。
<br/>　　迫于压力，寺庙只能将其交出，狂热的佛教徒们，将年轻僧人的衣服脱光，用烙铁毁掉他身上的邪灵图案，并将其关押到牢狱之中。
<br/>　　年轻僧人还是一字未说，甚至连惨叫声都不曾发出，脸上的表情，始终都很平静。
<br/>　　当天夜里，那所关押他的牢狱发生了暴乱，犯人们都被放出，监视牢狱的守卫，都被人无情虐杀了。
<br/>　　等到官府赶到的时候，狱中已经没有一个活人，所有死去的尸体，都被规整的摆在大厅，摆成了两个字：自由。
<br/>　　年轻僧人并没有离开，他就盘坐在大厅的那些尸体当中，满脸祥和。
<br/>　　当官兵小心翼翼靠近他的时候，却发现年轻僧人早就已经死去多时了。
<br/>　　但是僧人已死，魂却没有离开，此后，世间便多了一座寺庙，寺庙有一个老僧，以人为食，活人进入，便再也不会从中走出来。
<br/>　　从寺庙中出来，外面的雨势已经逐渐小了下来，刘哔漫步在雨中，他倒是不怎么着急了。
<br/>　　那个无名寺庙，以及那个无名老僧，给了他太多感触。
<br/>　　信仰这种东西，每个人多少都会有一些，但当信仰背弃了你，那种感觉却不是寻常就能接受的。
<br/>　　说起来，这所寺庙会在这个世界存在，也让刘哔有些意外。
<br/>　　不过想想倒也正常，毕竟这个位面的背景，和他原本世界发生的事情也大致相同。
<br/>　　想到这里，刘哔发现，他可以借助这一点，去做很多事情。
<br/>　　雨逐渐开始停歇，走在泥泞的土路上，刘哔收起了自己的翅膀。
