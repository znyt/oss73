　　见慕清霄将罗刹鬼莲收起，天鬼皇有些尴尬，罗刹鬼莲这种好东西，连他自己都没有。
<br/>　　想与慕清霄讨要，但又不敢开口，刚才那令他心惊肉跳的黑色火焰虽然消失，可依旧历历在目。
<br/>　　将空间戒指装进储存空间中，慕清霄并没有将东西给天鬼皇的意思。
<br/>　　目光在四周扫视一圈，见没有任何东西遗留后，慕清霄淡淡道：“双煞鬼王已死，阴寒域是你的了，给我带路吧。”
<br/>　　“请阁下稍等片刻，待本座吩咐过犬子与小女，便给阁下引路。”
<br/>　　离开地宫，慕清霄将外界的空间结界撤销后，便取出空间戒指，盘坐在院落中的槐树下给物品分类。
<br/>　　将能够炼丹的灵药分为一类，鬼族修炼的灵药与鬼丹分为一类。
<br/>　　将东西分好后，慕清霄也不准备将时间白白浪费，五心向天，运转心法，开始炼化阴阳玉中的焚寂剑。
<br/>　　随着时间的推移，两日的时间一闪即逝，天鬼皇也去而复返。
<br/>　　鬼冥与鬼厄已死，天鬼皇第一时间出现震慑宵小，并且让天魑皇和魉妹两人执掌阴寒域的事宜。
<br/>　　某些忠心双煞鬼王，意图反抗的鬼族修士，也在第一时间被天鬼皇击杀。
<br/>　　“阁下，阴寒域的事宜已经处理好。”
<br/>　　不多时，天鬼皇便带着一辆拖着宽敞车厢，踩着青色冥火的骏马出现在慕清霄的眼前。
<br/>　　眼前的骏马与世俗中的马匹有些不同，四肢马蹄与头顶上都燃着青色的火焰，瞳孔深蓝……
<br/>　　青冥骏，便是鬼界的交通工具，唯有天潢贵胄有资格乘坐。
<br/>　　虽说有些赶时间，但慕清霄对鬼界的事物都抱有一丝的兴趣，于是进入车厢，与天鬼皇踏上前往火鬼界的路程。
<br/>　　青冥骏并非在路途上行走，而是能够御空而行，而且比御剑的金丹修士更加迅速。
<br/>　　坐在青冥骏拉着的车厢中，慕清霄感觉不到一丝的颠簸，但能听到外面马蹄踏空的闷沉声响。
<br/>　　随手一挥，马蹄声被隔绝在外，慕清霄则盘坐在车厢中闭目养神。
<br/>　　至于天鬼皇，则热情的讲述着他对火鬼界的了解。
<br/>　　毕竟，慕清霄可是帮了他的大忙，有了阴寒域，往后他与族人就不用待在酆都了。
<br/>　　想到此处，天鬼皇对慕清霄越发的敬畏。
<br/>　　“阁下，据本座所知，火鬼界有天然的岩浆湖作为屏障，暂且不说本王，连鬼王中最强的阎罗鬼王都无法进入……”
<br/>　　闻言，慕清霄纹丝不动。
<br/>　　“岩浆，雷电，自然界许多力量对你们的鬼体都有克制作用，你们进不去，不代表我也进不去，你只需要告知入口便可。”
<br/>　　话语刚落，天鬼皇陷入沉默。
<br/>　　不知过了多久，慕清霄睁开双眼，青冥骏也停止移动。
<br/>　　“阁下，我们已经到了。”
<br/>　　打开车厢，慕清霄目光扫视一圈，视线中除了几座光秃秃的山峰，就唯有不远处的一条无尽长河。
<br/>　　看到长河的一瞬间，慕清霄眼神有些恍惚，暗红色的河水中透着黝黑。
<br/>　　“你不会告诉我，这里是火鬼界吧？”
<br/>　　此时，天鬼皇来到慕清霄身旁，望着无尽的暗红色河水，眼中满是忌惮，最终苦笑着摇了摇头。
<br/>　　“这里并非火鬼界，但此河却是通往火鬼界的必经之路，此河被称为冥河，不知存在鬼界多少年，无人知晓此河的来历。”
<br/>　　说着，天鬼皇不禁回忆起，天鬼一族留下的典籍中有记载。
<br/>　　冥河乃是鬼界的禁地，曾经鬼界高层前往河底探查，甚至出动了鬼仙，但那一去就完全消失了踪迹，鬼仙也再也没有回来。
<br/>　　鬼仙可是鬼界中最强的存在，谁也不知道冥河中存在什么，只知道凶险至极。
<br/>　　微微一叹，天鬼皇严肃道：“冥河风平浪静，不会有危险，唯独不能下河，河水拥有极高的腐蚀性，哪怕神佛金身都能腐蚀，而且上面被设下了禁空的禁制，想要渡河，需要一样工具。”
<br/>　　望着暗红色的河水，慕清霄愣愣出神，眼前的冥河，不会就是传说中孕育了冥河教祖的冥河吧？
<br/>　　冥河老祖，又称冥河教祖，曾说血海不枯，冥河不死。
<br/>　　六道轮回附近，鸿蒙开辟以来，生成地狱黄泉，其中有幽冥血海。
<br/>　　在幽冥血海中，天生孕育了一个胎盘，后成为冥河教祖，有大神通，演阿修罗一族。
<br/>　　其生而伴有两剑，曰元屠、阿鼻。
<br/>　　胎盘化为十二品血莲，又称十二品业火红莲，与西方接引道人的十二品金莲台，还有魔祖罗睺的十二品黑莲，共为先天三十六品混沌莲台所化。
<br/>　　其实，冥河开始时是跟鸿钧学道，看女娲造人成圣之后，学习女娲造人创造了阿修罗一族，也想成圣，但后来也没成功。
<br/>　　在神话中，冥河与女娲等圣人乃是同一时期的人物，可唯有他成就最低。
<br/>　　脑海中闪电般划过一幕幕信息，回过神来，慕清霄微微摇头，不管是不是冥河，他现在的目的是火灵珠。
<br/>　　况且，就算是冥河，也不是现在的他能够接触的。
<br/>　　冥河河水能腐蚀神佛金身，慕清霄可不认为，他的肉身能够强过传说中的诸天神佛。
<br/>　　“工具？”
<br/>　　见慕清霄露出困惑的眼神，天鬼皇视线看向冥河，微微一笑道：“已经来了。”
<br/>　　不多时，冥河上就泛起一个小黑点，以慕清霄的眼睛自然能看清楚，冥河上的是一艘小舟。
<br/>　　小舟通体墨黑，在冥河中缓行，却没有遭到河水的腐蚀。
<br/>　　“这是用生长在冥河岸边的冥木制造的冥舟，天上地下，唯有此舟能够渡河，哪怕实力强如神佛都不行。”
<br/>　　闻言，慕清霄若有所思，冥河乃是盘古之血所化，上空的禁制恐怕也不是结界，而是威压。
<br/>　　“看来，在我来鬼界前，你在其中早有部署，难怪阴寒域能够如此快就落入你的手中。”
<br/>　　天鬼皇不语，他毕竟是鬼王，智商也不低，如果没有把握，他怎么敢带慕清霄来鬼界。
<br/>　　哪怕露出一点风声，他们天鬼一族都唯有死无葬身之地。
