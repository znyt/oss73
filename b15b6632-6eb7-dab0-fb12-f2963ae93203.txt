　　第91章
<br/>　　子曰：不患寡而患不均。
<br/>　　这句话以程大雷的理解便是：不怕我没有，只怕别人有的比我多。没有比较就没有幸福感，本来程大雷还对手中的冷面包、冻腊肉难以下咽，看到高飞豹和小白狼带的干粮后，竟忽然觉得这些东西可口起来。
<br/>　　高飞豹这次带的干粮只是高粱面，小白狼也没有好多少，无非一些杂粮陪着咸菜。
<br/>　　“弟兄们多喝些酒，这天是真冷啊。多吃肉，吃肉才扛寒呐。”
<br/>　　说着，就听到高飞豹和小白狼两方传来咕咚咽口水的声音。
<br/>　　程大雷一扭头，看到高飞豹眼睛巴巴的往这边看。
<br/>　　“二爷，过来一块儿吃点喝点儿？”
<br/>　　程大雷这话一落地，高飞豹手下喽啰就是一片躁动。
<br/>　　“哈，不必了，我们都吃饱了。”高飞豹狠狠的瞥了一眼手下弟兄。
<br/>　　“你可别和我客气，都是自家弟兄，你要是客气，平白显得咱们生份了。”
<br/>　　“程当家多虑了，说实话，我最近有些不消化，吃不了太腥的，想要吃些清淡的。”高飞豹这次是死要面子活受罪，他已经打定主意，如果程大雷再和自己客气，那自己也就不客气了，吃他娘的喝他娘的。
<br/>　　“那好吧，我也就不客气了。”
<br/>　　“……”高飞豹。
<br/>　　眼看着程大雷那边大块朵颐，大口喝酒，大口吃肉。高飞豹一伙只能干咽吐沫，吃些高粱面拌盐巴。
<br/>　　“人家程当家都请咱们过去，二当家非得死撑。”
<br/>　　“我已经有两三个月没吃过肉了……”
<br/>　　“看看人家吃得什么，看看咱们吃得，唉，真是货比货要丢，人比人要死。”
<br/>　　手下聚在一起，嘀嘀咕咕说个不停。高飞豹也是哑巴吃黄连，有苦说不出，自己只是谦让一下，怎么程大雷就真不客气了。
<br/>　　“吵什么吵，咱们就没带好东西嘛，把我包袱里的黄豆饼拿出来，给兄弟们分了。”
<br/>　　“嚯，二爷你还给自己开小灶呢。”
<br/>　　“废什么话，那是临出来前老娘给装包袱里的。”
<br/>　　高飞豹包袱里的黄豆饼被翻出来，一人分了一小口，算是堵住了手下的嘴。
<br/>　　“军师，把马也喂一下，一路上全靠它们出力。”程大雷窝在墙角道。
<br/>　　“嗯。”
<br/>　　徐神机从雪橇上拿出个大布口袋，把里面东西掰碎了塞到马嘴里。
<br/>　　高飞豹这边正嚼得起劲，忽然一怔，问道：“徐军师，你喂马吃得什么啊？”
<br/>　　“也就些干豆饼，出门在外不能和在家里一样，委屈这些畜生了。”
<br/>　　高飞豹无语了。
<br/>　　回过头手下弟兄都大眼瞪小眼的看着他，嘴里正嚼的东西，不知该咽下去还是该吐出来。
<br/>　　“这么好的东西都喂了畜生，蛤蟆寨还真是财大气粗。”
<br/>　　“这就是作孽！”高飞豹冷哼一声：“看吧，姓程的以后生儿子肯定没**！”
<br/>　　高飞豹也只能在心底暗暗咬牙。其实青牛山各山寨过的都是一样的日子，大家平时种地，年景不好的时候出去打劫，都是苦出身，谁也别笑话谁。
<br/>　　可偏偏出了个程大雷，大半年时光让一座山寨富得流油。现在这日子过得，人家吃肉，咱们喝粥，人家穿面，咱们穿单，如此，也难怪高飞豹心里不痛快。
<br/>　　“程当家，这次打算对那家动手，咱们先对一对，到时候便撞了买卖。”高飞豹。
<br/>　　小白狼也向这边投来目光。说是买卖，其实就是打家劫舍，只不过城中的富户就这么多。要是三家抢到一家，事情就尴尬了。
<br/>　　虽然，目前已经够尴尬了。
<br/>　　程大雷笑笑：“估计撞不到一起，不知道你们两家打算去抢劫谁？”
<br/>　　“朱家。”
<br/>　　“吕家。”
<br/>　　从高飞豹和小白狼口中分别道出两个字。程大雷来之前调查过黑石城的情况，知道他们口中说的两家。
<br/>　　都是城外的地主，家里养着壮丁，至于是不是为富不仁，其实也并不重要。身为山贼，大多是只劫富，不对穷人动手。这自然不是因为什么职业操守，关键是穷人没什么好抢的。
<br/>　　“那委实撞不了，这次咱们干的不是一趟买卖。”
<br/>　　“那程当家准备去抢谁？”
<br/>　　“实不相瞒，我还没有考虑好。”
<br/>　　“程当家，我们可是直言相告，你这样吞吞吐吐，该不会想抢的和我们一样吧。”高飞豹。
<br/>　　“朱家，吕家嘛……你们不觉得太小了些么？”
<br/>　　“那是哪家？”高飞豹立刻问。
<br/>　　“莫非是王家？”小白狼。
<br/>　　不管朱家还是吕家，都是城外的地主，真让小白狼和高飞豹进城，他们是不敢的。而这王家不同，是城里的大家族。
<br/>　　“王家……也小了些。”
<br/>　　“那是贾家？”小白狼难以置信道：“贾家可是黑石城第一世家，家族不少子弟都在军中任职。程当家，如果你想动贾家，我劝你还是要三思而行。”
<br/>　　小白狼这么说，足以证明贾家在黑石城的实力。这个时代的治安不怎样，许多大家族都养着守院的家丁。不乏一些江湖豪客走投无路，暂时在其中落脚，所以里面并不缺乏高手。
<br/>　　“贾家吗……也小了些。”
<br/>　　小白狼和高飞豹都无语了。贾家还小，难道程大雷竟想打劫城主府。
<br/>　　事到如今，高飞豹已经不想和程大雷说话了。
<br/>　　如此一夜，匆匆过去。
<br/>　　到第二天天没亮，高飞豹和小白狼已经派出人去。他们不会在白天动手，打劫这份工作，既是体力活，也是个技术活。今天只是派出人去踩踩盘子，等彻底摸清楚对方情况后，才会动手。
<br/>　　程大雷醒来后，也把人派了出去。
<br/>　　如今，庙里就只剩下程大雷、小白狼、高飞豹三个人。
<br/>　　小白狼从包袱里掏出三颗骰子，晃了晃道：“闲着没事，玩两把？”
<br/>　　“来来。”高飞豹立刻来了兴趣。
<br/>　　“有什么好玩的，输了你们有钱给嘛？”程大雷倒是对赌博不感兴趣。
<br/>　　“输了没钱赔，我就把我自己输给你。”
<br/>　　恶名远播的女土匪，可不是那绣楼中的大小姐，反倒是程大雷有些不好意思了。
<br/>　　“玩两把就玩两把，你要是赢了，我就把我输给你。”
<br/>　　
<br/>(本章完)
