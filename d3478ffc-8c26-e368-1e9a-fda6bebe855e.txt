    陈兴眯了眯眼睛，盯着炀智勇的三角眼，没有说话。这种话不能接，接了就会显得低人一等，但是不接，又会显得胆小怕事。<br/>    <br/>    两难之际，叶阳清燕忽然上前一步，挡在陈兴身前。<br/>    <br/>    “小矮子，我姐夫怎么样，跟你有什么关系！”<br/>    <br/>    只见她一手插着腰，一手指着炀智勇，柳眉倒竖，杏眼圆睁，大声呵斥。红发如火，朱唇似血，浑身透着一股泼辣劲。<br/>    <br/>    这声呵斥，好似天籁之音，听得陈兴浑身舒坦，差点儿忍不住冲上去，狠狠亲她几口。真是渴了有水喝，困了有枕头，这买大送小的生意，一点儿也不比那抗火符文的差。<br/>    <br/>    炀智勇脸上一阵红一阵白，握紧着双拳，憋了半天才说道，“我是和他说话，不是你。”叶阳家是灰角城第一家族，比他们炀家势大。他就是再狂，也不敢轻易得罪。然后看向陈兴，挑衅道，“我想和你比试比试，敢吗？”<br/>    <br/>    说话的同时，炀智勇重心下沉，拉开架势。<br/>    <br/>    陈兴见状，立即将叶阳清燕拉到身后。虽然他的大部分属性都低于对方，但保护女人是男人的天性，陈兴也不例外。<br/>    <br/>    “姐夫，揍他！”叶阳清燕扬着拳头，叫嚣道。<br/>    <br/>    叶阳白柳则眉头轻蹙，眼中闪过一丝担忧。她知道陈兴的“增幅手段”每隔一段时间才能用一次，刚才救小女孩的时候已经用掉了，现在打起来，肯定会吃亏。<br/>    <br/>    陈兴侧过身体，注视着炀智勇，弓身曲臂，蓄势待发。他现在是骑虎难下，打起来麻烦事多，退缩又丢面子。炀智勇的三角眼就像毒蛇一般，死死地盯着他，稍有破绽，就会攻过来。<br/>    <br/>    发现这边的争斗，参加宴会的人纷纷望过来，整个大厅都陷入了安静之中。<br/>    <br/>    一名身穿红色大衣，脚踏黑皮靴的中年男人推开人群，走了出来，正是兰花镇的镇长——夏德·辛克莱。<br/>    <br/>    看见两人的架势，镇长吹起了小胡子，“干什么干什么！”<br/>    <br/>    “年轻人，不好好参加宴会，打什么架！”镇长板着脸，教训道，然后朝围观的人群招了招手，“散了散了，都散了，各玩各的，该跳舞的跳舞，该喝酒的喝酒……”<br/>    <br/>    “一个边缘镇的小男爵，也敢来管小爷的事，找死啊！”炀智勇一声暴喝，打断了镇长的话。<br/>    <br/>    镇长浑身一震，缓缓地转过身，动作僵硬，仿佛一台生锈的机器人，所有的关节生涩无比。他脸上带着震惊的、不可置信的表情。<br/>    <br/>    “你，你……”或许是过于激动，镇长连形象都忘了，习惯性地拈着兰花指，颤抖着手，指着炀智勇问道，“你，你说什么？”<br/>    <br/>    “说你这个小男爵，多管闲事！”炀智勇仰着头，重复道。<br/>    <br/>    “什么，你说我多管闲事！”镇长气得直跳，“这里是我的领地，我还多管闲事了！”<br/>    <br/>    “领地又怎么样？”炀智勇面带讥讽，完全不把镇长放在眼里，忽然脸色一沉，冷喝道，“小心我回去投诉你，摘了你的头衔！”<br/>    <br/>    “黄口小儿！”镇长怒发冲冠，大步上前，“本镇的头衔岂是你说摘就摘的！”<br/>    <br/>    说话的同时，镇长身上的气势徒然暴涨，红衣鼓荡，猎猎作响。炀智勇脸色一白，接连后退。就在这时，黑袍老者化作一团黑影，挡在少主身前。<br/>    <br/>    “大胆夏德，胆敢犯我灰角城炀家！”<br/>    <br/>    黑袍老者厉声喝道，声音尖锐嘶哑，仿佛指甲刮着玻璃，听得人浑身不舒服。<br/>    <br/>    “炀家又如何！”镇长气势不减，沉声喝问，“是你们炀家大，还是王国法律大！”<br/>    <br/>    在龙涎河联合王国的法律中，领地贵族的头衔只有侯爵以上的公国国主能授予和收回。同时，法律也规定了领地贵族的权利和义务。<br/>    <br/>    领地之内，即为国王，神圣不可侵犯。<br/>    <br/>    炀智勇和黑袍老者同时一愣，都不敢接话。在王国法律面前，炀家不过是蝼蚁。而且他们身为门阀世家，是王国法律的既得利益者，决不会公然否认法律的权威。<br/>    <br/>    见两人被镇住，镇长微微一笑，从大衣口袋取出雪茄，点燃，美美地抽了一口，吐出一大团白烟。<br/>    <br/>    “加上我们乌鸦市穆尔家呢？”<br/>    <br/>    忽然之间，大厅角落里响起一个平静的声音。<br/>    <br/>    “加上我们灰角城洛家呢？”“加上我们灰角城张家呢？”<br/>    <br/>    两个声音接连响起，一个阴柔，一个粗犷。<br/>    <br/>    镇长的笑容凝在了脸上，雪茄“吧嗒”的一声，掉在地上，火星溅得到处都是。<br/>    <br/>    “大人，算了，算了……”人群里跑出一个皮肤白净的年轻人，抓着镇长的衣袖，拼命向后拉扯。<br/>    <br/>    “我，我……”镇长气得胡子发抖，全身僵硬，像个坏掉的机械人偶。<br/>    <br/>    “他们都是大人物，你只是个小男爵，斗不过他们的。”小伍一边劝说，一边将镇长拉出众人的视线。<br/>    <br/>    镇长走后，炀智勇冷笑一声，转头看向陈兴，伸手指着他，三角眼凶光毕露：“来战！”<br/>    <br/>    “战你个头啊！”一声清喝，透亮清脆。叶阳清燕双手抱胸，高声说道，“我姐夫为什么要和你打，滚开，要是嫌手痒，自己打沙包去！”<br/>    <br/>    “我们队长探索时受了伤，不方便比试，请回吧。”叶阳白柳走到陈兴身边，面无表情地说道。<br/>    <br/>    “打，也得打，不打，也得打。”黑袍老者身形一晃，逼了过来。陈兴只感觉压力扑面而来，头皮刺痛无比。忽然眼前白影闪过，压力顿减。定眼看去，三叔公挡在了前面。<br/>    <br/>    “叶老三，你是什么意思？”黑袍老者问道，声音嘶哑难听。<br/>    <br/>    “这位陈公子是我们叶阳家的贵客。”三叔公慢悠悠地说道，神态自若，口气却不容置疑。<br/>    <br/>    “陈公子？”黑袍老者疑惑道。<br/>    <br/>    “对，东大陆陈家，陈公子。”三叔公说道。<br/>    <br/>    话音刚落，除了事先知道的张猛和洛少卿，罗伊和炀智勇都是面露震惊。大概是想不到，陈兴和他们一样，都是门阀世家的子弟。<br/>    <br/>    “你说什么，东大陆陈家，哈哈哈哈哈哈哈！”黑袍老者忽然狂笑起来，仿佛听见了世界上最好笑的笑话。<br/>    <br/>    “黑老怪，你笑什么！”三叔公面带怒容，大喝道。<br/>    <br/>    “东大陆陈家，哈哈哈，东大陆陈家，哈哈哈……”黑袍老者旁若无人地笑了好一会儿，突然面露阴狠，凹陷的双眼闪着幽光，尖声说道，“陈姓的人很多，但东大陆只有一个陈家，不过……”他顿了顿，看向陈兴，目光阴森可怖，“二十年前延陵城破，早就灭了！”<br/>    <br/>    陈兴心脏狂跳不止，尽管脸上没有表现出来，但冷汗瞬间浃湿了背后。<br/>    <br/>    “你，你可有证据？”三叔公脸色一凝，有些不相信地问道。炀家的这位客卿，就是从东大陆来的。<br/>    <br/>    “延陵陈家的雷蛇手，他可会？”黑袍老者背过双手，仰起头，自信满满地问道。<br/>    <br/>    “陈公子……”三叔公的脸色变得有些难看，伸出手，做了个“请”的手势。意思很明确，让他展示家族武学。<br/>    <br/>    “他说的，他说的都是真的吗！”叶阳清燕瞪大着眼睛，揪着陈兴的衣角。眼波盈盈如水，倒映着沉静的面容。<br/>
