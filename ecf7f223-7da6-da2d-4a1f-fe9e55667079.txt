　　殿宇之外。
<br/>　　萧逸离开后，一位猎妖殿人员，带着他前往总殿宝库。
<br/>　　“你便是大名鼎鼎的易霄副殿主？”
<br/>　　宝库处，一位负责这里的主殿主，惊讶地看着萧逸。
<br/>　　“啧啧，老夫虽然不知晓你的面容，但万万没想到，竟只是一个如此年轻的天骄。”
<br/>　　萧逸看了眼这位主殿主，心头暗暗一惊。
<br/>　　在他的感知中，面前这位老者，其气息犹胜灸炎。
<br/>　　而且对比灸炎的气息稍微有些不稳，老者的气息，沉稳至极。
<br/>　　证明，面前老者，必然是位踏入传奇多年的强者。
<br/>　　不过想来也是。
<br/>　　这里乃是猎妖殿总殿，而总殿宝库，也非寻常地方，这里的负责人自然是位极强的强者。
<br/>　　宝库这等地方的负责人，一般很少外出，也不必到外执行任务。
<br/>　　“喏。”老者取出三枚乾坤戒，递给萧逸。
<br/>　　“这三枚乾坤戒内的，便是你的奖励。”
<br/>　　“说起来，数天前，总殿主刚遣唐音二人前往六龙地域，便已吩咐老夫准备好这些奖励。”
<br/>　　“总殿主对你，有够自信的。”
<br/>　　萧逸点点头，接过乾坤戒，道谢一声，随后并未多说什么，转身离去。
<br/>　　离开宝库，萧逸往闭关室而去。
<br/>　　进入闭关室，萧逸一如既往地谨慎，在周遭打下禁制屏障，这才盘膝坐下。
<br/>　　萧逸并未急着修炼，而是皱眉思索着。
<br/>　　他在思索，三位总殿主唤他来总殿之事。
<br/>　　三位总殿主，皆是当世传奇，这等人物，非泛泛之辈。
<br/>　　而且，理应平日里很忙，即便有闲暇时间，也应该在闭关。
<br/>　　怎么可能会无缘无故要见他这个年轻天骄？
<br/>　　而三位总殿主，也只是凝视了他一阵，问了他些许寻常问题罢了。
<br/>　　萧逸心头虽然信任猎妖殿，但始终有些许保留。
<br/>　　故他之前的回答，半参真假，也有些许保留。
<br/>　　也有些话语，仅是单纯的有感而发。
<br/>　　想来，应当没什么事。
<br/>　　另外，他心头其实也有半分猜测。
<br/>　　“如果我的猜测是真的，那便麻烦了。”萧逸皱了皱眉，面具之下的脸庞，尽是苦涩。
<br/>　　“算了，暂且不管。”
<br/>　　萧逸取出三枚乾坤戒，眼中微微有些期待。
<br/>　　虽说只是小小三枚乾坤戒。
<br/>　　但刚才总殿主所说，可是足以让他眼馋的奖励。
<br/>　　萧逸淡淡一笑，感知了一番三枚乾坤戒。
<br/>　　下一秒，眼中，一阵惊喜。
<br/>　　第一枚乾坤戒，里头是大量妖兽内丹。
<br/>　　清一色圣皇境巅峰以上级别，包含各种属性。
<br/>　　林林总总，达数万之数。
<br/>　　第二个乾坤戒，里头是大量天材地宝。
<br/>　　清一色皇品巅峰天材地宝，其中不乏珍稀级别。
<br/>　　数量，同样在数万。
<br/>　　第三个乾坤戒，里头则是大量的火属性修炼之物。
<br/>　　天火玄精、火精石、炎圣灵液…
<br/>　　“嘶。”萧逸微微吸了口凉气，“三位总殿主，好大的手笔。”
<br/>　　三个乾坤戒，分别是妖兽内丹、天材地宝，以及火属性修炼资源。
<br/>　　很显然，三个乾坤戒，三种奖励，分别来自猎妖殿、药尊殿、炎殿三殿。
<br/>　　萧逸连忙取出药炉，手中火焰凝聚，开始炼丹。
<br/>　　如果他没有猜错的话，猎妖殿和药尊殿，分别给他妖兽内丹和天材地宝，是知晓他自己本身就是个出色的炼药师。
<br/>　　所以里头奖励，没有一颗丹药，没有半粒成丹。
<br/>　　一来，成丹和材料之间，并非等价，甚至是相距甚远。
<br/>　　二来，两殿，应该也想借此考考他的炼药本事。
<br/>　　一个时辰后。
<br/>　　萧逸收回火焰与炼药炉。
<br/>　　面前，已出现了一座丹药小山。
<br/>　　里头，每一粒皆是皇品巅峰丹药，且皆是完美级别。
<br/>　　“吸。”萧逸低喝一声，手掌虚张。
<br/>　　体内两大武魂齐开，一粒粒丹药的药力被快速吸收。
<br/>　　冰鸾剑，自不必多说。
<br/>　　而控火兽，单论紫色的深度，已经差不多要追平冰鸾剑了。
<br/>　　自然，控火兽的吸收速度，如今也异常惊人。
<br/>　　两大武魂齐开下，短短半个时辰，整座丹药小山便被吸摄一空。
<br/>　　原地，没有留下半分残渣。
<br/>　　这就是完美丹药，整粒丹药完美无暇，且没有半分杂质，尽是精纯药力。
<br/>　　嘭…
<br/>　　萧逸身上，霎时一阵气势突破。
<br/>　　待得气势稳下，萧逸的修为，已然突破。
<br/>　　“圣皇境九重了。”萧逸眼眸一喜。
<br/>　　离开六龙地域时，他的修为，不过堪堪突破圣皇境八重。
<br/>　　而今，这才短短数天，又再突破了一重。
<br/>　　而且，猎妖殿总殿和药尊殿总殿给他的妖兽内丹和天材地宝，品质果真极好。
<br/>　　炼制而出的丹药，效果也极佳。
<br/>　　现在的他，不仅突破到圣皇境九重，而且体内气泉高度，达到九成还多不少。
<br/>　　“吸。”萧逸再次低喝一声。
<br/>　　第三枚乾坤戒内，无数火属性修炼之物，被他快速吸收。
<br/>　　一道道火属性力量，吸入体内后，快速化作精纯元力，进入小世界之内。
<br/>　　又是半个时辰，萧逸身上，再次一阵气势突破。
<br/>　　萧逸内视了一眼体内小世界。
<br/>　　小世界中，气泉的元力高度已达十成，气泉已满。
<br/>　　这还不止，元力，还微微溢出了气泉些许，溢至小世界中。
<br/>　　这证明，他的修为，已然突破圣皇境九重，堪堪达到圣皇境巅峰。
<br/>　　当然，修炼过程中，他也同样在吸收天地武道力量。
<br/>　　圣皇境的修炼，需得体内气泉元力足够，加上掌控更多的一条条完整武道，方能突破。
<br/>　　萧逸算了算，如今，自身掌控的完整武道，达到9100条。
<br/>　　以他的天赋，参悟掌控天地武道，本就不是什么难事。
<br/>　　唯一能难住他的修炼的，仅仅是他那庞大的气泉与小世界，还有所需的大量修炼之物。
<br/>　　现今在拥有足够的修炼之物下，他的修为，自是一路飙升。
<br/>　　之前在六龙地域得到的十万乾坤戒，看似数量多，实则里头东西杂七杂八，有用的，并不多。
<br/>　　反倒是现今三殿给的这三枚乾坤戒，虽是小小三枚乾坤戒，但里头给的东西，尽皆精准而纯粹，反倒对他用途更大。
<br/>　　“本身修为，臻至圣皇境巅峰。”
<br/>　　“这才是真正的修为与实力上的提升。”
<br/>　　萧逸自语一声，缓缓站起，结束了修炼。
