　　火牙宗内。
<br/>　　一众武者，聚集在一起，联手破阵。
<br/>　　只不过很显然，根本奈何不得大阵。
<br/>　　而火牙宗内的一众邪修，则维持操控着大阵。
<br/>　　火牙大阵，本就是火牙宗的护宗大阵，自是阵基早就覆盖了整个火牙山峰。
<br/>　　不，准确来说，是火鸦大阵。
<br/>　　火鸦大阵，乃是炎殿的有名大阵，岂是那么好破的。
<br/>　　而火鸦大阵，又是上古十大凶阵之一，自然不是只有困敌之效，其更大的作用，乃是杀敌。
<br/>　　阵法屏障之内，一头头小型火鸦不断飞舞。
<br/>　　不过片刻，已成千百之数。
<br/>　　密集的火鸦，如同一只只火焰凶兽，朝一众武者袭来。
<br/>　　一众武者，不得不停止了对大阵屏障的攻击，转而与这一只只火鸦交战。
<br/>　　而一众邪修，则没有出手。
<br/>　　他们无法出手，也不必出手。
<br/>　　如今是他们操控维持着大阵，他们的元力，早就与整个大阵连为一体。
<br/>　　这也是一众武者难以破阵的原因。
<br/>　　简单来说，过万邪修的元力，已在火鸦大阵之内融为一体。
<br/>　　而一众武者，虽联手着，但元力始终是分散的。
<br/>　　二者，自然是有了莫大差距。
<br/>　　而且，火牙宗邪修，借大阵之力，凝聚火鸦，即可对付一众武者，根本无需出手。
<br/>　　嘭…
<br/>　　这时，一只火鸦飞掠而过，一个武者抵挡不及，瞬间火焰加身，随后被撞飞百米。
<br/>　　远远看去，这个武者宛若一个火球，重重轰飞着。
<br/>　　待得火焰消散，这个武者，已然浑身焦黑，深受重伤。
<br/>　　嘭…嘭…嘭…
<br/>　　成千上百的火鸦冲撞下，一众武者这边，可谓溃不成军。
<br/>　　“该死。”火牙城少城主咬了咬牙。
<br/>　　“火牙宗，虽说在这火牙山峰之上，可终究在火牙城之内。”
<br/>　　“我们在此鏖战，父亲看不到吗？”
<br/>　　他的父亲，自然是火牙城城主，火牙地域第一强者。
<br/>　　论实力，还在火牙宗主之上。
<br/>　　一旁，一个城主府长老脸色难看道，“没用的少城主。”
<br/>　　“火鸦大阵之内，火光冲天。”
<br/>　　“在火牙城内远远看来，只会觉得这火牙山峰置身一片火海之内。”
<br/>　　“而每次火牙盛事，火牙山峰本就是一片火海的状况。”
<br/>　　“大阵不破，没人会发现这里的异况。”
<br/>　　嘭…嘭…嘭…
<br/>　　这时，又是数十个武者火焰加身，被重重轰飞。
<br/>　　“该死，这些火鸦的实力，怕是随便一头都能媲美圣皇境巅峰妖兽。”遁火门少门主此刻灰头土脸。
<br/>　　一旁遁火门长老沉声道，“火鸦大阵，可是上古凶阵，当年第一代炎殿先辈所创，自是威力滔天。”
<br/>　　“那群邪修，虽然数量惊人，但若无火牙宗主和那些邪修长老出手，我们一众武者根本不惧。”
<br/>　　“可现在，有大阵加持，反倒是我们不敌了。”
<br/>　　一旁，一个三大家族的长老沉声道，“现在这般状况，还能接受。”
<br/>　　“那群邪修，终究实力有限，以他们的元力，还无法发挥出大阵的最强威力。”
<br/>　　“我听闻，上古之时，火鸦大阵由真正的强者所操控，可是上古火鸦降临，顷刻便焚毁千万里，宛若灭世。”
<br/>　　这些个长老，都有着武道大能的实力。
<br/>　　要他们破阵做不到，但护住自家公子还有抵挡这些小型火鸦，还是能勉强做到的。
<br/>　　“难不成我们就困死在此？”火牙城少城主咬牙切齿。
<br/>　　“即便暂时能抵挡，可这般消耗下，一旦诸位长老元力耗尽，我们在场的过万武者，必死无疑。”
<br/>　　遁火门少门主咬了咬牙，“那将是整个火牙地域的灾难。”
<br/>　　“不。”一个长老摇了摇头，双眼一眯，看向高空，“我们还有希望。”
<br/>　　“不错。”另外几个长老，点了点头，同样看向高空。
<br/>　　“我们抵挡着即可，真正的胜负，不在我们身上，也不是我们与火鸦大阵的胜负。”
<br/>　　“真正的希望，在他身上。”
<br/>　　一众长老的目光，同时看向了高空中的一道冷漠身影。
<br/>　　高空中，正是萧逸与老者的战斗之地。
<br/>　　“真正的希望，便在那位紫炎身上。”城主府长老沉声道，“他胜，我们活；他败，将再无人能挡这些邪修，我们必死无疑。”
<br/>　　不错，真正的胜负，在萧逸身上。
<br/>　　若他能击杀老者，还有火牙宗主等人。
<br/>　　这一场爆发在火牙宗内的战斗，将落下帷幕。
<br/>　　他败，在场一众武者，将再无生机。
<br/>　　“胜算有多少？”火牙城少城主追问道。
<br/>　　“难说。”身旁长老摇了摇头。
<br/>　　“嗯。”几个长老亦道，“那老者，我没看错的话，乃是火牙宗的太上长老。”
<br/>　　“多年隐世不出，还以为已经死了呢。”
<br/>　　“传闻，火牙宗还未创立之时，他就已位列绝世。”
<br/>　　“论实力，他怕是和我们火牙地域第一强者火牙城城主在同一层次。”
<br/>　　“不过。”几个长老沉声道，“紫炎易霄，也非泛泛之辈。”
<br/>　　“现今放眼中域，谁人不知道此子。”
<br/>　　“一月时间，横扫过百地域；至今为止，从无败绩。”
<br/>　　“他的声名，几乎是踩着一个个天骄、一个个成名强者，还有一件件惊人事迹而快速崛起的。”
<br/>　　“这二人之战，胜负还未可知。”
<br/>　　……
<br/>　　轰…轰…轰…
<br/>　　高空中，萧逸与老者的战斗，几乎每一次交手都伴随着剧烈轰鸣。
<br/>　　二人的手段和实力，完全超乎了所有人的想象。
<br/>　　老者掌控9993条完整武道，其绝世实力，远超火牙宗主。
<br/>　　每一掌打出，皆是黑气滔天，掌掌惊人。
<br/>　　而萧逸，拳头之上，一个火焰圆环，环绕其中。
<br/>　　火焰圆环，散发着五色光芒，流转不停。
<br/>　　五种属性各异的火焰，此时竟是完美契合，虽泾渭分明，却流转不停，生生不息，宛若轮回。
<br/>　　而火焰流转之间，圆环中心，却是一股让人心悸胆寒的惊天之力。
<br/>　　萧逸的拳头，紧紧而握，火焰流转，霎时速度惊人，一股寂灭的气息，油然而生。
<br/>　　拳间之力，仿佛生生打破了五色火焰的流转，一拳间，寂灭轮回。
<br/>　　轰…
<br/>　　萧逸一拳轰出，拳风所过，空间震颤破碎。
<br/>　　老者见状，瞳孔猛地一缩，连连后退躲避。
<br/>　　待得拳风落下，高空之中，生生被打出一条横跨百丈的空间裂痕。
