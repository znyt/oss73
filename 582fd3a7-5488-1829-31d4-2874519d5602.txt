　　“我倒是有个主意，可以避免打草惊蛇……”转轮王将他的主意说了出来，听完之后，崔钰没表态，赵公明和阎罗王都说好。
<br/>　　钟馗皱眉道：“让阴阳司去当前锋，这主意有点损吧。”
<br/>　　“这哪里会，我们这么做，也等于是帮了叶少阳，明着跟他们说，便没什么问题。”
<br/>　　几个大佬都说不错，于是事情就这么定下来，大家一直推举崔钰去找阴阳司，毕竟橙子就在他身边，崔钰也答应下来。
<br/>　　本来以为酆都大帝召集他们过来，是要开个会之类的，结果直接下了两道旨意，大伙有点意犹未尽的感觉，又讨论了一会，各自离去了。
<br/>　　钟馗跟崔钰一路，来到天子殿，从花园路过时，橙子和萧逸云两人一起在逗弄水里的金鱼，见到他们来，起身乐呵呵地打招呼，崔钰关照了几句，带着钟馗一起来到自己的厢房。
<br/>　　“他们还不知道？”钟馗问道。
<br/>　　“暂时都不知情，不过很快就会传开了。”崔钰扫了他一眼，“你来我这里做什么？”
<br/>　　钟馗开门见山：“老崔，你说，大帝今日两道旨意，是什么意思？”说完又补充道，“别给我说那些虚的，我想听真话，要么你就别说。”
<br/>　　他们都是一个级别的，彼此间说话也都随意得很。
<br/>　　崔钰想了一下，道：“大帝不甘心。”
<br/>　　随即解释道：“道风叛变，这一点，的确出乎大帝的意料，大帝震怒，但是不甘心，他需要找回道风的精魄，八成是要帮他聚魄重生，继续自己的使命。”
<br/>　　钟馗缓缓点头，“青衣是真死了，但未必就没有复生的机会，说实话，他若就这么死了，我也不甘心。”
<br/>　　崔钰沉吟不语。
<br/>　　“那大帝另一道旨意，捉拿星月奴，便又是为何？大帝……当真不在乎轩辕山发难？”
<br/>　　“当然在乎，但大帝也非如此不可了。”
<br/>　　面对钟馗的疑惑，崔钰解释道：“法术公会与人间法术界，争斗了几百年了，星月奴一直都还算守规矩，没有硬来，如今却抓人填充异界，本是趁人之危——阴司与太阴山大战在即，再与轩辕山交恶，如何有精力去应对？本来事关生死存亡，空界诸派也得帮忙，但而今人家与尸族开战，自顾不暇，自然顾不上阴司了。法术公会便是料定阴司不会有所动作，方才肆无忌惮。”
<br/>　　钟馗听了崔钰的分析，思索半晌，道：“可大帝还是没有妥协，若是抓不到星月奴还好，若是真抓到了，轩辕山必然施压，届时难道真开战不成？大帝……难道就不怕腹背受敌？到那时候，该当如何应对？”
<br/>　　崔钰叹道：“我倒是能理解师尊的心情，他这是争面子，就算明知不能战，也非战不可，否则将来法术公会占了人间，依然还是要战的，法术公会倒期望阴司先开战，这样道义上便能站得住了。”
<br/>　　钟馗怔了半晌，十足感慨地说道：“想我阴司，如今却是沦落到到处被人欺负，委实不甘！大帝而今这个说法，我是赞同的，不行就跟轩辕山干一仗！”
<br/>　　崔钰没有接着他的话往下说，想了一下，说道：“师兄，找回道风精魄这件事，还得你去。”
<br/>　　“我也不知去哪里寻，且去风之谷碰碰运气吧。”
<br/>　　两人又讨论了一番，钟馗便告辞了。
<br/>　　崔钰一个人站了半天，起身叫橙子进屋去……
<br/>　　穿过轩辕之门，秋莹拼命追赶着那团精魄，在混沌界疾飞而过，眼睛死死盯着那一丛精魄，不敢有丝毫分神，突然一道白影从后面飞来，回头看了一眼，是杨宫梓。
<br/>　　两人是见过的，杨宫梓一开始没认出她，看她回头才认出来。
<br/>　　“怎么回事！”杨宫梓问。
<br/>　　“这……宫梓姐姐，你快些跟上我，我没法停下来跟你说！”
<br/>　　杨宫梓于是跟她一起飞行。
<br/>　　“宫梓姐姐，有件事……你一定要挺住！”
<br/>　　杨宫梓脸色发白，咬着嘴唇，说道：“什么挺住不挺住的，快说！”
<br/>　　秋莹指着前方那团飞舞的精魄说道：“这……是道风大大的精魄，他……死了。”
<br/>　　杨宫梓浑身一颤，一时间连作法都忘了，直直地坠落下去。
<br/>　　“宫梓姐姐！”
<br/>　　秋莹想拉她一把，但犹豫了一下，还是纵身往前去了，口中说道：“姐姐你快跟上，我要追道风大大的精魄，怕追丢了呢！”
<br/>　　“人都死了，要精魄什么用！”杨宫梓伤心欲绝。
<br/>　　“我不知道，但是你看他这精魄，并没有往阴司方向飞呀，不知道要飞到哪里去！”
<br/>　　杨宫梓听了这话，心中一动，强忍着悲痛，跟她一起追赶道风的精魄。
<br/>　　道风的精魄，竟然望与阴司相反的方向飞去，起初杨宫梓以为这是要去风之谷，后来发现并不是，这一团精魄飞过了风之谷，却是钻进了死寂迷林。
<br/>　　两人都是一惊，急忙追上去，进了死寂迷林，却又一道邪风吹过，眼前立刻失去了道风精魄的踪迹。
<br/>　　死寂迷林各种邪气都十分浓郁，又生长着极为茂密的植物，如同人间的热带雨林，自有一方气候风貌，这里气息流转，变化极快，如同一个天然的法阵，任何生灵进去都要迷路。
<br/>　　这才稍遇阻力，立刻就不知道道风的精魄去了哪里。两人四下看去，丛林之中精魄倒多的是，都是死在密林中的生灵，精魄被密林气流所困，找不到出去的路，千万年来，也不知道积累了多少，随着气流缓缓飘动，跟下雪似的。
<br/>　　“这如何是好！”秋莹急得不行，精魄长的都一样，道风的精魄在这些中间一混，立刻也就找不到了。
<br/>　　“我知道他去哪了！”杨宫梓突然醒悟，让秋莹跟着自己，在丛林里摸索着前进，寻找通往丛林中心处的道路。
<br/>　　当初他跟道风一起来，还找了好些时候才进去，现在又得重来一遍，杨宫梓心急如焚，也只能慢慢分辨寻找。
<br/>　　死寂迷林最中间，菩提神木树下，立着一个穿长衫的书生，正是林三生。
