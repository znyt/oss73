　　轰隆隆！
<br/>　　就在清虚道长正被主持人采访的时候，东方海域之上，陡然传来了一阵巨大的轰鸣声，同时伴随着一道道血色金光闪烁，朝华夏的方向呼啸而来！
<br/>　　“看样子似乎是从大洋彼岸的米国方向来的，莫非是血魔殿的强者到了吗？”
<br/>　　“这气势，简直不是一般的强大啊！”
<br/>　　“才一方势力而已，就有如此气势，若是三大势力齐至，那又该是何等景象？”
<br/>　　……
<br/>　　在华夏众人的议论声中，那一道道血色光影，转眼便已经来到百里外的海域之上。
<br/>　　等到华夏一方的众人，看清楚那一团团血光中的人影之后，顿时一个个脸色大变。
<br/>　　只见，来者仅仅只有三四十人，可是却个个气息强横！煞气冲天！
<br/>　　相比之下，东海沿岸，虽然也聚集了不少华夏武者，少数也有数万人，可是在眼前的这三四十道身影跟前，却丝毫没有任何气势上的优势，反而被对方彻底压了下去！
<br/>　　“这就是血魔殿的实力吗？”
<br/>　　“他们居然出动了这么多强者！”
<br/>　　“这些人好强！莫非就是传说中的元婴强者？”
<br/>　　……
<br/>　　此刻，就连站在人群最前端的易山河，也顿时脸色煞白无比，眼眸中透着浓浓的凝重之色，甚至连身体也开始微微颤抖起来，
<br/>　　“居然全都是元婴强者！血魔殿的实力，竟然强大到如此程度！”
<br/>　　眼前的这一幕，显然大大出乎了易山河的意料之外。
<br/>　　他本以为，华夏有叶尘在，再加上华夏本身的强者，纵然比不上三大势力，至少自保应该还是有几分把握的。
<br/>　　可是现在，这可是足足三十多名元婴强者啊！
<br/>　　更让人绝望的是，这才仅仅只是一方势力而已，就已经完全压倒了华夏武道界，若是再加上另外两方势力，那还得了？
<br/>　　恐怕即便是叶尘，也没有想到会是这个结果吧！
<br/>　　就在所有人，都被血魔殿的强者所震慑的同时，一道无比尖锐刺耳的声音，陡然在九天之上响起！
<br/>　　“叶狂仙，没想到我们居然还有再见面的一天，你可还记得本祖吗？桀桀桀！！”
<br/>　　伴随这道无比难听的声音响起，一道血色人影，犹如凭空出现一般，陡然出现在那三十多名元婴强者跟前。
<br/>　　这是一名身穿血色长袍的高大老者，似乎是眼前这些强者之首。
<br/>　　轰隆隆！
<br/>　　这高大老者一出现，身上煞气冲天，竟然令风云剧变，整个东海之上，仿佛一下子进入了寒冬腊月，让发自内心感到心悸、颤抖！
<br/>　　“这气息……怎么会如此强大！难道是化神……”
<br/>　　易山河感应到老者身上发出的气息，顿时全身巨震，两眼不由得瞪得老大，简直不敢相信眼前发生的一切。
<br/>　　他早就听闻，三大势力之中，都有化神强者坐镇，没想到传言居然是真的！而且血魔殿的化神强者，还居然亲自来了！
<br/>　　就连叶尘看到这个老者，也不由得眉头微微一凝，
<br/>　　“魔祖！没想到你也活过来了！而且还成功突破到了化神境！”
<br/>　　原来，眼前这个血衣老者，赫然正是昔日的魔界最强者，魔祖！
<br/>　　十多年前，叶尘在华山之巅，险些命丧这老家伙之手，若非最后关头曦月神魂相助，叶尘也根本不可能杀得了他。
<br/>　　叶尘没想到，这家伙居然也复活了，而且还一举迈入了化神境，实力比十多年前，进步了十倍都不止！
<br/>　　当然了，魔祖如今的实力虽强，他却还没有放在眼里，叶尘只是感到惊讶，自己昔日的仇家，全部都被复活了过来，他们背后的那个人，到底想要干什么？
<br/>　　叶尘百思不得其解，
<br/>　　“桀桀！”
<br/>　　魔祖再次发出一道奸笑，俯视着下方的叶尘，一脸嘲讽道：
<br/>　　“这么多年过去了，你也只有区区元婴巅峰境界而已，居然还敢口出狂言，想要灭掉我们三方势力！”
<br/>　　“你以为如今的世界，还是十多年前的那个世界吗？”
<br/>　　“桀桀桀！”
<br/>　　叶尘缓缓抬起头，瞥了一眼空中，一脸得意之色的魔族，语气淡漠，
<br/>　　“既然十二年前我能杀了你，十二年后也是一样！”
<br/>　　轰！
<br/>　　魔祖听到这话，顿时两眼一瞪，身上的煞气再次暴涨，方圆数百里的天空，都被他身上的血气所笼罩，连同下方的海面，也跟着激荡不止。
<br/>　　蹬蹬蹬！
<br/>　　远在海岸边的华夏众武者，也全都脸色剧变，在那强大的煞气冲击之下，不由得连连倒退不止。
<br/>　　“这就是化神强者的威压吗？”
<br/>　　“一人颠覆一国，果然不是虚言啊！”
<br/>　　“太可怕了！”
<br/>　　……
<br/>　　众人纷纷惊呼起来，相比之下，在众人的眼中，叶尘就犹如汪洋中的一片轻舟，似乎随手都要倾覆。

<br/>　　“哈哈哈哈！”
<br/>　　又有一道狂放的笑声，在天地之间回荡开来。
<br/>　　“快看北面！那是什么？”
<br/>　　人群中惊呼了一声，众人纷纷朝北面望去，顿时个个满脸惊骇之色，
<br/>　　只见，北方海面之上，一团团金色人影爆射而来，而且这些人千奇百怪，有的竟生有双翅，有的一身鳞甲，有的头顶生角……
<br/>　　“这是……万妖宫的人！”

<br/>　　叶尘看到此人，也不由得眉头微微一皱，
<br/>　　这个昔日丧命在自己手上的妖界第一强者，居然也复活了，而且也突破到了化神境！
<br/>　　华夏众人此刻也感应到，九首妖皇身上，那丝毫不弱于魔祖的恐怖气息，顿时一个个脸色越发苍白，
<br/>　　“又是一名化神强者！”
<br/>　　然而，这还并不算完！
<br/>　　万妖宫的人马才刚刚赶到，南方海域之后，也跟着传来一阵巨响，一道道犹如天神般的身影，纷纷呼啸而至，
<br/>　　“光明教的强者也来了！”
<br/>　　看到这三大势力的人马齐至，无论是东海岸的华夏众武者，还是正在看直播的亿万观众，顿时被巨大的绝望所笼罩……
