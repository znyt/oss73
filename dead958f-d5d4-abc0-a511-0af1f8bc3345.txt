　　钟馗斜眼看了他一眼，骂道：“什么本王本王的，在老子面前还敢摆谱？给我滚远一点！”
<br/>　　小马撇了撇嘴，他倒是不敢跟钟馗硬刚，溜到后面去了。
<br/>　　林三生又跟叶少阳讲起自己来到这的原因——全为了叶少阳那句暗语，被他识破，知道叶少阳是在寻求帮助，于是打点了人马，来到承德，正好钟馗到处找道风不到，打算先去寻星月奴的晦气，也来到承德，于是一起进了古墓……
<br/>　　还好落下来的地方就在几十里外，支援及时，倒是真的救了叶少阳一伙人的命。
<br/>　　“钟天师，叫你一声老祖，这次真的谢了，算我欠你一份人情，将来我一定还你。”叶少阳郑重地对钟馗说道。
<br/>　　钟馗黑着脸没有作声，心里头却是挺受用的。
<br/>　　在这里见到小青小白等一干门人，叶少阳说不出的兴奋，逐个跟他们开玩笑。
<br/>　　“你怎么也来了，快来让我捏一下……”叶少阳一只手朝雪琪胸前伸过去，雪琪赶紧用双手护住，哪里想到叶少阳这一招是佯攻，手飞快上移，在她鼻子上捏了一把，这才吐出最后两个字：“鼻子。”
<br/>　　“耍流氓啊！”
<br/>　　雪琪白了他一眼，嘟嘴说道：“谁稀罕来救你啊，要不是雨晴一个劲地催我，怕你死在这里了，我才不来救你呢！”
<br/>　　提到谢雨晴，叶少阳心中感到一阵暖意。
<br/>　　“对了，你走之后，去了一个假叶少阳，差点骗过雨晴……”雪琪把假叶少阳的事讲了一遍，叶少阳听完，震惊不已，对这个叶少阳最后被道风干掉，心中却是五味杂陈，毕竟，那是另一个自己啊。
<br/>　　不过，他也庆幸这个“叶少阳”被干掉了，否则万一自己直面他，会是什么样一个画面？叶少阳没法去想。至少，自己一定不忍心去杀死另一个自己吧。
<br/>　　“这星月奴，真是无所不用其极啊……”钟馗第一次听到这件事，也是愤慨不已。
<br/>　　一行人回到京城。
<br/>　　瓜瓜和包子他们已经在巴巴地等着了，突然见到这么多小伙伴，大家都很兴奋，互相闹成一团，至于那成千上万地被唤醒的生灵，看到钟馗驾临，一个个都吓尿了，钟馗所到之处，这些生灵全都跪在了地上，吓的瑟瑟发抖。
<br/>　　是真的怕！
<br/>　　阴司大佬中，不论酆都大帝和地藏菩萨的话，最有权威的是崔钰和转轮王，但三界生灵，最最害怕的就是阎罗王和钟馗。
<br/>　　阎罗王铁面无私，钟馗恩怨分明，而且够狠，抓住鬼就生吞活剥，没人不怕。
<br/>　　钟馗从他们中间旁若无人地走过，脑袋都不转一下。
<br/>　　叶少阳也是会来事的，进了皇宫，立刻把钟馗请到龙椅上坐下，又让人奉香上茶，钟馗心里舒坦，摆手道：“别弄这没用的了，赶紧谈正事，那个鲲鹏，你要有办法赶紧问问他，星月奴在哪，我办完了差事还得赶紧回去！”
<br/>　　叶少阳把鲲鹏拉上来，交给包子，侵入了他的神识……这一招比什么刑罚都管用。
<br/>　　半晌，包子结束了作法，坐在鲲鹏面前怔怔发呆。
<br/>　　“怎么样？”叶少阳催问道。
<br/>　　“那个，我稍微整理一下思绪……”包子歪着头继续发呆，过了半晌，深吸一口气，对叶少阳道：“他神识保护意识好强的，我只能感知到一些片段……”
<br/>　　“星月奴在哪里？”
<br/>　　“在……在一个隐蔽的空间里，应该是个洞府，具体不知道在哪……”
<br/>　　“连鲲鹏也不知道？”
<br/>　　“他……应该不知道。在他的记忆中，当时星月奴开辟了这个世界，是作为基地用的，供她有源源不断地愿力吸收，等吸收足够了之后，她就进入了修所，闭关炼化九星叠气阵了……她带进去了几个弟子，为了确保安全，她没告诉任何人如何进入洞府，只给鲲鹏留了几句话，让她不得已之时，根据线索能找到洞府……”
<br/>　　“留了什么话？”叶少阳追问。
<br/>　　“我想想啊，一共，遥想孔雀王，佛祖身中藏，若进九星阁，须找小四哥。”
<br/>　　大伙听完，先是面面相觑，进而各自思索起这四句话的意思。
<br/>　　小马吐槽道：“这星月奴怕是有病吧，直接说出来就是了，干嘛还弄个谜语让人猜！”
<br/>　　“直接说出来，现在我们不就知道了。”叶少阳白了他一眼，继续思考。
<br/>　　“后面两句话，倒是好解释，她的修所，就是叫九星阁了，想进入要先找到小四哥……小四哥是谁？”
<br/>　　叶少阳环顾左右，大家都是一脸迷茫，只有张无生的表情有点不对，似乎有点惊愕。
<br/>　　“张师叔，你知道？”
<br/>　　“我、我怎么会知道，我心口有点疼……”张无生说着在台阶上坐下，揉着心口。
<br/>　　小马摆手道：“解谜这种事，找军师，我们干不来，就不费这脑子了。”
<br/>　　大伙于是朝林三生看去。
<br/>　　林三生眉头紧锁。
<br/>　　吴嘉伟挠着头说道：“奇怪啊，按说，星月奴如果留线索，必然是鲲鹏能看得懂的，不然不是白留了？包子，鲲鹏真的不知道这个‘小四哥’是谁？”
<br/>　　包子摇摇头。
<br/>　　林三生缓缓说道：“星月奴不留名字，而是丢下这么一道谜语，那就说明，鲲鹏不知道小四哥是谁……”
<br/>　　吴嘉伟还想反驳，林三生伸手止住，说道：
<br/>　　“我明白你的意思，你听我说完，这个小四哥，一定不是鲲鹏能想起来的，不然谜语就没了意义，但却是他能接触到的，譬如说，假如真出了什么意外——就像现在这样，鲲鹏是被我们抓了，如果他没被抓，一定会想办法去通知星月奴，这时候他就会认真思考这个谜语，寻找那个小三哥，他一定能通过什么线索，找到这个人。
<br/>　　懂我的意思吗，他不知道这个小四哥是谁，但能通过线索查出来，他脑海中没有记忆，自然是之前没有意外发生，他犯不着去解谜。”
<br/>　　不愧是军师，一席话说得大家都明白了。
<br/>　　林三生道：“对我们来说，这个什么‘小四哥’就不是那么好找了，只能站在鲲鹏的角度，寻找线索……”
