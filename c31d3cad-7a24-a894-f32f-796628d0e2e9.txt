　　五长老、七长老和八长老一时间脸色阴沉如水。
<br/>　　没有人敢公然违背族规。
<br/>　　气氛一时间变得凝重起来。
<br/>　　三长老萧重看向萧逸的目光中充满了喜意，他本来以为今天晚上萧逸肯定会受到不少委屈，但万万没想到，萧逸会强势反击，而且反击得如此犀利。
<br/>　　“真是萧家先祖保佑，逸儿他长大了阿。”萧重心里暗暗想着。
<br/>　　也是，一个多年的窝囊废，忽然变得如此成熟，如此勇敢，作为疼爱他的长辈，自然是欣喜至极的。
<br/>　　忽然，五长老身旁的萧若寒站了出来，率先对在场的长老执事们行了个礼。
<br/>　　而后说道，“诸位长老、执事，萧逸说的话，理是这个理。但，紫云洞修炼的机会，对于任何一个家族子弟来说都很重要。”
<br/>　　“如果说萧逸他作为少家主，能力压我们其他家族子弟，我们自然无话可说。但以他现在的微末修为，还给他特殊照顾，这将对其他家族子弟们很不公平。”
<br/>　　“甚至于，我们年轻一辈里，会产生诸多对家族的不满情绪。”
<br/>　　“我想，祖宗先辈当年定下规矩，必定是想让我们萧家蒸蒸日上，越来越好。而绝不愿意看到这种不公平的事，更不愿意看到家族子弟们心怀对家族的愤懑的。”
<br/>　　“所以。”萧若寒说了一大通，才沉声说道，“我认为，可以将规矩变通一下。”
<br/>　　萧若寒乃是萧家有名的天才，风度翩翩，又颇为成熟，是不少长老都看好的后辈。
<br/>　　而且，他今年才17岁，修为已达凡境七重，俨然家族年轻一辈第一人。
<br/>　　假以时日，突破到后天境。
<br/>　　很可能，他将成为萧家有史以来最年轻的执事，甚至于日后成为长老。
<br/>　　所以，他的话语在家族中还是很有分量的。
<br/>　　闻言，长老执事们都微微点头。
<br/>　　这时，一直没有说过话的四长老出言道，“若寒，你的话也有理。那么，你觉得该如何变通呢？”
<br/>　　四长老是家族中最公允之人，也一向中立。
<br/>　　萧若寒道，“禀四长老，我认为，为了公平，我们确实不应该直接剥除萧逸进紫云洞修炼的资格，却也不直接给予。”
<br/>　　“哦？”四长老微微疑惑。
<br/>　　萧若寒继续道，“半个月后，紫云洞开启前夕，家族会举行比武，按照规定，排名前九的子弟方能得到名额。我想，让萧逸也参加这次的比武，而不是直接豁免得到名额，让他凭真实实力抢夺他名额。”
<br/>　　萧逸在一旁听得连翻白眼，萧若寒这话，说了跟白说一个鸟样。
<br/>　　四长老点点头，看向萧逸，道，“萧逸，这是个相对公平的做法，你认为呢？”
<br/>　　“我拒绝。”萧逸脱口而出。
<br/>　　四长老皱起了眉头，道，“萧逸，我萧家的好男儿，可不该如此畏缩。武者，自当遇事刚勇，遇危不慌，奠定一颗坚定的不屈之心，方能在武道修炼上取得成就。”
<br/>　　萧逸耸耸肩，道，“紫云洞的修炼名额，本来就是我应得的权利。我何必多此一举，冒着失去的可能答应这个要求呢。”
<br/>　　“在我看来，如果我真的答应了，这不是畏缩不畏缩的问题，完全就是傻帽的行为。”
<br/>　　萧逸一番话，听得在场的长老执事们纷纷摇头，下意识地觉得萧逸是个无赖。
<br/>　　“这...”五长老等人心头暗骂，“萧逸这个废物，明明是个窝囊废，到哪里学了这些无赖行径。”
<br/>　　倒是一旁的萧若寒丝毫不急，仍旧谈笑风生，道，“萧逸少家主，莫非是你怕了吗？”
<br/>　　萧逸直视萧若寒，道，“不是怕不的问题，而是我答应了你的要求，事情将会变得有风险。一旦我参加比武落败了，那我要付出的代价太大了。”
<br/>　　“还是那句话，只有傻帽才会做这种损几利人的事情。”
<br/>　　萧若寒皱了皱眉头，如果萧逸一直咬定族规，一直无赖下去，油盐不进的话，他也没有办法。
<br/>　　这时，萧逸却笑了笑，道，“当然了，既然这个要求是你萧若寒提出来的。如果说，你也愿意承受跟我一样的风险和代价的话，我答应你的要求也无所谓。”
<br/>　　“这也是如你所说，公平嘛，不能单单我自己傻帽，你也得跟着傻帽。”
<br/>　　“粗鄙。”萧若寒听着萧逸一口一句‘傻帽’，皱起了眉头。
<br/>　　“你要我承受怎样的风险和代价？”萧若寒问道。
<br/>　　其实萧若寒这时心里是乐着的，只要萧逸愿意接受要求，那将一步一步跌入他设计好的陷阱中。
<br/>　　萧若寒此人，颇负心计。
<br/>　　“很简单。”萧逸道，“半个月后，我将参加家族的比武，若我败，便放弃这修炼的名额。若我胜，那你萧若寒，则放弃这次的名额，不得进入紫云洞修炼。”
<br/>　　萧逸说着，忽然站了起来，来到萧若寒面前，双眸寒光大作，喝道，“萧若寒，你敢是不敢？”
<br/>　　“我...”萧若寒忽然被萧逸喝了一声，差点就脱口而出说一声，‘敢’。但他马上反应了过来，沉默了起来。
<br/>　　如今能排在家族前十的家族子弟里，最弱的都是凡境六重境界。
<br/>　　按理说，以萧逸的微末实力，半个月后绝对不可能胜。
<br/>　　可是，萧若寒仍旧是迟疑了。因为，三年一次的紫云洞修炼极为重要，甚至于对他日后的修炼造成很大的影响。
<br/>　　武者奠定基础的最佳时间在18岁以下，他今年17岁了，不可能等到三年后的再一次紫云洞开启。
<br/>　　所以，这次的修炼名额对他的意义非凡，他不敢随意答应。
<br/>　　“怎么，萧若寒，你怕了吗？”萧逸一副咄咄逼人的语气。
<br/>　　同样是一句‘怕了吗’，由萧逸口中说出，却如一记狠辣的耳光，打在萧若寒这个天才脸上。
<br/>　　“什么萧家有名的天才，在我萧逸看来，你也不过是个孬种罢了。”
<br/>　　“你说什么？萧逸，你个小废物，你当真以为我怕了你？”萧若寒一向自负，从小都在天才的光环下长大，哪里别人骂他孬种。
<br/>　　忽然，‘啪’的一声。
<br/>　　萧逸竟毫无先兆地一巴掌向萧若寒扇了过去。
<br/>　　“混蛋，你说谁是废物？”萧逸一巴掌扇完，怒喝了一声。
<br/>　　“你...你敢打我？”萧若寒有些反应不过来。
<br/>　　他万万想不到，自己一个凡境七重，萧逸竟敢对他动手，而且是在大庭广众下扇自己耳光。
<br/>　　当感受到脸上那火辣辣的疼痛时，他才马上反应过来，顿时大怒，面露杀意，“小废物，你敢打我，我要杀了你！”
<br/>　　对于他这个万众瞩目的天才来说，被萧逸这个废物当众打脸，是一件非常耻辱的事。
<br/>　　“你敢？”萧逸丝毫不惧，甚至没有退缩，怒喝一声。
<br/>　　“我乃萧家少家主，地位仅次于家主，与长老齐平。你萧若寒不过是普通子弟，却敢辱骂我为‘废物’，此乃以下放上。”
<br/>　　“四长老。”萧逸看向一旁的四长老，问道，“按照族规，家族子弟不尊长辈，以下放上，该如何处置。”
<br/>　　四长老如实回答道，“轻则杖责三十，重则关禁闭一个月，情况更严重者，逐出萧家。”
<br/>　　“好。”萧逸冷笑一声，道，“萧若寒，有种你动我一下。”
<br/>　　“你...你....”萧若寒咬咬牙，差点没被气死。
<br/>　　萧家是有这么一条族规，但只要不是太过过分，长老们一般不会太过去计较。
<br/>　　至于少家主，以往的历代少家主都是家族年轻一辈第一人，根本不可能会有以下犯上的情况发生。
<br/>　　而萧逸是个异数，萧家最废的少家主。
<br/>　　而且，以往的他总是逆来顺受，窝囊无用，这才让其它家族子弟不把他这个少家族放在眼里。
<br/>　　但现在的萧逸，心智过人，杀伐果断，自然会利用一切可以利用的优势去对付萧若寒。
<br/>　　少家主的头衔是一点，三长老这个靠山也是一点。
<br/>　　刚才那重重地一巴掌，萧逸是为依依而打。
<br/>　　“老子再问你一句，敢是不敢！”萧逸喝道，“不敢的话，就少在老子面前废话。”
<br/>　　“小...我有什么不敢的。”萧若寒下意识地想骂‘小废物’，但及时停了下来，怒喝道，“萧逸，我答应你。半月之后，若你胜，我便放弃这次的修炼名额。”
<br/>　　“好。”萧逸嘴角咧过一丝得意的冷笑，“在场诸位长老执事见证，此事就此坐定，不得更改。”
<br/>　　......
<br/>　　僵持了一整天的长老会议，最终以家族中两个名声最大的子弟间的对峙而结束。
<br/>　　当然，萧若寒是声名在外；而萧逸是废名在外。
<br/>　　....
<br/>　　五长老府邸里，五长老面露担忧，道，“若寒，你答应那小废物，冲动了些。”
<br/>　　“放心吧父亲。”萧若寒胸有成足地说道，“那小废物不过是凡境一重罢了，与凡境六重的实力天差地远，彼此间如云泥之别。你觉得那小废物有可能会赢吗？”
<br/>　　“除非。”萧若寒冷笑道，“除非他半月后能晋升到凡境六重吧，不过，武道修炼何等艰难。当年我从凡境五重达到凡境六重，足足用了一年时间，更别说凡境一重了。半个月，那小废物能做些什么。”
<br/>　　“也是。”五长老点点头，顿时放下心来，道，“是我太过谨慎了。不过，杨伏那家伙怎么还未出手，差点扰乱了我们的今晚的计划。“
<br/>　　萧若寒脸色阴冷，思索了一下，道，”杨伏虽然是有名的杀手，但始终只有凡境九重，怕是难以潜入萧家吧。“
<br/>　　他们还不知道，杨伏早就死在萧逸手中了。
<br/>　　”不管怎样，现在那小废物已经落入我们的计算，不出数月，他死定了。”
<br/>　　“哈哈哈哈。”府邸里，传来了萧若寒父子俩的阵阵阴笑之声。
