　　第693章
<br/>　　这歌舒离委实小看了程大雷。城主府的防守不足不假，这是因为程大雷本就是绝世高手，若那个蠢贼闯进来，只能说算他倒霉。就连樊梨花，也是顶级高手，碰到二把刀的江湖人也不用惧他。
<br/>　　所以城主府根本没有卫兵，因为完全用不着。
<br/>　　但城主府防守不足，不代表凉州城的防守也不足。程大雷身边有钱通玉三名敌人，他怎敢麻痹大意，守军都是军中挑选出来的精锐。
<br/>　　歌舒离将事情做成的心情又格外迫切些，不等着天亮混出城，夜里就要摸黑出去。结果被守军撞个正着，于是便是如今的下场。
<br/>　　“是个好汉子呐。”程大雷望着脸色苍白的崔白玉，道：“伤了我手下数人，仍旧死战，是我手下的大将取了他性命。若他肯投降，本是打算留他一条性命的。但对于这样的汉子，或许死在杀场方才算是得偿所愿。”
<br/>　　崔白玉静默无声，歌舒离的人头给她很大刺激。她对歌舒离其实没有什么好感，但现在难免也有些伤感。更担忧的是程大雷会如何处置自己。
<br/>　　“来人。”程大雷拍拍手，令手下人将歌舒离的人头捧下去：“拿下去厚葬，莫辱没了这样的忠义好汉。”
<br/>　　随之，程大雷的目光落在崔白玉身上，仔细端详着这位无暇美人。
<br/>　　崔白玉却也不是傻瓜，知道了程大雷的用意所在。厚此薄彼，程大雷想说而没有说出口的话只有一句：你怎么不去死？
<br/>　　如果崔白玉有慷慨赴死的勇气，程大雷会省很多麻烦。而唐突这样的美人，程大雷自己又缺乏勇气。除此之外，程大雷当真不知道该如何是好。
<br/>　　只是做为担水劈柴的杂役，程大雷又不缺这些工人，难道只是每天摆在眼前好看么？委实会令程大雷心乱如麻，扰得他不得安静。
<br/>　　至于崔白玉图谋不轨的心思，程大雷倒不是很往心里去。反正，她也搅不起什么波浪。至于其他，程大雷当真不知她还能做什么。
<br/>　　程大雷轻咳一声道：“实话与你说，这几日已经不止一人向我讨过你，都被我按着不给。至于相府那边，我想你也已经放弃幻想。相府直到现在都没有任何表示，应该是已经放弃你了。或者换句话说，除了我这里，天下很大，却没有你的地盘。”
<br/>　　崔白玉身体瑟瑟而抖，她当然明白程大雷的话，其实从凉州城破那一刻，自己便输了。只不过，是她一直倔着不肯认输而已。
<br/>　　“你不甘心，这可以理解，有自己的小心思，我也明白。”程大雷继续道：“但我想说的是，帝国之内，如你这般经历，不是第一人，也绝不会是最后一人。你委屈，你不开心，这些都可以理解。但你的委屈和不开心，和其他人也并没有什么不同。你是崔相的掌上明珠，年纪轻轻便是帝国郡主，智慧无双，美貌倾城，但这些东西并不值多少银子。”
<br/>　　程大雷一言一句，声音并不重，却有力的敲打着崔白玉的心。在城主府这几天，她也听到外面一些风声，某位之前的贵妇太太，被一个小军官收了房，开始要死要活的，之后也是抹干眼泪过日子。身在乱世，本就是一个将道德底线一降再降的过程。
<br/>　　到此刻，崔白玉不得不承认，程大雷对自己还算不错。
<br/>　　“崔白玉，有些话我已对你讲过，现在我不介意再将一遍。”程大雷开始对其直呼其名：“你如果想体面的活着，最好于我证明你的价值。如果让我知道你的价值只有美貌，那么，我就只能利用你的美貌。若送给其他诸侯，毕竟是倾城美貌，换一座城池是奢望了，换几车粮食应该还是没问题的。”
<br/>　　崔白玉明白了程大雷的意思，他是要自己归附。这也代表自己将放弃曾经的身份，背叛家族。当然，一切表现已证明，家族已将自己放弃。
<br/>　　“当然，这是一个不太容易的选择。本当家给你一些时间，三天后，你可以选择在我这书房做事，还是去我的卧房？”
<br/>　　说罢，程大雷挥挥手，让人带崔白玉几个离开。他要处理的事情还有很多，没必要在崔白玉身上浪费时间。
<br/>　　“我答应你。”崔白玉开口：“不需要三天，我现在就可以答应你，此身愿为程当家做事。”
<br/>　　说着，屈膝跪倒，恭恭敬敬的冲程大雷行了一礼。
<br/>　　程大雷倒是愣了愣，口中下意识叹道：“巾帼不让须眉呐。”
<br/>　　背叛在这个世道不算稀罕事，但对生她养她的家族说背叛就背叛，怕是男子也做不到这么爽利，多多少少也要扯上正义的名头。
<br/>　　见崔白玉跪倒，小鹿三女也无可奈何的在其身后向程大雷跪下。她们三个是崔白玉的死忠，自然追随崔白玉。
<br/>　　程大雷摆摆手：“起来说话吧，既然你要做事，便过来做事，与我看看你本事如何，有没有同本大家做事的资格。”
<br/>　　崔白玉起身靠近书案，拿起程大雷正在翻开的材料。这个选择，对她来说自然谈不上愉快，但程大雷说的没错，她已没有其他选择。终究是比成为男人榻上的玩物更体面些。
<br/>　　先活着，然后活得更好些，这是崔白玉目前的想法。至于其他，先将丢失的颜面慢慢捡回来再说。
<br/>　　程大雷端详着这位美人，道：“你擅长什么，商业，农田水利，还是人员户口……本当家也好量才而用？”
<br/>　　崔白玉抬起头，红唇中吐出两个字：“军事。”
<br/>　　这口气倒是不小，此刻她距离程大雷迫近，更能嗅到她身上的异香。程大雷只感觉情丝绵延，险些有些把持不住。
<br/>　　“喏，这个就不必麻烦你了，本当家就是领兵打仗的天才。”程大雷道：“以后，你就做本当家的助手。另外，我觉得你还是将面纱带上，以后不要靠我这么近。”
<br/>　　崔白玉冷漠如冰，看不出心中在想什么。她盯着手中的资料，却也明白程大雷在处理什么事。
<br/>　　税收。
<br/>(本章完)
