<br/>　　金家老祖被楚风眠一下子摄出身形来，表情却还是依然冷傲无比，高高在上，似乎是根本没有将楚风眠放在心上一样。
<br/>　　他乃是金家老祖，秦皇国度中都近乎无敌的人物，在圣者之境，无人能敌，赵无敌在他面前都只能是平起平坐。
<br/>　　在他的眼里，楚风眠这样一个小辈，自然是毫不放在心上，就算是楚风眠一招打出了他的身形来，金家老祖也是丝毫不惧。
<br/>　　“臣服？今日，便是你们金家的灭亡之日，还想要楚某臣服？”

<br/>　　金家老祖勃然大怒道。
<br/>　　他这样一尊巅峰圣者的无敌人物，居然被楚风眠称之为蝼蚁，简直令他愤怒到了极点。
<br/>　　在这愤怒之下，他再也不顾生死大劫的威胁，全力爆发出来。

<br/>　　这河流中，满是金色灵力，一下子向着楚风眠冲刷过来。
<br/>　　几乎是每时每刻，其中的无数金色灵力，都在冲刷这楚风眠的身躯，要将楚风眠的身躯都要打的破碎。
<br/>　　这金色灵力的冲刷，足以让坚固的灵器，都给彻底摧毁。

<br/>　　战龙决的力量被激发了出来，令现在的楚风眠身躯简直就是一尊人形的远古战龙。
<br/>　　这一股股金色灵力，冲刷到楚风眠的身躯，非但是无法造成任何一丝伤害，还能够凝炼楚风眠的身躯。
<br/>　　“该死！此人的身躯怎么会坚固到如此地步！我的力量，居然都无法毁灭他的肉身，反而是令他的肉身，凝炼的更加坚固！”
<br/>　　金家老祖心中不由的怒声说道。
<br/>　　楚风眠的肉身，实在是坚韧无比，简直不像是人的肉身。
<br/>　　“不过小子，你的肉身就算是在强大，也没有任何的意义。”
<br/>　　金家老祖手心一动，居然是出现了一本书，随着金家老祖手心一动，在这书上，无穷无尽的威压，顿时爆发了出来。
<br/>　　“这金玄古书，已经是数百年没有动用过来，小子，现在跪下臣服，你是唯一的希望。”

<br/>　　金家老祖冷笑一声道。
<br/>　　这金玄古书，可是地级极品的灵器，是他本来打算用来度过生死大劫的底牌，如今为了对付楚风眠，他的底牌尽出。

<br/>　　“老家伙，要跪下的人是你，金玄古书？给我破碎！”
<br/>　　楚风眠冷哼一声，突然手中心神一动，战龙之爪凝聚出来，直接对着那金玄古书，硬生生的拍了过去。
<br/>　　这一道战龙之爪，拍在金玄古书的上面，顿时这金玄古书，都是一下子被打出无数裂痕。
<br/>　　这一件地级极品灵器，居然是在楚风眠的手中，直接打碎。
<br/>　　与此同时，楚风眠也是身形一动，一把抓住了金家老祖的喉咙，猛然对着地上扔了过去。
<br/>　　“轰隆！”
<br/>　　一声巨响，金家老祖，直接被楚风眠抓着脖子，像是丢一只小鸡一样的扔在了地上去，这简直是令人不可思议到了极点。
<br/>　　那一个个金家圣者，眼睛瞪得豆大，许久没有开口，简直不知道说些什么。
<br/>　　他们金家的无敌人物，这可是一位巅峰圣者，居然在楚风眠的手中，如此的不堪一击？

<br/>　　楚风眠双眼一动，看着那被楚风眠砸在地上的金家老祖，还缓缓的站了起来，他眉头一挑，身上无尽龙威顿时压迫了下去。
