<br/>
<br/>        夜老一路跟着林水瑶，亲眼看着她升级打怪，对于那个才20岁的少女，夜老深深的折服。
<br/>
<br/>        后来林水瑶玩够了，有一天她突然奇想，觉得自己的人生可以进入到新阶段了，那就是她想生孩子了，她想当妈咪了。
<br/>
<br/>        夜老还记得有那么一天林水瑶跟他说，她要寻找这世上最完美的基因，她要去偷基因，然后人工受孕，她想生一个可爱的女儿。
<br/>
<br/>        林水瑶速度很快，她当时就筛选了所有人的基因，最后找到了她想到的那个最完美基因，那个最完美的基因就来自于厉君墨。
<br/>
<br/>        那天晚上，林水瑶就带着最先进的针管去偷厉君墨的基因了，这些夜老都知道，夜老唯一不知道的是竟然这么巧，也就是那天晚上，他的女儿夜滢竟然也对厉君墨下手了，在厉君墨的杯子里下了药。
<br/>
<br/>        结果就是，厉君墨提前察觉，进错了房间，林水瑶准确的找进去，夜老不知道中间发生了什么，林水瑶是打算用针管偷取基因的，但是那个针管没派上用场，她自己竟然跟厉君墨滚上了床，两个人亲自上阵了。
<br/>
<br/>        后来，林水瑶如愿怀孕了，生下了一个女儿。
<br/>
<br/>        所以当夜滢将自己看到的一切告诉夜老时，夜老迅速将这件事的来龙去脉都摸清楚了，他没有告诉夜滢，也没有告诉任何人，他是当年唯一一个知情者。
