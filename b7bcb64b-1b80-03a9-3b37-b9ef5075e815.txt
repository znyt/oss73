　　第306章
<br/>　　都说不要走漏消息，可世上哪有不透风的墙。靠上千庄稼汉保守秘密，几乎相当于扯淡，于是一传十，十传百，这家的小舅子，那家的邻居……本来大家对于给山贼做工，避之如猛虎，但有了上次的经验，给的钱多，待遇还好，庄稼汉都愿意趁着农闲时挣些钱贴补家用。
<br/>　　一村一镇，都被老婆哄着去找当地的工头说情。如赵三壮这些人，又是脸皮比较薄的，即便往日有些矛盾，求到门前他们也抹不开脸拒绝。
<br/>　　于是，到了这一日，刘悲看着赶到蛤蟆寨的人数，也是很久没回过神来。
<br/>　　乌压压一片，人数一清点，足有三千余人。
<br/>　　对于做工来说，人自然越多越好，可人多意味着消耗也大。工钱倒不是太大问题，就算每人一两银子，钱也不多。关键是每天的口粮，让人干活，总不能让人吃不饱饭，三千壮劳力一起打开腮帮子，可是一笔巨大的消耗。
<br/>　　当然，人来了也不能赶出去，刘悲开始给这些人登记造册，依照各人本事不同进行分工。指挥三千人做工，也不是一件简单的差事，所以由李行哉配合他，李行哉在这些山贼中，算是有文化的。
<br/>　　这三千人中，并不缺乏浑水摸鱼的，刘悲直接给踢了出来。
<br/>　　“干嘛啊干嘛啊，这样的人也来凑热闹。”刘悲拎出一个瘦骨嶙峋的懒汉：“我们讲规矩，真不把我们当坏人呐。”
<br/>　　踢出了数百人，都是想来滥竽充数的。这些人聚在一起，眼看就要被人赶走，有人气急败坏，忍不住说出声来：“你们给山贼干活，我们去报官，大家谁也做不成！”
<br/>　　众工人吓得不清，这些人无非是趁着农闲挣笔外快，可事情要是传出去，勾结山贼可是重罪。
<br/>　　“谁说的？”刘悲猛然睁大眼睛，杀机外泄：“告诉我是谁说的？”
<br/>　　雅雀无声，这群人都识趣的闭上嘴巴。周围淡淡的血腥味，诸人眼中的煞气，树叶上没有被雨洗掉的血滴，偶尔走过时踩到的白骨……这些都提醒他们，这些人不是什么好好先生，都是杀人不眨眼的恶贼，而现在他们身处的是贼窝子。
<br/>　　“既然你们不想走，那就不要走了。”刘悲冷笑一声。
<br/>　　人群呼啦啦跪倒一片，口中不住哀求着。
<br/>　　“大王爷爷，我们只是一时胡说，求求你们，不要杀我们。”
<br/>　　“把各人名字都登记下来，问清楚他们家住在哪里，家里都有什么人。”刘悲的狠劲比程大雷有过之而无不及：“这些人就暂且留在山寨，什么活最苦最累，就交给他们干。”
<br/>　　三千人中，难免有偷奸耍滑的，这也是无可避免的事情。对于这个问题，刘悲将他们按照各村各镇分组，一人偷懒，全组扣工钱。
<br/>　　在加上有建筑精通的加成，这倒也不算什么太大的问题，工程一旦开始，进度是快的。
<br/>　　关键问题是，工程开始后的各种消耗。每天人吃马嚼，不是一笔小数目，而建设山寨需要的大木，山石岛上都没有，只能从外地购买后运过来，这些同样需要钱。
<br/>　　于是，开辟商路就变成了迫在眉睫的一件事。
<br/>　　这件事由程大雷亲自负责，程大雷需要的药材、盐巴、茶叶已经被送过来。这次的目的只是开拓市场，所以程大雷只带了二十驮货，这批货物先送到落玉寨，再从落玉寨出关。
<br/>　　程大雷亲自护送，云中龙、乔鹤、游九楼三个，程大雷也没有放过他们。大家现在是一根绳上的蚂蚱，初次合作，万一对方有二心呢，直接绑在一起，通过接触打消隔阂，慢慢建立信任。
<br/>　　而主要护送任务，由秦蛮的蛮字队负责，人数也不需要太多，只要五十人就可以了。
<br/>　　但这件事有个不是问题的问题，出关必须经过落玉寨，落玉寨有一种矮马，吃苦耐劳，适合走山路，所以落玉寨是个重要的中转点。可经过落玉寨，程大雷就不得不见到樊梨花。
<br/>　　实话实说，草原姑娘的火辣和热情，让程大雷有些不知该如何应对。
<br/>　　面对这个不是问题的问题，程大雷想了个不是主意的主意。
<br/>　　他让其他人负责接货，装货，自己在外面等着，如此岂非就不用见到樊梨花了。
<br/>　　程大雷枕着双手躺在草坪上，百无聊赖的看着头顶的白云发呆。
<br/>　　现在的程大雷已经有不少崇拜者，而在外面，他的名声传得更匪夷所思。可很少有人知道，真正的他，无非是个乏味无聊，偶尔有些恶趣味的人而已。
<br/>　　这个世界是很静的，没有无处不在的监控，没有无处不在的网络讯号，你可以用一个月的时间等一个人的信，也可以用一个下午的时间盯着一朵白云发呆。
<br/>　　同样，这世界也充满饥饿，瘟疫，流离失所，仗势欺人……
<br/>　　不知躺了多久，程大雷已经有些犯困了，忽然听到马蹄声。他这才站到草坡上看过去，只见秦蛮、云中龙一伙赶着矮马驮着货物来了。
<br/>　　“大当家，大当家……”秦蛮一伙围了过来：“东西都准备好了，咱们随时可以出发了。”
<br/>　　“嗯。”程大雷点点头，没有见到樊梨花，他不知应该得意还是应该沮丧，随口道：“我让你们找个向导引路，你们找到了没有？”
<br/>　　“当然找到了，这可是个好向导，大当家要不要猜猜是谁？”秦蛮。
<br/>　　“还能是谁……”
<br/>　　程大雷看过去，只见队伍中走出一人，胭脂马，绣绒刀，身上用斗篷遮着脸，在将斗篷揭开时，露出一张吹弹可破的俏脸。眸若寒霜，眉绽杀机。
<br/>　　程大雷一哆嗦，他倒不是不想见到樊梨花，关键是程大雷打不过她。这姑娘又虎又二，她要是发起飙来，可是不管不顾，逮着程大雷就是一顿胖揍，程大当家不要面子嘛。
<br/>　　“就是樊寨主啊，樊寨主从小在这里长大，周围地形熟得很，大当家你没想到吧？”秦蛮哈哈大笑：“惊不惊喜，意不意外？”
<br/>
