　　　　母猫产崽当天，整个房子开始剧烈的震动起来，满屋生光，有一股异光不断闪烁。
<br/>　　余子童惊骇不已，向前一看，母猫既然已经不知何时暴毙而亡，偌大的腹部以为会产下一窝的猫崽，没想到只有一只。
<br/>　　十月怀胎，一胎生一崽。
<br/>　　此猫崽才不过成人拳头那般大，但是周身有淡淡光泽。
<br/>　　一双小眼睛睁大，金光迸发，黄金瞳由此而来。
<br/>　　这是一只超强血脉的变异猫种，天生异象，无与伦比，天上掉了一个大馅饼，砸到了余子童的身上。
<br/>　　随着黄金瞳慢慢的长大，余子童靠着此猫成为了白家的一品捉妖师，享受高人一等的待遇，虽然说不上荣华富贵，但也吃喝不愁。
<br/>　　白家重名声，他如此出丑，不但解决不了工地的鼠灾，还会名誉扫地，黑袍男子心肠一狠，他甚至都有生命的危险。
<br/>　　此时，吐完一口血的余子童额头渐渐露出汗渍，心里紧张至极，涂小安在一旁看得直摇头，就连苏羽也叹气的不忍看他。
<br/>　　白家的捉妖师如此的弱？还是那个生物掌控者林凡太厉害了。
<br/>　　涂小安也没想到这个余子童那么的不堪一击，好歹打个几个回合，让别人掌掌眼。
<br/>　　你到好，提剑冲过去，立刻倒飞回来，这不是瞎折腾吗。
<br/>　　涂小安的摇头，被余子童看到，顿时心中恨得牙痒痒，若不是他在旁边挑唆，自己根本不用出手，也不会丢那么大的人。
<br/>　　而且看样子，苏羽老板也对他非常的不满起来。
<br/>　　那小子自己明明是元素修武者，完全有能力对付生物掌控者，却偏偏不出手，故意看自己狼狈后的笑话。
<br/>　　可恶...
<br/>　　涂小安自然也察觉到了余子童那怨恨的目光，他脸上露出淡淡的笑容，随后开口：“余大师，看来你也太不给力了，你的猫对上二阶的鼠精都没落败，而你冲上去就败下阵来，我都不知道你在干什么”
<br/>　　什么意思，说他哗众取宠？
<br/>　　余子童狼狈的站起来，手中握着长剑，指着涂小安愤怒：“混蛋，他是生物掌控者，我区区一个一品捉妖师，如何能对付，你不要站着说话不腰疼”
<br/>　　“哦，原来如此”涂小安恍然大悟的说：“既然余大师对付不了，早说呀，何必上去自取其辱”
<br/>　　我TM还不是被你逼的上去。
<br/>　　这句话余子童在心中咆哮，她现在因为知道涂小安是元素修武者，不能太得罪。
<br/>　　“哼...苏老板，余某没用，就此告辞，反正你也有高人相助”
<br/>　　余子童没有脸面在继续待下去，况且他知道自己几斤几两，根本不是生物掌控者的对手。
<br/>　　说罢，他朝着工地喊了一句：“宝贝，我们走”
<br/>　　这是招呼黄金瞳离开。
<br/>　　但此时工地上两只小型生物的战斗却异常的凶猛，就好像是两位武林高手，打起架来，风卷残云，飞沙走石。
<br/>　　一时之间，既然还是未分胜负，猫对付老鼠很简单，偏偏这只老鼠的境界比这只猫要强的多。
<br/>　　可谓针尖对麦芒了。
<br/>　　黄金瞳越战越勇，威势滔滔，势必要将挑衅它的这只老鼠给抓住，对于余子童的叫喊置若罔闻。
<br/>　　“黄金瞳，给我回来，不用打了”
<br/>　　余子童再次大喊了一声，可在战斗的黄金瞳完全忽略了自己的主人，眼中只有那只鼠精。
<br/>　　此猫天生异象，桀骜不驯，换句话来说，自尊心非常的强大，岂能容一只老鼠在自己的面前蹦跶。
<br/>　　就算这是一只二阶的鼠精也不可以，非要将它拿下不可。
<br/>　　余子童本来就心中窝着火，无处可发，自己的宠物还无视自己，顿时勃然大怒，一张脸都扭曲了起来，大喊：“畜生，还不快回来”
<br/>　　这一叫，可谓震耳欲聋，直接将工地上战到白热化的猫鼠分开了。
<br/>　　黄金瞳怪叫了一声，炸立的毛发落下，疑惑的看着自己怒发冲冠的主人。
<br/>　　余子童提着剑，快速的走到了黄金瞳的面前，愤怒的火在胸中燃烧着，直接大力一脚踢过去。
<br/>　　黄金瞳措不及防，被自己的主人一脚踢飞出去，重重的惨叫一声，如断了线的风筝落地。
<br/>　　“连你也可以不将我放在眼里吗”
<br/>　　余子童一声质问，将火发泄在了此猫身上。
<br/>　　呜呜呜....
<br/>　　黄金瞳何等桀骜，看到主人生气，脸面匍匐在了余子童的脚下，好无凶相，端端是一只可怜的小猫。
<br/>　　“以后在不听话，我打死你”
<br/>　　余子童总算在自己的宠物猫面前，找回了一点尊严。
<br/>　　起码，这只猫不管何等凶煞，在他的面前，一直是顺从的，不敢忤逆。
<br/>　　涂小安蹙眉起来，对余子童的行为感到反感，这只好猫是没遇到好主人呀。
<br/>　　他这种行为，不但涂小安看不过眼，就连不远处的黑袍男子都微微有点生气，开口：“如此高贵品种的猫，却跟了一个废物的主人，实在可惜也”
<br/>　　“你们...”
<br/>　　余子童冷哼几声，转身就走，走了几步后，身后的黄金瞳还趴在地上，眼中看着那只鼠精，非常的不甘心。
<br/>　　“还不走”
<br/>　　黄金瞳没动，看着那只小小鼠精在黑袍的身边蹦蹦跳跳起来，像是说你快来抓我呀。
<br/>　　“嗷...”
<br/>　　黄金瞳浑身毛发瞬间炸立，威势迸发，非常想将那只挑衅的老鼠抓到。
<br/>　　“走”
<br/>　　余子童恨不得立刻离开这里，除了苏羽，好像每个人都比他厉害，就算是那个保镖，他都感觉心中没底。
<br/>　　留着这里，就是自取其辱。
<br/>　　可气的是，自己的猫却不想离开，就想跟鼠精决一胜负。
<br/>　　“皮又痒了是吧”
<br/>　　余子童唤不动此猫，顿时又觉得没面子，气得脸色涨红，浑身有些颤抖，又是一脚踢过去。
<br/>　　但这次，他没踢动黄金瞳，浑身毛发炸里的黄金瞳眼中死死盯着老鼠。
<br/>　　那小白鼠眼中戏谑而灵性的道：“猫儿，快跟你主人回家吧，本包子大人不吃你就是，瞧你也怪可怜的”
<br/>　　“嗷...”
<br/>　　黄金瞳准备扑过去，但余子童立刻大喝：“不准去，这里没我们的事情了”
<br/>　　余子童可不傻，苏羽都不需要他了，他还让自己的猫去捉鼠干嘛。
