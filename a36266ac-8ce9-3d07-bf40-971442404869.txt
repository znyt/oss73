　　其实，围攻他们的人里，也有强者，如果一起发力猛攻，也能拖住他们一时半会，然而当四宝祭出金身罗汉之后，所有看到的人都骇然失色，一时间不敢上前，这才让四宝一干人穿过人群。叶少阳急忙前去接应，汇合到一处。
<br/>　　“卧槽，山羊，那有个老头特么的好厉害，召唤出来的不知道是什么鬼东西！我们那个时代，可是没有这么牛逼的法术！”四宝一见面就跟叶少阳吐槽起来。
<br/>　　叶少阳扫了一眼，看大家都在，也就放心了，转头望着张晓寒。张晓寒出神地盯着四宝，说道：“你是什么人，你方才用的法术，可是罗汉金身？”
<br/>　　四宝没好奇地说道：“你问这干啥？”
<br/>　　“据我说知，罗汉金身出自达摩降魔经，是佛门一件无上修行法门。”
<br/>　　一个干巴巴有点阴冷的声音从人群后面传来，人群自动分开，鬼姨走了过来。
<br/>　　她一直没有动手，一直站在高处观察着叶少阳和他这几个帮手。
<br/>　　鬼姨一边走来，一边定定地望着四宝，说道：“你既然会这罗汉金身，我也不问你是怎么学来的，我只跟你说一件事。”
<br/>　　四宝皱眉看着她。
<br/>　　“加入我法术公会。”
<br/>　　四宝噗嗤一笑，“阿姨，我们现在是两军交战，你就这么挖人墙角真的好吗？”
<br/>　　“没有永远的朋友，只有永远的利益，我不知道叶少阳给了你什么利益，让你替他卖命，但是身怀绝技，却声明不显，你难道愿意甘愿如此？只要你加入我公会，我便立你为人间佛门第一弟子，你以为如何？”
<br/>　　四宝咧了咧嘴，道：“你说了算？”
<br/>　　“自然，凭我法术公会的声望，还没什么事是做不到的，不光是你，还有你身边这几位朋友，我看的出，实力都是极为不俗，若可合作，必有厚报。”
<br/>　　四宝一只手搭在叶少阳肩膀上，笑道：“那他呢？”
<br/>　　鬼姨瞥了叶少阳一眼，缓缓说道：“他今日辱我门威，便有通天手段，我也留不得他。”
<br/>　　张晓寒听见这句话，松了口气。
<br/>　　四宝摇头说道：“这不好，我们是兄弟，你光招募我，留着他，他会骂我的。”
<br/>　　“死人就不会骂人了。”鬼姨冷冷说道。
<br/>　　四宝哈哈大笑，说道：“不跟你们开玩笑了，别说我们人少欺负人多，你们尽管来吧，单挑还是群架，佛爷我都奉陪。”
<br/>　　鬼姨面色一变，望着神情坚定的这几个人，有些疑惑地说道：“叶少阳到底给了你们什么好处，让你们如此服从？”
<br/>　　四宝跟吴嘉伟互相看了一眼，都没有回答这个问题。没有任何利益的驱动，彼此之间却愿意为对方去死，整个法术联盟的成员，都是这样一种情况，就算是新加入不久的凤兮也不例外，因为……他们是兄弟。
<br/>　　“这位大和尚，你是哪一派的弟子？”一个老和尚拨开众人，走了过来，望着四宝，神色之中有些激动。
<br/>　　“我……”四宝犹豫了一下，道：“贫僧五台山弟子。”
<br/>　　“五台山！”那老和尚跳起来，上下打量他好半天，皱眉道：“贫僧怎么没见过你啊？”
<br/>　　“您是……”
<br/>　　“贫僧五台山方丈了心。”
<br/>　　了心禅师！！
<br/>　　四宝差一点没站住，他是五台山掌教的传人，自然听说过了心禅师的名字，算起来，这位了尘禅师还是自己的太师祖，而且是真正的嫡传，今天居然见到了真人，四宝怎么能不激动。
<br/>　　“不要暴露。”叶少阳在他耳边低声说道。
<br/>　　四宝醒悟过来，支吾着说自己只是五台山下一座小寺的弟子，名义上是五台山弟子，但是从来都没有资格上山。
<br/>　　了尘禅师一听，心中疑惑顿消：像五台山、九华山这样的大宗派，在山门左右，都有很多小的寺庙，依附着这些大门派，名义上也算是大宗派的弟子，真遇到大事什么的，也能拉出来帮个忙打个架，最不济也能充个人场。因此这些大宗派也都乐得默许他们的存在。
<br/>　　了尘禅师身为五台山方丈，门下弟子无数，自己都辨认不过来，自然不会去认识一个外宗的弟子，上前一步，问道：“你是怎么修成罗汉金身的？”
<br/>　　“这个……”四宝看了叶少阳一眼，本想说是他给的，想了想也是没必要说实话，只说是机缘巧合。
<br/>　　了尘一听，缓缓点头，口中直说“造化”，神色显得十分激动。
<br/>　　“够了！！”张晓寒大喝一声，他不想再为不相干的人不相干的事情浪费时间，阴冷的目光烁烁地望着叶少阳，说道：“叶少阳，你想怎么死！”
<br/>　　“单挑怎么样？我满足你一下。”叶少阳歪头看着他，“来一场生死之战吧。”
<br/>　　张晓寒冷笑，“你很聪明，我这边人数十倍于你，要杀你等易如反掌，我为何要跟你单独决斗？”
<br/>　　“因为你要面子。”叶少阳笑了笑，然后点点头，“你到手的媳妇被我抢走了，还当着这么多人的面，你肯定想把我碎尸万段，如果靠着人多打赢我，并不算光彩，所以，你想一个人战胜我，把我踩在脚下，只有这样才能找回面子。”
<br/>　　张晓寒静静地看着他，道：“你有什么资格跟我决斗？”
<br/>　　叶少阳转身看了一眼身后，道：“这，就是资格。”
<br/>　　他身后，站着那些个法术公会的弟子，有被自己打伤的，还有不少从山体斜坡上爬上来的，身体被树丛之类的刮的全是伤，走路都一瘸一拐的，互相之间搀扶着，本来是追叶少阳的，后来看双方对话，暂时不打了，就在后面跟着，经过叶少阳一提醒，所有人目光都落在他们身上。这狼狈相……
<br/>　　四宝等人肆无忌惮地嘲笑起来。
<br/>　　张晓寒和鬼姨等人脸色很不好看，呵斥他们退下。
<br/>　　张晓寒望着叶少阳，咬牙切齿地，还想说点什么，叶少阳摆了一下手，抢在他前面说道：“好了好了，我已经听够你装逼了，都是老一套，没意思，要打就快过来。”
