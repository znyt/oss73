　　　　终于要上架了，就说几句吧。
<br/>　　原本清白是不想写上架感言的，因为看过很多的书，一发上架感言，底下留言就是一片，什么要收费了，我不看了，我弃坑了之类的话，其实还蛮伤人的。
<br/>　　这就宛如一对热恋中的情侣，其中一人忽然说，我们分手吧，转身便走。
<br/>　　上架收费，这是在所难免的，因为作者也要吃饭，不然每天那么费尽心思的写作是为什么。
<br/>　　我知道很多人会因为收费了，选择离开，在这里对这些要离开的读者说声抱歉，因为我也要生活，柴米油盐也都要花钱。
<br/>　　请谅解吧。
<br/>　　同时我也希望那些准备离开的人，不要在底下留言说什么离开之类的话，因为我看到会伤心。
<br/>　　这本书上个月二十三号，编辑就让我上架了，但我婉拒了，硬生生拖到了现在，一是想大家多看一点免费的章节，二也是成绩还不是特别理想。
<br/>　　但一本书，终归是要上架，当然我希望所有的读者都能够订阅支持，我也知道大多数都是学生，不求打赏，只求正版订阅即可。
<br/>　　另外，那些决定不看的，清白也希望你们看在之前免费看了三十六万字的份上，明天给这本书一个首订，我想这个应该不算过分吧，哪怕当成我们之间的分手费...
<br/>　　因为书第一天上架的订阅特别的重要，就好像迎来人生的第一个高考，是这本书的转折点。
<br/>　　首订几分钱就搞定了的事，我相信大家应该不会拒绝我吧。
<br/>　　在说一点关于这本书的事情吧...
<br/>　　很多老粉丝都知道，这是我的小号，我之前写了一本关于蚊子的异兽文。
<br/>　　为什么这次改写蛇，最重要的一个原因是，我自己是属蛇的，我想写一本蛇类小说。
<br/>　　既然是写蛇，我就想好好的写关于蛇的故事，所以这本书是不化龙的。
<br/>　　因为但凡蛇类小蛇，最后的进化趋势，都是化龙，我就想新颖一点，我没有说龙不好，咱们号称龙的传人，已龙为图腾，但这只是一本蛇类小说而已。
<br/>　　扯太远就有点本末倒置了。
<br/>　　既然是写蛇，我还是希望写的跟别人与众不同一点，它不需要过分充满杀戮，不需要见人就吃，它也可以善良有趣点。
<br/>　　所以，这本书的开头，就是用家蛇代入，已蛇王为主题。
<br/>　　人有善恶，蛇亦如此，蛇王之所以被人信仰，就是它不会肆意的杀戮，已人为苟且。
<br/>　　大家见惯了蛇凶煞的一面，我就写蛇另外没有的一面出来。
<br/>　　我觉得一本书，总要有一点内涵的，有点属于自己的东西，当然，有人会说小说而已，看的爽就行，可能我更贪心一点吧。
<br/>　　在这里，很感谢一路陪伴过来的朋友，大家每天投票，每天催更，每天打赏，其实对我都是莫大的动力。
<br/>　　我不知道最后留下来的会有多少人，但我需要你们的陪伴，需要你们给的动力。
<br/>　　这本书，就是写给你们看的，没有你们，它毫无价值。
<br/>　　再者，大家说的最多，其实就是清白的更新慢，其实我知道自己更新的真不快，但请原谅，我真的是龟速，每天写之前，还要坐在电脑前先发呆个半小时，才能开始码字。
<br/>　　不敢写的太快，写太快，容易水文，求质量，速度就肯定会慢下来。
<br/>　　但上架之后，我会更加努力的更新，尽量让大家看的更爽一点，更多一点。
<br/>　　明天中午十二点上架，我们不见不散好吗...
<br/>　　最后用尽全身力气怒吼一句：求订阅，求订阅，继续求订阅...
<br/>
