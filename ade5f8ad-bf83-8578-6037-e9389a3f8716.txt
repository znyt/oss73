　　听闻破空之声，众人立刻转头望去。
<br/>　　只见一大帮人，浩浩荡荡的向此地飞来。
<br/>　　这帮人足有两千余人，正是黑山要塞驻扎的军队，而为首之人正是一身银色甲谓，威风凛凛的祝英。
<br/>　　在祝英的身边，还有环山矿脉的守将王副尉。
<br/>　　嗖嗖嗖！！
<br/>　　祝英带着两千余名甲士，呼吸间便来到了现场。
<br/>　　看到任云都等一众神武学府的人，个个相安无事，人数也只是少了十几人，祝英眸中异色一闪。
<br/>　　“各位，血瞳族的人呢？”祝英率人来到了任云都等人的身前，朗声问道。
<br/>　　“被我们击退了！”任云都道。
<br/>　　“已经退了？你们没事吧？”祝英面上露出惊讶之色，随即沉声问道。
<br/>　　“我们没事！”任云都面无表情的摇了摇头，心中却是阴郁不已。
<br/>　　黑山要塞据此不过三十余万里，就算不用传讯，黑山要塞的强者应该也能察觉此地的状况，居然到现在才来增援！
<br/>　　这说明什么？
<br/>　　说明祝英很可能是故意不来，让他们自己在这里抵挡血瞳族之人。
<br/>　　苏莫和其他神武学府的人，也不是傻子，都是对祝英有些不爽，战斗结束了，对方倒是来了。
<br/>　　“没有就好！”
<br/>　　祝英点了点头，随即立刻对王副尉吩咐道：“王副尉，立刻清点人员损失，以及矿脉的情况！”
<br/>　　“是！”中年将军王副尉立刻领命，随即立刻带人查看情况。
<br/>　　苏莫等神武学府的人，则是在一旁静静等待。
<br/>　　片刻之后，王副尉回来了，脸色略显难看的回到了祝英的身前。
<br/>　　“王副尉，怎么了？”祝英见王副尉脸色难看，皱眉问道。
<br/>　　“回禀尉将大人，环山矿脉守卫共一千零三十人，陨落了九百余人，还剩九十五人！“王副尉道。
<br/>　　“嗯，还有呢？”祝英点了点头，继续问道，黑甲守卫都在此地，剩多少人他一眼就能看到，根本不用禀报。
<br/>　　“矿脉全部塌陷了，我下去查看了一番，灵石矿全部不见了！”王副尉面色难看的说道。
<br/>　　“什么？”
<br/>　　祝英闻言顿时脸色一沉，随即怒声喝问道：“怎么回事？”
<br/>　　不待王副尉回答，他又看向任云都，冷喝道：“任副尉，你们不是打败了血瞳族的人吗？矿脉为何不见了？”
<br/>　　祝英是真的怒了，这个矿脉非同小可，虽然只是极品灵石矿脉，但产出量极大。
<br/>　　虽然他们开采的灵石，要上缴圣朝国库，但会暗自克扣三成，每年都会给他祝家提供一大笔资源。
<br/>　　而现在，灵石矿脉居然不见了，他如何能不怒？
<br/>　　虽然只要有龙脉在，还会继续形成新的灵石矿脉，但这需要一个过程。
<br/>　　这个过程对于他们祝家来说，就是巨大的损失。
<br/>　　“这……！”面对祝英的喝问，任云都顿时语塞，他只顾着率人抵挡血瞳族之人的攻击，哪里还能顾及到矿脉！
<br/>　　“尉将大人，矿脉不见了只是小事……！”王副尉正欲继续禀报，但话说了一半，便被打断了。
<br/>　　“什么？矿脉不见了居然还是小事？”
<br/>　　祝英闻听王副尉的话，顿时打断了对方的话，怒喝道：“你知不知道此矿脉的重要性，此矿脉虽然不大，但能供养圣朝多少军队？”
<br/>　　呃！
<br/>　　王副尉被祝英一顿猛喝，顿时有些懵了，那并不是说矿脉不见了是小事，而是想说还有更大的事。
<br/>　　“查，尽快查出原委，是不是被血瞳族的人暗中抢走了？”祝英继续怒喝道。
<br/>　　“尉将大人……！”王副尉苦着脸，想发火但又不敢。
<br/>　　“还不快去，磨磨蹭蹭成何体统！”祝英见王副尉的模样，再次怒喝道。
<br/>　　王副尉深深的吸了口气，咬了咬牙，大声道：“尉将大人，龙脉不见了，彻底消失了！”
<br/>　　此言一出，现场一片寂静，变得雅趣无声。
<br/>　　所有人都瞪大了眼睛，静静的盯着王副尉。
<br/>　　龙脉不见了？
<br/>　　这怎么可能？
<br/>　　众人都知道龙脉的珍贵，只要龙脉在此地，此地的灵石矿脉永远都开采不完。
<br/>　　龙脉之气会不断的形成灵石，永无止境，直到龙脉苏醒离开。
<br/>　　可以说，一条龙脉，就是一棵摇钱树。
<br/>　　但现在，龙脉居然不见了？
<br/>　　难道是大战惊动了龙脉，这也不太可能啊！
<br/>　　因为众所周知，龙脉一般只潜藏在大地深处，最起码都是百万里深的地底，怎么可能被惊动？
<br/>　　他们的大战虽然动静很大，但还不至于惊动地心深处百万里远的龙脉吧！
<br/>　　“你说什么？再说一遍？”祝英脸色狂变，一把揪住了王副尉的脖颈，大声咆哮了起来。
<br/>　　“属下说……龙脉不见了，地底再没有灵气喷涌！”王副尉铁青着脸说道。
<br/>　　“怎么可能？”祝英怒发冲冠，面色立刻铁青了下来，龙脉不见了，那此处灵石矿脉就彻底废了，他们的损失无法估量。
<br/>　　“尉将大人，我刚才亲自查探了，灵气的确是没有了，说明龙脉不见了！”王副尉声音颤抖着说道，他真怕祝英一怒之下迁怒与他。
<br/>　　“快！带我去看看！”祝英一把将王副尉推开，大声咆哮道。
<br/>　　随即，王副尉立刻带着祝英，飞到矿脉的遗址上方，两人随后便钻进了大地之下。
<br/>　　现场，众人默不作声。
<br/>　　无论是神武学府的弟子，还是矿脉的守卫，亦是祝英带来的士兵，均是一脸怔怔。
<br/>　　龙脉不见了，矿脉要彻底废了啊！
<br/>　　祝英如此愤怒，也在情理之中。
<br/>　　任云都脸色阴沉无比，心中暗叹了一口气，龙脉不见了，这要是要塞的祝隆将军怪罪下来，他怕是也要受罚。
<br/>　　毕竟，王副尉不在，他负责矿脉的监管，监管不力，难免对方不会怪罪！
<br/>　　苏莫面色平淡，仿佛此时与自己无关一般，只是在默默的疗伤。
<br/>　　之前的战斗，虽然不是在矿脉的上方，但强大的余波，已经将矿脉所在之地完全震塌了，高山都夷为平地了，一切都成了废墟。
<br/>　　他也没有留下什么痕迹，谁能想到是他做的！
<br/>　　除非有人查看他的玉镯空间，但这是不可能的事情！
<br/>　　随后，众人的身形都降落了下来，落在了矿脉所在山谷的废墟上，静等祝英和王副尉回来。
