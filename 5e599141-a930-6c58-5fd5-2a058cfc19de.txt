　　绝龙岭，太师还在浴血奋战，只是战局太过惨烈，太师仅仅在垂死挣扎。
<br/>　　“尔等暂退，我来与闻太师一会！”
<br/>　　突然，远处一道声音传来，众弟子抬头，只见云雾之中，一翡翠袍道人踏空而来，正是终南山玉柱洞云中子。
<br/>　　杨戬等人闻言迅速退开数十里外，太师道：“云道兄，你今日到此，是为我送行吗？”
<br/>　　云中子行了一礼，道：“闻道兄，此乃绝龙岭，你的命数到了，休要怪我！”
<br/>　　语罢，云中子一挥袖，自云层之中，飞出八根苍茫大柱，按八卦方位分布在八方，无数符文隐隐流动，将太师禁锢其中。
<br/>　　太师惨笑道：“天亡我大商，尔等乱臣贼子，今日我闻仲就算死也不会让尔等好过！”
<br/>　　说着，他驾着麒麟，手持金鞭，飞入空中，麒麟哀鸣，金鞭交响，像是在为一位英雄送行。
<br/>　　八根大柱陡然间光芒大盛，明光流转，每根大柱中都飞出四十九条赤龙，通体火焰纷飞，嘎嘎作响，向着中间围拢过来。
<br/>　　太师目露神光，奋起反抗，身下麒麟长鸣，不甘被灭。云中子在阵外发雷，八方大柱怦然合在一起，激起万丈火焰，冲向高空。
<br/>　　……
<br/>　　朝歌城中。
<br/>　　“陛下，太师闻仲已败亡。”诸葛亮对着纣王道。
<br/>　　“死了么？”纣王喃喃道，“黄飞虎等人没有出手吗？”
<br/>　　“陛下，黄飞虎等人恐怕早已被人察觉了，他传回的信息之中，有好多都是假的。”
<br/>　　“嗯，朕已知晓，此次是朕大意了，早就知道了阐教的不同，朕还以之前的态度对待，此次失败却是给了朕一个教训。”纣王沉默片刻，说道道。
<br/>　　“陛下，如今该如何是好？”诸葛亮道。
<br/>　　“等。”
<br/>　　“等？”
<br/>　　“是啊，等，等截教的反应，朕不信阐教如此大肆灭杀截教众仙，通天教主却无丝毫反应。就算他没有反应，我等也该去激化他们之间的矛盾。”纣王道。“况且，就算此次败了，我等还有机会，就当是来历练，下一个世界，不再会出现这种情况。”
<br/>　　“是，陛下。”
<br/>　　上皇峰头，金灵圣母娇躯突然一晃，多宝道人叹道：“师妹！”
<br/>　　金灵圣母突然伸出双臂，探入无限虚空，用手一捏，一颗数千里大小的赤色星辰轰然间粉碎，化为碎片粉尘，从金灵圣母手指间流泻下来。多宝道人摇头叹息。
<br/>　　残阳如血，西周大军早已退返西岐，绝龙岭上尸横遍地，血流成河，鲜血染红了山林，白雪都被覆盖。
<br/>　　突然，一个黑袍神秘人出现在山岭之上，他取出一柄黑幡，飞入上空，摇动起来，瞬间，黑幡如一个黑洞，爆发出强烈的吸扯力，尸体之中，飞出无数光点，无数乳白色的灵魂腾空而起，汇聚成一条微微发白的生魂长河，被黑幡吞噬。
<br/>　　不一会儿，黑幡消失，那神秘人摇头一晃，身影也随之消失不见。
<br/>　　半月之后，金灵圣母出山，阻击周军，广成子与之大战，数个回合，一枚大印翻天，砸落天灵盖，金灵圣母一点真灵飞入封神榜。
<br/>　　又半月，广成子独上碧游宫，通天教主命人领进宫中，广成子说自己前来是为了还截教众位道兄的法宝，而且还了法宝，便告辞离开了。
<br/>　　截教众仙愤怒不已，感觉自己被对方羞辱了，不能忍受。杀我教中师兄妹，竟然还来教中还法宝，这不是看不起截教吗，简直欺人太甚！
<br/>　　“欺人太甚！”多宝等人目眦尽裂，愤怒不已。
<br/>　　龟灵圣母气急道：“启禀师尊，我两教弟子逢此量劫，本是各凭胸中所学，施展本事，生死各安天命，本来也无话可说。但可恨那玉虚弟子，打死我截教众仙不够，而今还上我碧游宫还宝，美名其曰归还宝物，可那厮哪里是还宝了，分明就是逞其豪强，卖弄精神，当面侮辱我碧游宫道法，简直欺人太甚！”
<br/>　　多宝听见龟灵圣母说话失了分寸，连忙喝道：“师妹，休要胡言！”
<br/>　　龟灵圣母心中一凛，连忙扣首道：“师尊，弟子心急，言语失去分寸，请师尊责罚。”
<br/>　　上首榻上，通天教主闭目不语。截教众仙俱皆不敢言语。
<br/>　　五日后，纣王上碧游宫，不知道与通天教主谈了些什么，众人只知，第二天，教主便传出消息，他将于五月初九，于界牌关前摆下万仙阵，邀诸仙前来破阵，一决高低！
<br/>　　此话一出，碧游宫人皆兴奋不已，广成子等人也心惊胆颤，他们只是因封神榜上数量不够，需要从截教之中拉几个出来，这才设法上碧游宫。可如今的事实却弄得两教教主大战，若真是教主大战，宇宙翻覆，三界崩塌，到那时他可罪过了。
<br/>　　昆仑山，玉虚宫中。
<br/>　　“师尊，弟子罪过！”广成子等人在下方伏首跪拜道。
<br/>　　“嗯，此事本座已然知晓，既然如此，那么，广成子，此次大战之后，尔等便封山禁过万年吧。”元始瞟了众人一眼，淡淡道。
<br/>　　“是，多谢师尊宽恕！”闻言，众人都大松一口气，拜谢道。其实，他们又哪里知晓，元始早已知晓事情会是这样，所以他并没有太过在意此事，不过，犯了错误，终究是要惩罚的。
<br/>　　“尔等都去好好准备吧，莫让他人在万仙阵内看了阐教笑话！”元始又道。
<br/>　　“是，师尊！”众人闻言，都默默退下。
<br/>　　西方灵山。
<br/>　　“师兄，我等西方教大兴的机会快要来了，此次阐教与截教两教大战，势必会两败俱伤，我等可借此机会，大兴西方教。”准提有些兴奋道，他那愁苦的脸上，终于露出了一抹微笑。
<br/>　　接引双手合十，那一直悲悯世人的脸上，也难得露出一丝笑容，道：“师弟，量劫过后，我西方教定然大兴！”
<br/>　　“是，师兄。”准提道，“不过，师兄，此次大战，东方人杰地灵，我等也好去度些资质好的天才入我西方教。此等机会不容错过啊！”
<br/>　　“阿弥陀佛，苦海无边，万丈红尘，我等慈悲为怀，是该去解救世人脱离苦海，超脱红尘。东方三千红尘客，合该与我西方有缘！”接引一脸慈悲道。
<br/>　　“师兄此言大善！”准提也一脸济世救苦的模样。
<br/>　　劫魔殿，冥河老祖不知何时来到殿中，与万劫魔尊一番长谈，两者商议如何合作，共对西方教等问题，相谈甚欢！
<br/>　　冥河走后，万劫魔尊脸上露出诡异的神情，道：“天数已定，西方大兴？那可不一定，若西方大兴，我魔教又如何大兴，所以，西方不该兴！我若成魔，佛奈我何！哈哈！”
<br/>　　冥界。
<br/>　　“启禀帝君，最近有一神秘人，常与僵尸一族将臣联系，不知该如何应对？”一鬼兵对着三生大帝汇报道。
<br/>　　“此事我已知晓，三界大战，看来是有人想要扰乱我冥界，我自会处理！”三生大帝目光闪烁，淡淡道。
<br/>　　“是，帝君！”
<br/>　　人族祖地。
<br/>　　“禀武祖，妖魔猖獗，魔族入侵我九州大地，我等该如何应对？”一长老问道。
<br/>　　“再遣五皇与之一战，可派遣武宫弟子出山，降妖伏魔，势必要夺回我人族之地！”武祖玄天命令道。
<br/>　　“是，武祖！”
<br/>　　风云变幻，大乱将起！
