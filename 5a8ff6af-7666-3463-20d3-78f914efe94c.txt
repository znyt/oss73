　　元始创精神世界灵神界，巨大的动静早已惊动洪荒，惊天的圣威凌驾于苍穹之上，威压天地众生，想让人不知道都难。
<br/>　　但元始创造完世界后，便消失不见了，谁也不知道他去了哪里，这让人们面面相觑，不知道该说什么为好。弄出如此大的动静，什么事都没有，难道元始在与洪荒苍生开个玩笑？这怎么可能，圣人又哪会无聊到如此地步？
<br/>　　就在众人心思各异，胡思乱想之际，元始再次出现了，出现在原先的虚空之中。
<br/>　　瞬间，无尽紫气汇聚而来，宛若朝圣一般将他拱了起来，一朵朵巨大的金花，显现在虚空，瑞气千条，仙气缭绕，圣音袅袅，骄阳横空……
<br/>　　圣人动，天地惊！
<br/>　　虽然元始是混元大罗金仙，不是圣人，但修为到了一定境界，能够带动天地异象，那是再常见不过的事情了。
<br/>　　惊天异象，轰天动地，瞬间便再次吸引洪荒无数生灵的目光，他们纷纷猜测元始这是要干什么？
<br/>　　“如此大的惊天异象，圣主这是要干什么？”
<br/>　　“天尊显威，难道是有人招惹到了天尊？”
<br/>　　“定然会有大事发生，否则玉清圣人也不会弄出如此大的动静？……”
<br/>　　“……”
<br/>　　就在众人议论纷纷之际，元始那浩大威严的声音开始传遍洪荒天地，响彻九霄：
<br/>　　“十年之后，灵神界启，凡有缘者，皆可入之！”
<br/>　　浩大的圣音，传遍天际，传到洪荒每一个生灵的耳中，在他们脑海之中炸响。凡是能够听到元始声音之人，脑海之中都会自然而然的浮现出一方朦朦胧胧的神秘世界，宛若有无尽神力在牵引着他们，能够让他们看到世界之中的一草一木，一鸟一兽……相应的，灵神界之中的一些信息，也会出现在他们的脑海之中。
<br/>　　在知道灵神界的信息之后，洪荒彻底沸腾了。
<br/>　　“什么？世上竟有此等世界存在，简直不可思议，圣主的实力究竟达到了何等恐怖地步？”
<br/>　　“嘶！灵神界，九重天，一重一境界，精神进入，宛若真身降临，感同身受？在其中死亡并不会真正的死亡，而且可以无限复活！嘶！这怎么可能？！”
<br/>　　“神功秘术，道法神通，灵兵神器……应有尽有，真是……太好了！”
<br/>　　“同级战斗，决出最强！哈哈，我喜欢！战！”
<br/>　　“怎么会有这样的世界，简直是不可思议！在灵神界之中的领悟，更是能够带回道肉身之中？嘶！”
<br/>　　“万族争霸，洪荒风云再起！”
<br/>　　“……”
<br/>　　在知道这样的信息后，不管是什么人，都在第一时间震惊了。他们不敢相信，世界上会有这样的地方，这种世界，对于他们这样的修炼者，简直就是天堂啊！
<br/>　　没有死亡，可以无限复活，不仅有神功秘籍等好处可以拿，还可以与无尽种族同级作战，增强自身的战斗经验，磨练自己的意志，最重要的是，这些都能够反馈到肉体身上。
<br/>　　可以无限复活这一个条件，便是无数人无法拒绝的。要知道，修士之间，除了正常死亡或者作战之时被杀死，其他大多数时间都是修炼神通秘籍，或者创造功法时走火入魔死亡的。
<br/>　　若是能够无限复活，那每个人便有无数次机会前去尝试，一次不行，那就两次，两次不行，那就三次，三次不行，那就更多次……
<br/>　　有了这个条件，洪荒便会再次出现无数神功，千万不要先看人的智慧，毕竟，人的脑洞是无限的，什么样的想法都可能出现。
<br/>　　不去管其他，元始挥手间洒下无数光雨，大片大片的光雨，如雨水般飞向洪荒各地，有人族部落，有魔族地盘，有妖兽山洞，有大海秘藏……
<br/>　　洪荒各地，无论是九幽黄泉，幽冥地府，还是九霄天外，天庭住址，无论是无垠四海，还是茫茫大地，树林，沼泽，绝地，禁区，等各处，都有光雨落下。
<br/>　　这些光雨，便是灵神界的接引界令，有万亿之多，它们可能被遗漏在某个地方，不被查寻到，也可能被某个修士捡到。捡到它的，可能是普通人，也可能是超级魔头，谁也不知道，它们的主人会是谁？
<br/>　　这一刻，看到虚空之中飞下的无尽光雨，洪荒所有人都心头火热，因为他们从脑海之中的信息之中早已察觉到，那是灵神界的接引令牌，只有持有灵神令者，才可以进入灵神界，这也是元始所说的有缘者。
<br/>　　众多大神通者们纷纷出手，想要在虚空之中截取神令。
<br/>　　“轰！”
<br/>　　西方天际之上，一道巨大的金色手掌出现，猛然向着虚空中那些降临的光雨抓去。
<br/>　　“哼，本座的东西是你们可以随便想拿就拿的吗？”元始心中一声冷笑，毫不在意。
<br/>　　果然，那些光雨在碰到巨掌之后，没有丝毫停顿，透体而过，继续向着其他地方飞去，丝毫没有被金色手掌影响到。
<br/>　　那手掌好像不信邪，再次抓了几次，但依旧什么都没有抓到。
<br/>　　不仅西方，其他地方亦是如此，各路大神们纷纷施展出大神通，向着天空中的神令抓去，可惜，谁也没有抓到，那些神令该去什么地方，依旧去了什么地方，丝毫没有被影响到。
<br/>　　他们顿时便知道了，这其中被元始做了各种限制，防止有大神通出手，不给其他修士留下机缘。
<br/>　　众人无奈，若是其他人还好，至少可以威胁一番，弄到更多的神令，但面对元始，他们心中没有丝毫那样的心思，若真是如此，那样只会死的更早。
<br/>　　没有办法，众人只好凭借自己的机缘，去寻找神令，或者去抢夺别人的令牌。可是，别人拿到神令，都会选择第一时间炼化，没有多少人会留了下来。一旦炼化，别人便不能将之取走，因为神令早已与宿主连为一体了，宿主死，则神令散。
<br/>　　这一刻，洪荒各地都沸腾了，天空之上，无数修士飞来飞去，向着四周寻找神令，他们多希望自己能够得到一枚神令，进入灵神界，从此改变命运。
<br/>　　这一日，注定是血雨腥风。
<br/>　　“哈哈，我得到了一枚灵神令！哈……”
<br/>　　“噗嗤——”
<br/>　　可惜，还没有等他将话说完，他的声音便戛然而止了，却是被旁边的人杀了。
<br/>　　“灵神令是我的，谁也休想抢去！”一人手持神令大笑道。
<br/>　　“死！”
<br/>　　就在此时，一道淡漠的声音从高空传来，将他捏成碎末，爆炸开来。
<br/>　　……
<br/>　　“哥哥，你看！我得到了灵神令了！”一个青年手持一枚神令走到他哥哥面前，欣喜道。
<br/>　　“什么？我看看！”另一个男子激动道，之后一把夺过神令，看了看。
<br/>　　“果然是神令，哈哈！”男子大笑道，说着，他的眼中闪过一道异色，之后转身便将一柄剑插入旁边青年身体之内。
<br/>　　“哥哥，为……什么……”青年脸色苍白，一口鲜血喷出，接着踉跄着倒在地上。
<br/>　　“小志，不要怪我，灵神令只有一枚，而我们却有两人，你知道的……”
<br/>　　“原来如此，可是……我想说的是……我有两枚……”可惜，还没等青年说完，他便死了。
<br/>　　男子顿时愣住了！
<br/>　　……
<br/>　　诸如此类的事情，这一天，在洪荒之中不断上演着。谁也不知道，仅仅只是为了一枚灵神令，洪荒便会死去多少人！
<br/>　　元始心如明火，对于此类事情，他早有所料，但这些都不能让他脸色波动分毫，他心如止水，早已将生死看透了。
<br/>　　这个世界，没有实力，什么都保不住，能怪谁？弱小即为原罪，元始要让他们明白，这个世界上，只有实力，才是永恒。
<br/>　　如此，等他们进入灵神界后，便会更加刻苦，更加疯狂了，那样，对于元始更有利。
<br/>　　不再去管其他，元始一个转身，便消失于虚空之中，回到了昆仑山。
<br/>　　……
