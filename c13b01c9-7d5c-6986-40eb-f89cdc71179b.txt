　　　　这十九级的帝王眼镜王蛇就这样被涂小安残忍的吞噬了，从始至终，它都没有一点反抗之力。
<br/>　　应该说是做不了反应，就死翘翘了。
<br/>　　几十万的毒蛇被吓的不轻，惊叫声不绝于耳，此起彼伏。
<br/>　　“爽...”
<br/>　　涂小安将一条十六米的巨蟒给吞下，自己本身连一点变化都没有，直接化为了二十万的能量值。
<br/>　　二十万呀，搁在以前够他升十级的了。
<br/>　　可现在连一级都升不了。
<br/>　　哼，在他蛇王面前，不臣服就是死，显然，这里是他练级刷怪的好地方。
<br/>　　他想要快速提升等级，唯有吞噬高等级的觉醒蛇种。
<br/>　　另外，他也有段时间他没有靠吞噬生物，来提升等级了，做任务反而成了他的捷径。
<br/>　　因为吞噬普通生物，不知道要吞噬多少，要是大范围的吞噬，就成了屠杀，破坏大自然的平衡。
<br/>　　“吼...”
<br/>　　又是一声震耳欲聋的咆哮声冲天而起，只见金环巨蟒携带着十八米巍峨的蛇躯朝着涂小安野蛮冲撞了过来。
<br/>　　那阵仗简直就是地动山摇，飞沙走石。
<br/>　　这条蟒气息浑厚，虽然等级一样是十九级，但比那条帝王眼镜王蛇却要强的多。
<br/>　　它几乎是一脚已经迈入蛇精的行列，这一脚要是迈的过去，它将口吐人言，成为二阶生物。
<br/>　　可惜，它选择挑战涂小安，那么将永远不会有口吐人言的机会了。
<br/>　　论肉身强度，这条金环巨蟒可以将一栋别墅都给摧毁，力大无穷，别说别墅，小山都能夷为平地。
<br/>　　金色蛇头笔直的撞来，就跟一辆大货车似的，想直接将涂小安这条十五米的大蛇给撞飞，甚至是撞的粉身碎骨。
<br/>　　见此一幕，涂小安心中冷笑，泰山崩于前而面不改色，直接用自己的蛇头正面迎敌，蛇头对蛇头的碰撞到了一起。
<br/>　　“轰...”
<br/>　　空气中层层爆响，一股狂暴的气浪自两蟒为中心的激荡席卷出去，顷刻间飞沙走石，地面剧烈震动起来，方圆几里的地面被瞬间掀开，裂痕如蜘蛛网四处蔓延。
<br/>　　数千条小蛇被气浪形成的风暴刮的无影无踪，一时之间，产生的异象恐怖如斯。
<br/>　　嚓...
<br/>　　一声好像骨骼粉碎的声音响起，金环巨蟒脸色大变，顿时间头破血流，发出了无比凄凉的惨叫声。
<br/>　　昂...昂...昂...
<br/>　　只见金环巨蟒的蛇头在空中扶摇直上的摇曳，样子非常痛苦，蛇躯不断的后退。
<br/>　　退着退着...几个呼吸后...
<br/>　　轰...十八米的金环巨蟒蛇躯如麻绳一样的散落在地，奄奄一息。
<br/>　　一招KO。
<br/>　　金环巨蟒跟涂小安两两相撞，瞬间失去战斗力，它的头颅骨头恐怕早已粉碎。
<br/>　　此时，十五米的金银蛇游曳到它面前，看着金环巨蟒如迟暮的老人，动都动不了一下，冷淡无比的道：“跟本蛇王比肉身力量，你是作死”
<br/>　　涂小安的蛇躯从一级就被系统开始强化，如今整整十八级，何止刀枪不入，况且如今原型，力量更是大的没边。
<br/>　　这一撞，直接将金环巨蟒给撞成了傻逼。
<br/>　　正如两拳相撞，力量决定一切。
<br/>　　“安息吧，你败了，就是要被我吞”
<br/>　　紧接着，又是一副巨蟒吃巨蟒的残忍画面，同样是巨蟒，同样是觉醒蛇种，这金环巨蟒还不是涂小安的一招之敌。
<br/>　　两者实力差距太大。
<br/>　　体型大，等级高，无用，战斗力决定一切。
<br/>　　涂小安现在虽然是十八级，但他可是跟上古蛇种，二十七级的蛇精打的不落下风。
<br/>　　就这样，开阔地上，几十万毒蛇看着金银蛇吞着另外一条巨蟒，这一方诸侯，就这样被吃了，吃的连渣渣都没用，直接化为涂小安的能量值。
<br/>　　“叮...恭喜宿主成功吞噬十九级金环巨蟒，获得250000进化能量值”
<br/>　　“叮...恭喜宿主进化能力值达到500000，等级提升到十九级”
<br/>　　两条蟒加起来是四十五万能量值，在身上涂小安本身还有一些能量值的累积，直接提升等级。
<br/>　　旋即一道白光猛然笼罩在十五米的金银巨蟒身上，如温煦的阳光，舒服级了，他的体型已肉眼可见的速度再次暴涨了三米。
<br/>　　等白光消失，他已经是十八米的巨蟒级别了，气息更加的磅礴冷冽，将几十万的毒蛇吓的恨不得钻入地底下，根本无法承受。
<br/>　　“现在轮到你了”
<br/>　　等级提升完毕，涂小安将蛇眸锁定在那条精致的小蛇身上去。
<br/>　　这条看似一丢丢大的小蛇，才是真正的对手。
<br/>　　那两条不过是开胃菜，根本不在涂小安的眼中。
<br/>　　变异魔鬼蛇，能用魔鬼来形象，这种蛇绝对不是泛泛之辈，性情应该非常的凶煞可怖。
<br/>　　“我无意如你为敌，你走你的阳光道，我走我的独木桥”
<br/>　　此蛇被涂小安锁定，既然微张蛇口，发出了沙哑阴森的声音，听了让人忍不住的鸡皮疙瘩。
<br/>　　二阶生物，蛇精级别，能口吐人言，涂小安一点都不意外，反而饶有兴致的看着它，妖冶的眸子中化过一丝玩味：“那便臣服于我，我可以不吞你”
<br/>　　收服一条蛇精级别的毒蛇，还是好处多多。
<br/>　　放在家中也是带刀侍卫，镇宅之兽。
<br/>　　“找死”
<br/>　　魔鬼蛇闻言，顿时狰狞可怖，通灵摄厉起来，看着暴涨到十八米的巨蟒，浑身散发出一股恐怖的气息，周身的空气都颤栗起来：“你别以为吃了两个没用的东西，就以为能对付我，我的强大是你所不能想象的”
<br/>　　“我的目标是蛇沟深处，无意与你浪费时间”
<br/>　　涂小安的巨大体型迅速缩小，变成了一米长度，然后悬浮在空中，淡淡的道：“那巧了，我们的目的是一样的，你臣服于我，我带你去”
<br/>　　“混帐，你在逼我出手”魔鬼蛇一句话说出，周围形成一股股旋风，气息愈演愈烈，独特的蛇类气息将附近的毒蛇早给吓的远远。
<br/>　　留下一大片真空地带，给这两条能口吐人言的蛇精。
<br/>　　这两条蛇，都是它们仰望的存在。
<br/>　　“你只有两者选择，一是臣服于我，二就是被我吞，助我修为”
<br/>　　什么魔鬼蛇，我本蛇王面前放肆，我让你变成死鬼蛇。
<br/>　　涂小安见此蛇不凡，有收服之心，要是不上道，那么就只能吞了。
<br/>　　这种蛇血脉是强，但是吞了还不够让涂小安开启第二次进化。
<br/>　　不过提升一级，估计还是可以的。
<br/>　　那么就让这条蛇帮自己真正步入二阶生物的行列吧。
<br/>　　成为二阶生物，蛇沟深处就算有在大的凶险，他都有底气面对一切。
