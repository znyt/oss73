　　群情沸腾起来，欢呼着，但很快有人组织，让大家都安静，听叶少阳讲点什么。
<br/>　　“现在是收买人心的时候，你说点鼓舞人心的话！”张无生在他身边低声说道。
<br/>　　“说什么？这……我真的不行啊，师叔不行你来，你来说几句……”
<br/>　　“不行就练！”张无生溜走了，吐出一口气，幸亏自己不是主角，这场面，赶上万人大会了……
<br/>　　“呃……我是叶少阳。”
<br/>　　叶少阳紧张地手都没地方放，望着下方黑压压的人群，挤出来一个笑容，自己感觉比哭得都难看，当下勉强集中心神，介绍了一下当前的情况，不过说得结结巴巴的。
<br/>　　“那个，情况大家都知道了……嗯，我也不敢说自己能领导大家，但我们现在是一个组织，大家要相信组织，团结一致……嗯，我说什么来着，不抛弃不放弃，众志成城，万众一心，一起渡过难关……”
<br/>　　越说越乱，叶少阳自己都不知道说的什么了。而且因为紧张，声音也很小，下面很多人都听不清楚。叶少阳还在组织语言，小马突然跳到他边上，一只脚瞪着城垛子，大声冲下面喊道：“我是马王爷，风之谷白云城主，大家认识吗！”
<br/>　　“不认识！”
<br/>　　有人回了一句，大伙纷纷起哄。
<br/>　　小马一点也不害羞，大笑起来，一把搂着叶少阳的胳膊，吼道：“不认识我不要紧，你们总认识他吧，茅山掌教、人间当代道神、天选之子、捉鬼联盟的扛把子、阴司七十三司阴阳司的老大，他的名头说出来，比那些专家教授和传统作家都多，你们认不认识他，大声告诉我！！”
<br/>　　“认识！”
<br/>　　大伙又起哄。
<br/>　　“认识就好，捉鬼联盟相比你们也知道，我，白眉，四宝禅师，哪一个不是名震天下的法师，当然，我而今是鬼……还有阴阳司诸位兄弟，你们也都清楚的。”
<br/>　　小马越说越豪情，用板砖砸着自己胸口，得意洋洋地喊道，“我们这帮人，阴阳五煞和十三鹰，今日可都来了！听过我们的名号吗，纵横三界莫敢不服，哼哼，我们干啥来了，就是为了对付法术公会，带大家回去。”
<br/>　　“这个世界，是法术公会的星月奴创造的，我知道，你们很多人都害怕，害怕什么？法术公会背景深啊，轩辕山啊，七大长老、上古异兽、三大盘古神僧，牛逼啊。但是又怎么样，他们想统治人间，我们五煞十三鹰，第一个不答应！”
<br/>　　叶少阳趁着小马喘息的工夫，悄声问道：“神马是五煞十三鹰？”
<br/>　　小马不理他，把板砖往城垛上一敲，粗着嗓门吼道：“我们在人间，他妈的只干一件事，公平，公平，还是他妈的公平！法术公会想要在人间搞事，咱们就干他，生死看淡，不服就干，是不是这个道理，大声告诉我！”
<br/>　　“是！！”
<br/>　　人群中奇迹般的爆发出了整齐的回应。
<br/>　　“而今他们把我们抓来，困在这里，简直他妈的欺负人，我不管你们都是干什么的，是正是邪，等回到人间，大家该打的打，该斗的斗，但是在这里，大伙要团结一致，听从指挥，我命由己不由天，法术公会又怎么了，惹到我们，就一个字，什么？”
<br/>　　“干！！”
<br/>　　群情激昂。
<br/>　　小马甚是得意，继续问道：“谁是天下第一，谁是我们的领袖！”
<br/>　　双手指向站在一旁，一脸懵逼的叶少阳。
<br/>　　“叶少阳，叶少阳，叶少阳——”
<br/>　　山呼海啸一般的呼唤。
<br/>　　所有生灵的情绪，都被小马一番慷慨激昂又匪气十足的话语打动了。他们多半是邪物，或者是基层的民间散修，江湖气本来就重，又热血好斗，听不惯什么文绉绉的台词，小马的这番话，却是正中下怀。
<br/>　　这就是演讲的力量。说到底也是一种洗脑。
<br/>　　同时，因为叶少阳话不多，一直酷酷地站在边上（其实是紧张），反而加重了神秘的感觉，在他们心中，真正的大人物本来就该是这样的。
<br/>　　小马是够兄弟的，把气氛鼓舞起来之后，就立刻跳下城垛，把舞台留给叶少阳一人。叶少阳也不知道做什么——他其实也不用做什么，就这么站在城垛上。
<br/>　　北京城南门最高的城头上，如同一座山，叶少阳就站在山顶，一览众山小地俯瞰众生。
<br/>　　没人知道他心里在想什么，但所有人都把他当成了领袖。
<br/>　　多年以后，在场的很多活下来的邪物仍然清晰地记着这极具象征意义的一幕：人间最强的那个人，曾经站在异界的城楼上，接受众生的膜拜。
<br/>　　天亮之后，一个临时的联盟诞生了，在京城四周筑起了一道防线。
<br/>　　虽然从远方的城市不断还有邪物涌来，但已经不再能造成混乱。叶少阳安排了专人守在四周，专门打醒那些邪物，然后再接受进联盟之中。
<br/>　　腾永清失踪了，叶少阳发动了很多人去找，但一直没有找到。大伙虽然担心，但也没什么办法。
<br/>　　叶少阳让包子对陈悦和皇帝分别施展读心术，倒也得到了一些意想不到的结果：
<br/>　　陈悦的记忆碎片之中，显示她来自人间，住在乡下，是一个很普通的姑娘，好像十几岁就不上学了，一直在家务农。
<br/>　　非常文静，腼腆，没有什么朋友，家里也没有亲人，只有一个奶奶。
<br/>　　这些都很正常，唯一让大伙不能理解的是：每天晚上，她的奶奶都会起身，教她吐纳……入定之后，好像奶奶还会传授很多，但都是深层的记忆，而且有一种自我保护的意思……这些记忆，连陈悦本人差不多都不知道。包子自然更无法获取。
<br/>　　除了这些之外，包子并没有从陈悦意识中获得更多的意识随便，因此这次使用读心术，非但没能解决她的身世问题，更加深了他们心中的疑惑。
<br/>　　叶少阳和张无生轮番又检查了一遍陈悦的经脉，体内的确没有罡气存在……
<br/>　　这就很矛盾了，一来人可以说谎话，但记忆是不可能作假的，但她身上没有罡气，况且，在她记忆碎片中，教她修炼的居然是她奶奶，一个乡下的老太婆……
<br/>(本章完)
