　　“噗通！”
<br/>　　“噗通！”
<br/>　　“噗通！”
<br/>　　这些海盗不过就是比普通人要凶恶一些的悍匪，何曾见识过如此手段？
<br/>　　加上他们再联想起之前，自己的同伴被叶尘打爆的情景，顿时一个个吓得全都跪倒在地上，开始拼命的磕头求饶，浑身瑟瑟发抖。
<br/>　　就连那些人质们，也忍不住跪在地上顶礼膜拜起来，一群人喜极而泣，
<br/>　　“真的是神仙下凡了！我们有救了！”
<br/>　　叶尘冷哼一声，低头瞥了一眼那黑人首领，
<br/>　　“还不动手？”
<br/>　　黑人首领感受到叶尘身上的杀意，顿时吓得一个激灵，当即抽出腰间的匕首，一咬牙，狠狠斩在自己的左手手腕之上。
<br/>　　“啊！”
<br/>　　随着一声惨叫声，黑人首领的左手，已经被他自己斩了下来。
<br/>　　其他海盗看到这一幕，个个脸色煞白，可是他们却也清楚，倘若不这么做，必定是死路一条，于是也纷纷拔出匕首，朝自己另外一只手斩落。
<br/>　　一道道惨叫声此起彼伏，一旁的人质也全都吓得纷纷倒退。
<br/>　　一句话，就让这些凶神恶煞的海盗们，全都自断一只手！
<br/>　　这少年，也太可怕了！
<br/>　　叶尘一脸平静，转而望向一旁的庄菲菲，
<br/>　　“你是怎么被他们抓到这里的？还有，这里是什么地方？”
<br/>　　庄菲菲叹了一口气，一脸幽怨道：
<br/>　　“我也是被公司给坑了，说是让我来非洲录制一个探险类的综艺节目，没想到居然遇上了这些索马里的海盗，也多亏本小姐机智，不然这次可就完蛋了！”
<br/>　　叶尘听到这话，顿时微微一愕，
<br/>　　“你是说，我们现在，在非洲？”
<br/>　　庄菲菲眨了眨明亮的大眼睛，
<br/>　　“应该是吧！对了，叶尘，你怎么会出现在这里？是若曦妹妹请你来救我的吗？我记得之前，我群发过一条求救信息……”
<br/>　　听完庄菲菲的解释之后，叶尘一阵默然，
<br/>　　他也没有想到，传送阵被毁坏之后，传送的位置，居然偏差了数万里之遥！
<br/>　　问清楚缘由之后，叶尘这才又转而望向那黑人首领，
<br/>　　“三天之内，将他们送回华夏，我便饶了你们！”
<br/>　　那黑人统领顿时一脸为难之色，他们身为索马里海盗，平日里也就只敢在印度洋东半部活动，哪里有胆子闯过大洋，进入华夏海域之中，
<br/>　　“先生，这个恐怕……”
<br/>　　黑人首领强忍着剧痛，正要向叶尘解释。
<br/>　　然而就在这个时候，远处海上忽然传来了一阵轰鸣声。
<br/>　　黑人首领的一名手下道：
<br/>　　“老大，不好了！好像是华夏海军追过来了！”
<br/>　　众海盗顿时个个脸色大变。
<br/>　　黑人首领也不敢擅自做主，转而望向叶尘。
<br/>　　叶尘摆了摆手，
<br/>　　“来的正好，命人迎上去！”
<br/>　　黑人首领顿时两眼一瞪，
<br/>　　“先生，对方要是开火怎么办？”
<br/>　　叶尘冷冷道：
<br/>　　“这个你不用管，只需要按照我的吩咐去做！”
<br/>　　黑人首领听到这话，顿时身体一颤，不敢再多说。
<br/>　　随着海盗船迅速靠近，不一会的功夫，一艘军舰出现在前方海面之上，上面赫然飘扬着一面代表华夏国的红旗。
<br/>　　“你们在这里等着，我去去就来！”
<br/>　　说完这话，叶尘直接腾空而起，在众人无比震撼的目光中，踏空而行，直奔那艘军舰而去。
<br/>　　而此时，那艘华夏国的军舰之上，众人看到眼前的一幕，也顿时炸开了锅，
<br/>　　“那是什么东西？”
<br/>　　“好像是个人！而且还是从海盗船上来的！”
<br/>　　“难道是那帮海盗请的帮手？”
<br/>　　“敌袭！快开火！”
<br/>　　……
<br/>　　随着军舰指挥官一声令下，一枚枚炮弹飞射而出，朝叶尘铺天盖地袭来。
<br/>　　后方的海盗船上，黑人首领等人顿时纷纷面露喜色。
<br/>　　“老大，那小子竟然就这么冲过去了，这次肯定死定了！”
<br/>　　“就算他是神仙，也挡不住华夏国的炮火吧？”
<br/>　　“没想到啊！反倒是华夏海军救了我们！”
<br/>　　“咱们快赶紧逃吧！”
<br/>　　……
<br/>　　众海盗之前被叶尘逼迫自断一手，全都恨的牙根直痒痒，如今看到叶尘自寻死路，顿时一个个兴奋不已，有的甚至已经开始肆无忌惮在庄菲菲的娇躯上打量起来，脸上纷纷露出一抹淫笑。
<br/>　　而庄菲菲等人则全都脸色大变，望着前方那漫天的炮弹，还有那道瘦弱的身影，顿时一阵绝望。
<br/>　　然而，
<br/>　　接下来发生的一幕，却所有人目瞪口呆。
<br/>　　只见，
<br/>　　那漫天的炮弹爆炸过后，天空上那道瘦弱的身影，却毫发未损，依旧在踏空而行，朝华夏军舰逼近。
<br/>　　“什么！”
<br/>　　无论是军舰上华夏海军，还是后方海盗船上的众人，看到这震撼的一幕，全都被震慑住了。
<br/>　　继而，
<br/>　　一道犹如九天神雷般的声音，陡然在海面上响起，
<br/>　　“华夏军人听着，我乃华夏叶狂仙，让你们的最高指挥官出来见我！”
<br/>　　轰隆隆！
<br/>　　声音悠悠扬扬的在海面上传递开来，清清楚楚的传递到每个人的耳中。
<br/>　　华夏军舰上的众人反应过来之后，顿时纷纷惊呼起来，
<br/>　　“是叶仙人！我之前在网上见过，叶仙人一人横扫了米国数万军队，逼的米国国会投降认输！”
<br/>　　“没错！的确是叶上将！”
<br/>　　“什么叶上将？你们还不知道吧，叶仙人已经被国家授予元帅军衔，如今已经是叶帅了！”
<br/>　　……
<br/>　　众人纷纷欢呼起来，有人甚至直接跪俯在地上，开始顶礼膜拜。
<br/>　　要知道，叶尘的事迹在军中流传开来之后，如今已经是所有华夏军人心中的神，甚至崇拜程度，远远超过了当初的叶天歌！
<br/>　　“远洋号指挥官李军洋，拜见叶帅！”
<br/>　　一名肩头扛着两杠三星的中年海军军官，快速来到军舰的最前方，冲着天空上的叶尘躬身行礼道。
<br/>　　叶尘身体一晃，便跨越数百米的距离，来到军舰之上，
<br/>　　“李上校不必多礼，前面的海盗船上有178名华夏人质，你去把他们接应过来！”
<br/>　　叶尘这话才刚刚说完，却猛然发现后方的海盗船，突然朝反方向加速离去！
<br/>　　很显然，那些海盗打算逃跑！
<br/>　　叶尘见状，不由得眉头一皱，
<br/>　　“原本还打算饶你们一命，既然你们不知死活，那就去死吧！”
<br/>　　声音落下的同时，李军洋等人，只感觉眼前一花，叶尘已经消失不见，下一刻已经在出现在十里开外，几乎瞬间就追上了那艘想要逃跑的海盗船……
