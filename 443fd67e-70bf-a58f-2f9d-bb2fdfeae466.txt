　　　　一把年纪的老头，被吓的不轻，仿佛灵魂都在颤栗。
<br/>　　他这种怕，不是一般人看到一条蛇的惧怕，更像是心中秘密被洞悉的恐惧。
<br/>　　赤果果的，无处遁形。
<br/>　　赵老爷子年轻的时候可是有名的捕蛇人，岂能会怕一条蛇，但眼前出现的这条蛇，真的让他怕了，让他惶恐不安。
<br/>　　一条蛇盯着一个人看，看的他头皮发麻。
<br/>　　涂小安就这样看着匍匐在地的赵老爷子，这样一位德高望重的老人，如今在他眼中，是那么的可恶跟不堪。
<br/>　　人面兽心，不过如此吧。
<br/>　　“她姓什么”
<br/>　　金银蛇望着悬浮的红心木，幽幽的口吐人言。
<br/>　　赵老爷子匍匐着浑身打哆嗦，不敢不答：“姓赵，叫赵欣兰”
<br/>　　“她应该会是一名很出色的人民教师吧，可最后却惨死在了你们赵家”
<br/>　　赵老爷子闻言，老眼一凸，高高在上蛇王，果然洞察一切，这段尘封五十年的往事，世上除了他几乎没有人知晓，就算是待在赵家几十年的管家，也不知情。
<br/>　　秘密真真切切的被揭发，他再无侥幸。
<br/>　　“小人有罪，小人有罪”
<br/>　　赵老爷子惶恐的再次跪起了头，嘭嘭嘭作响，样子甚是可怜。
<br/>　　如若平日，一个八十多岁的老人，这般样子，涂小安还真于心不忍，但他真的可怜吗，那女子才叫可怜，一尸两命，事后连全尸都没有，骨头被做成一块红心木，替赵家看宅积福。
<br/>　　六月飞霜。
<br/>　　那条有情有义的鸡冠蛇就更不用说了，涂小安对它是有愧的。
<br/>　　“你既然知道有罪，那应该知道怎么做了吧”
<br/>　　涂小安冷淡无波的看着赵老爷子，那双蛇眸冰冷之极。
<br/>　　“小人...小人知道”
<br/>　　赵老爷子浑身颤栗，今晚终归逃不过一死。
<br/>　　蛇王在前，他连一丝反抗都不能，先前那把匕首再次出现，赵老爷子缓缓的将匕首抵在了自己的脖子上。
<br/>　　握着匕首的手，充满颤抖，空气中弥漫着令人窒息的味道。
<br/>　　一秒。
<br/>　　二秒。
<br/>　　三秒。
<br/>　　赵老爷子张大的瞳孔中充满恐怖，每一秒钟的滴嗒声，都像是一把铅锤在他的心上敲击了一下。
<br/>　　涂小安静静的冷眸看着，看着这位老头自尽，这也是完成鸡冠蛇没有完成的心愿。
<br/>　　可...赵老爷子却迟迟没有动手，那张满是皱纹的老脸，惊恐之极。
<br/>　　他怕了，他畏惧死亡。
<br/>　　“你不想死对吧”昂首而立的金银蛇忽然一语：“或许我可以给你一个将功补过的机会”
<br/>　　赵老爷子几乎就已经绝望了，闻言，顿时喜上眉梢，匕首悄然放下：“请蛇王大人明示”
<br/>　　此等表情呈现，涂小安满心的失望，看来人都是怕死的，能活谁想死。
<br/>　　越老也越怕死。
<br/>　　对很多老年人来说，“死”或与“死”相关的各种字眼，似乎成了他们最大的忌讳，而且，好象随着时间的推移，年龄越大越害怕死亡。
<br/>　　之前赵老爷子在宅院，恳求鸡冠蛇放过赵家一门，决定自尽，这点让涂小安开始的时候有点佩服。
<br/>　　可现如今，他虽然对赵欣兰充满愧疚，可真的让他以死谢罪，到贪生怕死了起来。
<br/>　　全无一点德高望重的样子。
<br/>　　“将你如何对赵欣兰的所作所为，一五一十的给我写在纸上，然后对着赵宅村的村民面前说出来”
<br/>　　那个人民教师，不该无声无息的死，惨死了，骨头还要被做成红心木，帮赵家看宅积福五十年。
<br/>　　涂小安本来想给赵老爷子最体面的一种死法，可惜，他并不是诚心的悔悟。
<br/>　　否则整整五十年，红心木不可能高高悬挂赵家房梁，赵欣兰如何能瞑目。
<br/>　　这种将尸体骨头做成红心木，是一种阴邪的做法，鬼魂不得超生，化为家蛇。
<br/>　　跟涂小安当初死后想化为家蛇守护家人是一个意思。
<br/>　　只是他是出自自愿，又有系统加身，才完成家蛇重生，但赵欣兰死的何其冤枉，一尸二命不说，死了还不得超生，被困在赵家。
<br/>　　赵老爷子闻言，不敢有半点的反驳，比起不用死，这点算的了什么。
<br/>　　再说赵欣兰本来就不是他所杀的，是他以前的原配老婆杀的，与他何干，顶多颜面无存。
<br/>　　就这样，一张白纸上，密密麻麻的写上了一段尘封五十年的往事，记录了一位年轻女老师的人生悲剧。
<br/>　　赵家门前，已经汇聚了赵宅村的所有村民。
<br/>　　原来这个夜晚，整个村开始都陷入了恐慌，之后又看到蛇王显化的大吉之兆，一个个在黑沉沉的夜，挨家挨户的讨论起来。
<br/>　　此时，赵家人出门叫一个个村民召集到赵家门口，说是赵老爷子大事宣布。
<br/>　　赵家家蛇杀人，轰动一时，半夜蛇王显化，降妖伏魔，现在赵老爷子将大家召集起来，肯定是说什么了不得的大事。
<br/>　　蛇王在赵宅村显化，这可是惊天动地的大事，等天一亮，传到白镇，几乎能让整个村都风光一把。
<br/>　　村民非常非常的兴奋，就等赵老爷子出来说话了。
<br/>　　坐着轮椅的赵老爷子被管家推了出来，没有一个人知道赵老爷子究竟到底想说些什么。
<br/>　　赵老爷子望着百来口村民，黑压压的人头，他的手粗糙得像老松树皮，裂开了一道道口子，手心中用力的握着一张写满字的纸。
<br/>　　今天之后，或许他可以活，但也名誉扫地了。
<br/>　　蛇王要他自己解开自己的伤疤，也仅仅比死好受一点。
<br/>　　好受也仅仅对他而言罢了。
<br/>　　“各位，我赵昌民，深更半夜将大家召集起来，是想告诉大家一件事情”
<br/>　　一个村民兴奋的说：“村长，是不是蛇王大人给你留下什么话，要传给我们”
<br/>　　“蛇王在我们赵宅村显化，这是几百年修的福呀”
<br/>　　那些信奉蛇王传说的村民非常的开心，跟中了五百万大奖似的。
<br/>　　这其实不难理解，比如你在自家附近看到了一条神龙显化，威严不凡，你激不激动。
<br/>　　赵老爷子死死的攥着那张纸，张了张嘴，可却无法念出来。
<br/>　　念出来之后，就是自己打自己的脸，赵家几乎就完了。
<br/>　　可要是不念，那无形之中有道蛇眸死死的盯着他，如影随形。
<br/>　　蛇王想杀他，易如反掌。
