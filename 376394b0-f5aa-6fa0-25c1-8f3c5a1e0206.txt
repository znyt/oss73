　　也许其他人不懂的掌控了造化之力的可怕。
<br/>　　可是这转轮和尚的身份，非同寻常，他来自的净土星域之中，有着许多曾经佛门无敌强者记载而下的古籍。
<br/>　　曾经的许多佛门无敌强者，甚至是许多佛祖，都尝试过想要掌控造化之力。
<br/>　　可无一人成功。
<br/>　　佛祖，可是足以媲美仙帝的无敌强者，在任何的时代之中，都是无敌的存在。
<br/>　　佛祖都做不到的事情，今日居然是在楚风眠的身上出现了，让他如何不震惊。
<br/>　　“此人的来历实在是太可怕了，不知道此人是有着奇遇偶然而得，还是背后有着惊天来历！今日绝不能跟此人在战下去！”
<br/>　　转轮和尚看向楚风眠的眼神之中，露出深深的忌惮，不愿在战。
<br/>　　本来转轮和尚，还打算借助这佛寺之中的种种佛门禁制之力，来跟楚风眠一战的。
<br/>　　毕竟在这佛寺之中，转轮和尚可以源源不断的汲取佛门之力，堪称无敌。
<br/>　　可是这造化之力一出，一改他的想法，一旦是楚风眠连续用造化之力轰击他，只怕是他都会来不及恢复，都会被当即斩杀。
<br/>　　佛门的许多强者佛尊，可都是死在天劫之下，这转轮和尚可不想要落得如此一个下场。
<br/>　　“至少救下此人再走，此人也是圣龙秘境的弟子，定然对着小子有所了解！”
<br/>　　这转轮和尚目光闪烁，放弃跟楚风眠争斗，直接冲着小耀王寒夜冲了过去。
<br/>　　可楚风眠的速度却是比起转轮和尚要更快，在打破了转轮和尚的救赎之光后，楚风眠剑锋当即落下。
<br/>　　这小耀王寒夜的身躯，顿时是在剑锋之下，爆炸开来，肉身被无情摧毁，精血满天飞舞。
<br/>　　“吞天秘术！”
<br/>　　几乎是在小耀王寒夜身躯被打破的瞬间，吞天秘术催动而出，当即是开始疯狂的吞噬这小耀王寒夜的精血。
<br/>　　一旦是所有精血被全部吞噬，便是代表小耀王寒夜的陨落。
<br/>　　“放过我！楚风眠，不楚兄，放了我，我愿意为你效力，为你去对付无道！甚至是让圣院宗为你效力！”
<br/>　　小耀王寒夜疯狂的咆哮道，他被楚风眠击败，无敌之心以破。
<br/>　　现在哪里还有当初的一代霸主的模样，已经是成为了一条可怜狗，不断的哀求着楚风眠。
<br/>　　楚风眠听到这小耀王寒夜的哀求，脸色不为所动，这小耀王寒夜，无数次的算计楚风眠。
<br/>　　楚风眠可不会将这样一个随时算计他的人，放在身边来。
<br/>　　“死！”
<br/>　　楚风眠冷哼一声，顿时灵力运转注入到吞天秘术之中，这吞天秘术力量当即加快，疯狂的吞噬这小耀王寒夜的精血。
<br/>　　“众生佛法！生死轮回！”
<br/>　　突然之间，转轮和尚咆哮一声，气息暴涨，在身躯之上，更是燃烧起了白色火焰。
<br/>　　这白色火焰，不是楚风眠所见到过的任何一种火焰，说是火焰，在其中却有着死亡的气息。
<br/>　　转轮和尚燃烧起了白色火焰，刹那之间猛然冲入到了吞天秘术之中。
<br/>　　这突然的举动，让楚风眠都没有想到，这转轮和尚，居然是敢冲入到吞天秘术之中。
<br/>　　要知道这吞天秘术的强大，九劫古帝深入其中，都是必死无疑，哪怕是这转轮和尚再强，这也是一种找死的行为。
<br/>　　为了救这小耀王寒夜，转轮和尚也是不惜代价了，居然是肉身进入吞天秘术之中，打算救人。
<br/>　　楚风眠惊讶刹那，便是冷喝一声，吞天秘术的目标，转换为这转轮和尚，吞天之力，旋即催动起吞天秘术来。
<br/>　　楚风眠所有的力量灌入到吞天秘术之中，吞天秘术的吞噬力，顿时攀升了数十倍。
<br/>　　吞天之力，疯狂的吞噬这转轮和尚身上的灵力。
<br/>　　可是在这转轮和尚背后，燃烧起的这一层白色火焰，居然是在抵挡着吞天秘术的力量。
<br/>　　足以吞噬诸天的吞天秘术，居然是第一次对一个人，无可奈何。
<br/>　　“这是什么秘术？”
<br/>　　楚风眠眉头一皱，他猛然望向这白色火焰，居然是看到这白色火焰之中，居然是浮现了一道巨大无比的六角轮盘。
<br/>　　这六角轮盘，仿佛代表着诸天生死，佛门之中的生死轮回之道，在其中体现的淋漓尽致。
<br/>　　“六道轮回！这是佛门之中最为神秘的轮回宗的秘术！”
<br/>　　楚风眠突然想到了一个古老的传闻。
<br/>　　据说是在佛门之中的无数分支之中，最为繁盛的，当属净土宗，最为强大的，当属须弥宗，但是最为神秘的，却当属轮回宗。
<br/>　　这三宗，便是号称佛门之中的三大宗门，也是佛门的核心。
<br/>　　诸天轮回便是轮回宗的根本。
<br/>　　据说轮回宗的强者，足以不入轮回，转世重生，掌控轮回之力。
<br/>　　对轮回转世，楚风眠曾经的不屑一顾的，但是在他重生归来之后，突然才发觉，这所谈轮回之事，并非是虚无缥缈。
<br/>　　对于轮回宗，楚风眠知晓的并不多，他也曾经想要去寻找关于轮回宗的记载，却是寥寥无几。
<br/>　　轮回宗，在佛门之中都神秘无比，更何况在外，据说轮回宗的弟子，更是万年之间都难以出世一人。
<br/>　　没想到，眼前这转轮和尚，居然施展的是轮回宗的秘术，此人多半是轮回宗的弟子。
<br/>　　“必须要擒下此人！”
<br/>　　楚风眠眼神炙热无比，如果这轮回宗内，真的有着关于轮回，转世重生的记载，也许楚风眠也可以从其中，寻找到他二世为人，转世重生的秘密。

<br/>　　楚风眠刹那之间，凝聚血脉之力，从天凝聚，化为了一尊巨大无比的大印，当即镇压而下。
<br/>　　可是当着龙皇大印，即将镇压在转轮和尚身上的一刻，他背后的白色火焰，其中的六道轮盘，将这龙皇大印的力量，全部抵挡而住。
<br/>　　在这六道轮盘之内，仿佛是天下间的一切力量，都难以对着转轮和尚，造成伤害。
<br/>　　“自在天地！”
<br/>　　就当楚风眠再度出手的一刻，这转轮和尚的身形，突然之间化为了一道金光，当即飞入到了佛寺的通道之中，消失无踪。
<br/>　　临走之时，这转轮和尚，抓走了小耀王寒夜的一团精血。
