　　胡梦浩当然答不出来，但却不影响他面瘫的继续维持镇定。
<br/>　　曹春辉最初的惊疑不定此时已经彻底放下了心，确定胡梦浩根本翻不出浪花。
<br/>　　罗杰的耐心也到达了极限，他盯着胡梦浩，如果此人不能给他一个满意答复，他保证不会让其活到明天。
<br/>　　在屋内的一片安静中，从未开过口的楚云天却忽然出声：“我好想明白了胡梦浩副院长的意思。”
<br/>　　“什么！？”曹春辉大惊，不可置信的望着楚云天，不明白这个家伙怎么这时候强出头。
<br/>　　罗杰关注胡梦浩的目光，也转移到了楚云天身上：“说说看。”
<br/>　　楚云天点头道：“其实很简单，胡梦浩副院长其实也都说了出来，只是我等没能第一时间领会，早在文明时代的时候，胡梦浩副院长就是门萨内智力排名第二的存在，无论是曹春辉院长还是我，都被他甩开了好几条街。”
<br/>　　“天才的想法有时候的确很难理解，好在这段时间我与胡副院长也算熟悉，对他过度跳跃的思维方式有一定适应能力。”
<br/>　　说到这里楚云天淡然一笑：“胡梦浩副院长一开始就提到，安罗市的丧尸清理一定有丧尸引诱剂的功劳，否则如何才能做到将一个城市的丧尸都彻底清除，包括那些蚊虫丧尸呢？”
<br/>　　罗杰被楚云天的话带着走，也对刚得知眼前这个胡梦浩竟然是门萨排名第二的存在，感到相当惊讶。
<br/>　　看到罗杰能听进去，楚云天定了定神，继续道：“紧接着胡梦浩副院长才冒昧的反对了曹春辉院长的提议，并提出了另一想法，也就是丧尸如何控制的问题，这两者间是有联系的。”
<br/>　　“这时候的曹春辉院长很不满意，在胡梦浩副院长想要继续说下去的时候，很不凑巧的呵斥了几句，于是在接下来罗杰少爷具体问的时候，胡梦浩副院长因为被院长呵斥而心情不好，一气之下不想明说了。”
<br/>　　说到这里的楚云天再次一笑，颇有些劝诫意味的看了胡梦浩一眼：“副院长啊，我知道您很多时候不喜反复解释，更讨厌在您最专业的方向被不如您的人质疑，天才嘛，都是孤独的！”
<br/>　　“但是您也要明白，罗杰少爷并不懂专业研究，您又何必为了跟曹院长置气而不给罗杰少爷面子呢？”
<br/>　　楚云天弯弯绕绕的，先是帮胡梦浩开脱，又将其捧上高峰，还黑了一把曹春辉，说研究上曹春辉不如胡梦浩。
<br/>　　曹春辉当场气的面色煞白，声音都尖锐了起来：“你这个老不死的东西，老子好吃好喝的供着你，你竟然在这里跟胡梦浩串通一气来针对我？好啊！你俩倒是说说，这控制丧尸的研究重点到底在哪，说不出来我就当场宰了你们！”
<br/>　　罗杰眯起了眼睛，看着这三人的争执，露出了一抹嘲讽的笑容：“我只给最后一次解释的机会，胡梦浩你接着把没说完的话说下去，要是解释的出来，这院长位置今日起就是你的，从此再无人利用权威压着你，但要是解释不出来，你就去当生化试验的活人试验品吧！”
<br/>　　这一朝着两处极端发展的话一出口，曹春辉就当场吓的浑身颤抖，他没想到今日的一次会面，竟然让他院长的位置不保，但很快曹春辉又定了定神，心道胡梦浩和楚云天两人说了半天，都没将重点说出来，说明根本就是在胡说八道，哪有什么重点研究方向，根本就是在故意拖延时间。
<br/>　　原本胡梦浩的确如曹春辉认定的那般，毫无方向不说还差点将自己陷入危机，但经过了楚云天的一番逻辑整理，反而脑中灵光一闪。
<br/>　　他抬头直视着罗杰，眼中带着楚云天给他安排人设中天才固有的傲气和坚定，道：“丧尸引诱剂可以诱导丧尸发狂，让它们的行动方向改变，其实就已经初步进入了如何控制的门槛，所以我才说这一研究方向的重点在楚涵在狼牙那里，想必他们一定有了初步想法，才会出现丧尸引诱剂这一发明。”
<br/>　　“所谓万事开头难，狼牙研究部都已经走出了第一步，我们接下来只需要方向正确，想要彻底搞明白丧尸如何控制这一问题，也只是时间问题。”
<br/>　　胡梦浩的一番话说的流畅自然，气场更带着天才科学家的傲然，以及在自己熟知领域上的绝对自信。
<br/>　　罗杰将信将疑的神色瞬间变化，显然这一解释他非常满意，并且内心也极其认同。
<br/>　　曹春辉则就瞬间面色煞白，他没想到胡梦浩真的有想法，竟然还合理，让他想从中找漏洞都一时半会儿找不出来。
<br/>　　楚云天淡定的低下头，掩去了眼中的一抹笑意。
<br/>　　罗杰看向胡梦浩的眼神不禁带上了欣赏，面带喜悦的说道：“我怎么没想到呢？这方向根本就是很明显啊！哎我说你们其他人怎么也没想到呢？曹春辉？！”
<br/>　　最后一个点名，罗杰的声音更是刷然冷了下来。
<br/>　　胡梦浩接着说道：“这一研究主题早在拿到丧尸引诱剂的时候我就有想法了，只不过那时候的曹院长只允许我琢磨这一药剂如何调配，根本不给机会去研究其他，这一想法才被搁置下来，若不然……”
<br/>　　接下来的话胡梦浩没有继续说下去，只是烦躁又厌恶的皱了皱眉。
<br/>　　事态发展到这个地步，胡梦浩没说完的话其他人哪里还不明白？
<br/>　　罗杰一下子就扭头朝着面色惨白的曹春辉大喝：“你个蠢货！拿着我的赞助在这里一事无成！还压制研究项目无法进展，要不是你这个废物什么都不懂，这丧尸的控制方法说不定早就研究出来了！”
<br/>　　“不，不是这样的！”曹春辉又一次跪下，朝着罗杰连连磕头。
<br/>　　大怒之下的罗杰暴躁的再次咆哮：“竟然敢在我面前耍心思，连那些药剂的调配都不是你的功劳，你还有脸在我面前邀功？这两年你过的很好啊，在这里为所欲为影响我的大事进度！难怪我花了这么多代价培养起来的研究院，却处处不如狼牙那个小小的研究部，都是你这个废物干的好事！给我拖下去，扔进丧尸群！”
