<br/>  滚烫泛红的铁水流出，顿时让所有人都是一惊，哪怕是在此干活的工匠都是这样。<br/>　　<br/>　　  他们都从事冶铁多年，能直接溶炼成铁水的可并不多见。<br/>　　<br/>　　  “这么短的时间就成了？”黎正惊讶道。<br/>　　<br/>　　  “你们发现没，这流出的铁水很干净，好像并未沾染太多的杂质。”席大师靠近看后，也是惊道。<br/>　　<br/>　　  在一旁的王康笑着默不作声，能做到这般自然是因为地炉的独特设计。<br/>　　<br/>　　  这个地炉的建造，王康很大程度的参造了现代平转炼钢炉的设计。<br/>　　<br/>　　  铁矿在熔炼的过程中，燃料炉渣或者炼铁后的残余与铁液会分离过滤，铁液会逐渐下渗，进到一个特制的容器。<br/>　　<br/>　　  而后流进延伸出来的径管，通向另外一地炉，这样形成一个完成的生产线。<br/>　　<br/>　　  席大师开口道：“康少爷，你能把铁矿熔炼到这种程度，等铁水凝固就已经是上好的铁胚了，若是经过敲打锤炼，质量更是上佳啊！”<br/>　　<br/>　　  他在冶造局就是负责冶铁，自然明白这个道理。<br/>　　<br/>　　  王康摇头：“这还不够！”<br/>　　<br/>　　  若真按席大师说的那样，确实也能够得到钢，但太废人力也太废时间。<br/>　　<br/>　　  钢与铁的本质区别，其实是含碳量的不同，含碳量超过2％的铁，叫生铁；含碳量低于0.05％的铁，叫熟铁；含碳量在0.05％－2％当中的铁，称为钢。<br/>　　<br/>　　  像现在溶炼出的铁水凝固后，就是生铁，生铁的含碳量超标，所以需要继续炼制。<br/>　　<br/>　　  几人都是继续看着，此刻铁水已经流向了另外一个地炉，这个地炉跟之前的一个又是不同。<br/>　　<br/>　　  因为这个地炉里面没有层层的架构，里面类似于是一个密闭的圆桶。<br/>　　<br/>　　  此刻，这个地炉里的火早就生起，而且经过这么长时间的闷蕴，温度已经极高。<br/>　　<br/>　　  而在这时，有工匠从地炉一侧的开口倾倒进了铁粉。<br/>　　<br/>　　  看到这一幕，黎正一惊，忙着道：“你已经将铁溶炼到这种程度，为什么还要加入铁粉？”<br/>　　<br/>　　  “这不是多此一举？将原本的熔炼毁掉吗？”<br/>　　<br/>　　  “康少爷，这是败笔啊，快让他们停下！”席大师也忙着道，他们根本就无法理解。<br/>　　<br/>　　  闻言，王康淡淡道：“经过第一炉炼制出来的，可以称之为生铁，生铁的含碳量高，所以需要二次熔炼。”<br/>　　<br/>　　  “什么含碳量？这是什么东西？”黎正几人听得都是有点懵逼。<br/>　　<br/>　　  “含碳量就是判断炼出材质的一个标准。”王康道。<br/>　　<br/>　　  可这话让他们更是迷糊。<br/>　　<br/>　　  王康一拍脑门，“你们还是先看着吧，等会我再给你们解释。”<br/>　　<br/>　　  此刻，铁粉的倾倒已经结束，由于是靠近地炉作业，工匠热的是直趟着汗水……<br/>　　<br/>　　  “那个又是在做什么？”<br/>　　<br/>　　  王辅臣指着地炉旁，一个工匠在重复转动着一个摇臂，看这个摇臂似乎是从地炉里延伸出来的。<br/>　　<br/>　　  “这个是用来搅拌的。”王康解释道。<br/>　　<br/>　　  “搅拌什么东西？”席大师问道，今天这一番所见，确实让他们很是迷糊。<br/>　　<br/>　　  王康的每一步，他们都不能理解，要知道他们三人可都是冶造局的老工匠，在冶铁领域都是专长。<br/>　　<br/>　　  此刻却是向王康问东问西，这一点连他们自己都没察觉。<br/>　　<br/>　　  王康解释道：“加入铁矿粉，同时不断搅拌，利用空气中的氧去掉生铁中一部分的碳，使生铁中的碳含量降低！”<br/>　　<br/>　　  “之所以高温熔炼是为了去除其中的渣，这样就可以直接获得钢！”<br/>　　<br/>　　  王康沉声道：“我将这种炼钢的方法，称之为炒钢法！”<br/>　　<br/>　　  不得不说，古代的劳动人民很伟大，冶铁炼钢一直都居于世界领先地位，大概的发展历程，经过几个阶段。<br/>　　<br/>　　  最早所用的是灌钢法，<br/>　　<br/>　　  而后是块炼法，块炼法就是将铁矿石炼成海绵状的固体块。<br/>　　<br/>　　  因为炼制粗糙，质地较软，含渣滓也多，所以需要经过锻造，简单讲就是烧烧打打，打打烧烧，如此重复……<br/>　　<br/>　　  由此提高碳成分的含量，而且分布均匀，锻造出钢。<br/>　　<br/>　　  这也是百炼成钢的由来！<br/>　　<br/>　　  像王康所处的这个时代，大致就是用的这种方法，只是他们还没意识到有钢的存在。<br/>　　<br/>　　  席大师曾提纯过铁胚的质量，应该就是这般。<br/>　　<br/>　　  而王康所用的是一种更为先进的方法，炒钢法，首次出现在是在西汉中晚期，这项技术的发明是炼钢技术的重大突破。<br/>　　<br/>　　  结合现在的条件环境，被王康采用，不过王康在这个过程中又结合了前世的一些知识，比如地炉的建造……<br/>　　<br/>　　  “炒钢法？”几人愕然。<br/>　　<br/>　　  “钢就是你说的比铁还坚硬的材质吗？”黎正问道。<br/>　　<br/>　　  “对！”<br/>　　<br/>　　  几人的问话，打断王康的思绪。<br/>　　<br/>　　  席大师深吸一口气，“那我们就拭目以待了，如果你真的能够成功，那真是一个创举！”<br/>　　<br/>　　  此刻他们的怀疑也降低不少，因为王康这种新式的炼制方法，又很多未曾听过的理论让他们不由信服。<br/>　　<br/>　　  铁粉与铁水在搅拌之中融合渗透，这个过程同样重要，而且时间，温度也要把握好。<br/>　　<br/>　　  若是碳的成分增多，则没有效果，若是过了，这成了熟铁，等于同样是废了。<br/>　　<br/>　　  不过王康事先就已经想到这个问题，铁矿粉与铁水的配比，也都正好！<br/>　　<br/>　　  这般过了不多时，王康亲自过去指挥，“停止，开炉！”<br/>　　<br/>　　  闻言，所有工匠们也都是一怔，因为最终的成果要出来了，是成功还是失败，就在此刻！<br/>　　<br/>　　  在王康的吩咐下，所有的工匠都去搬动一个摇臂，这是一个机关，可以打开内炉的一个口子，将铁水流出！<br/>　　<br/>　　  很快在一番操作下，经过二次熔炼的铁水，顺着事先布好的通道流了出来。<br/>　　<br/>　　  流向的位置是一个土模！<br/>　　<br/>　　  这个土模成圆形，长有三米多，如果有人注意，就会发现这个长度正好是跟开挖出隧道的高度相同！<br/>　　<br/>　　  “康少爷你是要直接做成品吗？不需要锤炼？”席大师惊问道。<br/>　　<br/>　　  “不需要！”王康笑着应道。<br/>　　<br/>　　  “第一个已经完成，换下一个模具，快点！”工头大喊道。<br/>　　<br/>　　  很快有人立即放置新的模具，继续灌入铁水。<br/>　　<br/>　　  而这时席大师却直接脱了上衣，对着王康大声道：“康少爷，我要亲自检验，这炼制出来的是否如你所说的那样，比铁还要坚硬！”<br/>　　<br/>　　  “哈哈！”<br/>　　<br/>　　  王康大笑道：“随你检验……”<br/>　　<br/>　　  【作者题外话】：看我的书，不但要你们爽，还要让你们学到知识，还有一更，银票，收藏<br/><br/><br/>
