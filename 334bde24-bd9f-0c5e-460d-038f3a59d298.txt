　　虽然叶尘早就预料到，赫连铁树会找上门来，却没有想到，他竟然来的这么快！
<br/>　　就在叶尘准备起身，将其灭掉的时候，高台上那名紫袍老者，忽然开口了，
<br/>　　“赫连将军，老夫不知道你和这位先生是什么恩怨，不过他既然在我们万象拍卖场之中，那便是我们的顾客，还请您遵守我们拍卖场的规矩！”
<br/>　　“你说什么？”
<br/>　　赫连铁树显然十分恼火，不由得两眼一瞪，身上释放出一股强大的气势。
<br/>　　轰！
<br/>　　一些离得近的，顿时被那股强大的气势，压得透不过气来。
<br/>　　然而，
<br/>　　那紫袍老者阿布多，却依旧一脸平淡，无悲无喜道：
<br/>　　“老夫还是那句话，无论是谁，既然进了我万象拍卖场，那就要遵守我们拍卖场的规矩，谁也不能例外！”
<br/>　　“倘若您要找的人，愿意自己出去，老夫自然没有意见，否则就请您等到拍卖会结束之后，再解决你们的私人恩怨！”
<br/>　　“若是任何人在这里闹事，那便是和我阿里商会为敌！”
<br/>　　赫连铁树听到这话，眉头顿时皱的更紧了，
<br/>　　阿里商会，乃是魔界第一商会，势力遍布七大主城，便是魔皇宫都不敢随意招惹。
<br/>　　最主要的是，眼前这个阿布多，乃是阿里商会的高层，向来以脾气古板著称。
<br/>　　一念及此，赫连铁树不由得深吸了一口气，又狠狠瞪视了一眼叶尘，嘴角露出一抹残酷的笑容，寒声道：
<br/>　　“小子！你若是现在乖乖自己滚出去，本将军或许可以留你一条全尸！”
<br/>　　纳兰柔然和戴安娜两人，早已经吓得浑身娇躯颤抖。
<br/>　　整个魔界都大名鼎鼎的赫连铁树！
<br/>　　她们又岂会没有听说过？
<br/>　　这可是真正的冲霄境强者！
<br/>　　纳兰柔然心惊胆颤之余，不由得朝旁边的叶尘望去，却见他脸色平静依旧，只是淡淡瞥了一眼赫连铁树，缓缓道：
<br/>　　“你若是现在离开，我可以饶你一条狗命！”
<br/>　　轰！
<br/>　　叶尘这话一出口，不但纳兰柔然和戴安娜两人，险些一头晕厥过去。
<br/>　　整个拍卖场，也瞬间爆炸，顿时全都开始议论纷纷起来。
<br/>　　“这小子什么来头？竟然敢和赫连将军这么说话！”
<br/>　　“你们都还没有听说吗？据说赫连将军府的三公子赫连鹏飞，被当众打成了一滩烂泥，看样子，凶手应该就是这家伙了！”
<br/>　　“不是吧！这家伙也太彪悍了吧！连赫连将军府都敢招惹？”
<br/>　　“杀了人不赶紧逃跑，竟然还敢来到这里，难不成，他以为，区区一个万象拍卖场，便能够保住他的小命吗？”
<br/>　　“赫连将军府的实力，可并不比阿里商会差多少，纵然赫连将军不愿意和阿里商会交恶，但想让他放弃杀子之仇，也绝无可能啊！”
<br/>　　……
<br/>　　众人议论纷纷，
<br/>　　望向叶尘的目光之中，惊愕、怜悯、嘲弄……各种神态，不一而足。
<br/>　　而赫连铁树听到叶尘这话，更是气得肺都要炸了，恨不得立即冲过去，把这个可恶的家伙生吞活剥了。
<br/>　　不过，他到底也是枭雄人物，知道阿里商会轻易不可招惹，最终还是生生忍住了。
<br/>　　“好好！小子，本将军就让你再多活片刻，等到拍卖会结束，便是你的死期！”
<br/>　　说完这话，赫连铁树直接走出了拍卖厅，不过在临走之前，狠狠瞪了叶尘一眼。
<br/>　　然而，叶尘却仿佛没有看到他一般，神色平静依旧。
<br/>　　就连众人看到这一幕，也不由得又再次悄悄议论起来，
<br/>　　“这小子的实力如何，虽然还看不出来，不过面对赫连将军，竟然还能如此淡定，单单是这份胆识，恐怕天下也没几个人能做到！”
<br/>　　“嘿！我看他也不过是死撑罢了，其实心里多半怕得要命！”
<br/>　　“说不定此人，有着什么深厚背景，也未可知呢！”
<br/>　　“一切等拍卖会结束之后，自然便见分晓！”
<br/>　　……
<br/>　　眼看着整个拍卖场，再次喧嚣起来，高台上的那位紫袍老者阿布多，当即重重咳了一声，朗声道：
<br/>　　“肃静！”
<br/>　　“我宣布，本次拍卖会，正式开始！”
<br/>　　众人听到这话，这才安静了下来，目光重新转移到拍卖台上。
<br/>　　“接下来，请出我们本次拍卖会的第一件拍品！”
<br/>　　随着阿布多声音落下，一名身材火辣、穿着暴露的魔族女郎，捧着一个白色玉盘，姿态优雅的从后台走了出来。
<br/>　　而在那白色玉盘之中，则是一株紫色叶瓣的植物，根部则是白色，周身有淡淡的流光浮现，一看就知不是凡品。
<br/>　　“诸位请看，这是一株八百年份的紫叶甘罗，对修炼神魂力量，有着极强的效果……”
<br/>　　阿布多将第一件拍品的功效，仔仔细细讲解了一遍之后，
<br/>　　“一号拍品，起拍价十万魔珠，现在开始竞拍！”
<br/>　　随着阿布多声音落下，很快便有人加入了竞拍行列。
<br/>　　“我出二十万魔珠！”
<br/>　　“我出三十万！”
<br/>　　“三十五万！”
<br/>　　……
<br/>　　最终，这第一件拍品，以五十五万魔珠的价格，被一名贵宾包厢的贵族买了去。
<br/>　　叶尘看到这一幕，不由得暗暗感慨，
<br/>　　眼前的这种场景，勾起了他前世的记忆，熟悉而又遥远。
<br/>　　一眨眼的功夫，已经拍出了数十件拍品，而整个拍卖场的氛围，也达到了一个白热化的状态。
<br/>　　很多人纷纷为了同一件宝贝，争相竞拍，互不相容，争得的脸红脖子粗。
<br/>　　而叶尘见自己等的东西，迟迟没有出现，已经有些不耐起来。
<br/>　　终于，就在叶尘，忍不住直接冲到后台寻找的时候。
<br/>　　“诸位，接下来的这件拍品，和其他拍品有些不同，姑且算是一个奇珍异宝吧！”
<br/>　　“有请我们的第四十五件拍品，冰山美人！”
<br/>　　随着阿布多的声音落下，两名身穿壮硕的魔族大汉，扛着一个用红布包裹的重物，缓步来到高台之前。
<br/>　　众人顿时纷纷议论起来，
<br/>　　“冰山美人？这万象拍卖场，什么时候也开始贩卖奴隶了？”
<br/>　　“我看没那么简单！”
<br/>　　“怎么突然感觉好冷？”
<br/>　　……
<br/>　　伴随着众人的议论声，阿布多微微一笑，随手一挥，
<br/>　　唰！
<br/>　　那张红布直接飞落在一旁，
<br/>　　下方出现一枚三米来高的蓝色冰石，不但华美无比，而且透着一股逼人的寒气，整个拍卖厅的温度，都因此而急剧下降。
<br/>　　而在那精美的蓝色冰石之中，则还有这一道绝美的女子身影。
