　　睁开眼，感受着胸口的隐隐疼痛，苏重长舒一口气。
<br/>　　“还好，活下来了。”
<br/>　　是的，他再一次进行了穿越。每次穿越他都会借尸还魂，宿主状态都不怎么好。
<br/>　　这次的宿主尤其倒霉，苏重刚醒过来就感觉胸口剧痛。
<br/>　　只是下意识查看一下，他就发现自己心脏上竟然插着一根肋骨，自己的肋骨。
<br/>　　这还怎么玩？
<br/>　　宿主就是个十多岁小孩，胸口被人狠狠拍了一掌。肋骨直接刺进心脏！谁特么这么狠啊。
<br/>　　他刚穿越过来，毫无办法，只能花费本源，调动身体力量，把心脏处肌肉收缩生长，将那根肋骨死死夹住。保证血液不外流。
<br/>　　然后他就只能听天由命了。
<br/>　　“不知道是谁救了我？”苏重查询宿主残缺记忆。
<br/>　　一片混乱全无头绪，不是上树掏鸟蛋就是下河摸鱼虾，根本没什么有用信息。
<br/>　　“等等？张小凡是什么鬼？林惊羽又是哪个？”苏重整个人都不好了。
<br/>　　在整个村子的小孩，都是二蛋狗剩之类的名字中，小凡和惊羽这两个名字实在太突兀。
<br/>　　“原来是诛仙啊。”苏重了然。
<br/>　　既然是诛仙位面，他那肋骨插胸口的伤势就不算什么了。都能飞天遁地了，还能治不好他的伤？
<br/>　　起身推开房门，远处是连绵山脉，云雾笼罩好似仙境。
<br/>　　深吸一口气，一股清凉直透身心。
<br/>　　“果然不愧是诛仙位面，空气清新无污染。甚至带着某种活性成分，就是不修炼也能延年益寿啊。”苏重感慨。
<br/>　　“二蛋！你醒啦！”一个灰衣木讷小子兴冲冲跑进院子，看到苏重立刻高喊起来。
<br/>　　苏重嘴角抽了抽，他想起来了，村里小孩名字随意，他就是代表——二蛋！
<br/>　　“小凡啊，这是哪里？”苏重满脸复杂的装迷糊。
<br/>　　听听人家这名字，小凡，还有那个林惊羽。这从名字开始，人家就注定不平凡。
<br/>　　以后如果和别人放对，双方互爆姓名。
<br/>　　一个说我叫张小凡，一个说我叫林惊羽，到了苏重这，怎么办？我叫李二蛋？搞笑呢！
<br/>　　想想就浑身发颤。二蛋是不能二蛋的。还不如叫尔旦，起码还说得过去。
<br/>　　“这里是青云山。是师傅救的你！走，跟我去见师傅。”说着张小凡就拉着苏重跑出门外。
<br/>　　一路往山上奔驰，很快就来到一处精致院落。
<br/>　　“师傅，师傅，二蛋他醒了！”张小凡敲门闯进去，喜悦道。
<br/>　　“毛毛躁躁像什么样子。”一个胖乎乎的到装似严肃的训斥了张小凡一句，然后又温和的看着苏重：“来，我看看你的伤怎么样了。”
<br/>　　苏重老老实实的走上前。即便这个胖子笑的再温和，他也不敢掉以轻心。
<br/>　　强大灵魂让苏重感官敏锐。只是靠近这个胖子，苏重就有一种浓重的危机感。
<br/>　　这是田不易？不愧是一山首座。
<br/>　　“伤势回复的不错。你以后就是我的弟子了，排行第八。对了你原本叫什么名字来着。”田不易收回搭在苏重手腕上得手温和问到。
<br/>　　苏重还在感慨刚才冲入体内的温凉力量，没反应过来。张小凡却直接开口，苏重想阻止都来不及！
<br/>　　“他叫二蛋！”
<br/>　　“二蛋？呵呵……”田胖子无良的笑了。
<br/>　　坐在田胖子旁边的温婉妇人也不由轻笑起来。苏重知道这是田不易妻子，自己未来师娘。
<br/>　　苏重也无奈，小凡啊小凡，你的嘴怎么那么快呢？我还没开口，你就什么都秃噜啦。不是说天生木讷的吗？
<br/>　　“小名，这是小名！”苏重强调。
<br/>　　“那你大名叫什么？”师娘苏茹笑吟吟问道。
<br/>　　“小子大名叫李尔旦。尔雅的尔，旦夕的旦。”苏重可不想顶着李二蛋的名字打天下。
<br/>　　“尔旦？二蛋啊！”
<br/>　　苏重：“……”这还没完了是吧！
<br/>　　“你先在山上修养一段伤势。虽然心脏上的伤口已经长好，还是养一段时间为好。过段时间，再让你大师兄传你修炼之法。”田不易道。
<br/>　　又嘱托了几句，苏重一一点头答应，然后就跟着张小凡回小院修养。
<br/>　　“本以为也是个呆瓜，不过现在看来，还是个心思灵活的。”田不易笑道。
<br/>　　当初通天峰殿前收徒，张小凡木讷平凡，和林惊羽一比差距更是大。而且还顺带被塞了一个病秧子，田不易这几天一直心气不顺。
<br/>　　今天见苏重举止稳重，眼神灵活，显然不是愚笨之辈。这让他心里略微宽慰。
<br/>　　“希望修行资质也能看的过去吧。”田不易感慨。
<br/>　　大竹峰一众弟子都很优秀，但却没有个拔尖扛鼎之辈。他一直对此忧心。
<br/>　　晚上吃饭，苏重和一众师兄弟一起吃饭。
<br/>　　他也认识了几位自己未来师兄。
<br/>　　大竹峰人少，师娘苏茹教导严格。这几个师兄弟不管实力如何，品性却没问题。对苏重这个小师弟表示欢迎。
<br/>　　只是不知道从哪里听说了苏重的名字。一个个二蛋二蛋的叫的欢实。
<br/>　　特别是杜必书和田不易女儿田灵儿。更是逮着他二蛋的名字不放。
<br/>　　苏重都绝望了，自己好不容易穿越到一个仙侠世界，不应该是大袖飘飘仙风道骨的吗，名字叫二蛋，整个画风都不对了好吧！
<br/>　　三天后，还是那个小院。
<br/>　　“二蛋啊……”
<br/>　　苏重看着眼前这个貌似忠厚的大师兄，满脸无奈。自己这个名字算是……火啦？
<br/>　　二蛋就二蛋吧，只要能修仙就成。
<br/>　　“这是咱们青云根本太极玄清道的前四层，我今天来叫你如何修炼。对了，你识字吧。”
<br/>　　“在村里念过几年私塾。”苏重这到没说谎。
<br/>　　他们村里有个老秀才办的私塾，张小凡林惊羽都在哪里呆过一段时间。虽然没学到什么大学问，认字还是可以的。
<br/>　　“那好，我今天先给你讲第一层，秘籍你留着，以后有不懂的就问我。”
<br/>　　然后宋大仁就开始给苏重讲授功法。
<br/>　　苏重听的认真，旁边张小凡一脸云里雾里。他只知道如何修炼。对功法内那些云山雾罩的字句不敢兴趣。
<br/>　　恰恰相反，苏重最感兴趣的，就是那些功法描述文字。其字里行间蕴藏着功法的神韵。他能从这些文字了解功法创造之初的本意。
<br/>　　宋大仁很快讲完第一层功法，然后就离开。张小凡也提着把柴刀去后山砍竹锻炼身体。
<br/>　　苏重因为伤势未痊愈，可以再休息几天。
<br/>　　等人都走了，苏重立刻拿出太极玄清道研究起来。
<br/>　　这可是修仙功法，虽然不能长生，但延寿还是没问题的。
<br/>　　而且战斗力惊人，还能飞天。他之前可从未接触过这种高端功法。
<br/>　　“原来是通过吸引灵气入体，逐渐改变身心的路子。”
<br/>　　苏重很快就明白功法本质。他虽然不知道什么是灵气，但大体也就是一种高等能量。
<br/>　　太极玄清道就是通过这种能量改善身体素质，同时用灵气孕养精神壮大灵魂。然后精气神合一，修成独有的法力，借此来控制飞剑法宝。达到飞天遁地的目的。
