　　“杀！”
<br/>　　动用了战魂之力的唐古，实力大增，怒喝一声，双手握刀，狠狠的向前一劈。
<br/>　　刺啦！
<br/>　　耀眼的刀光冲天而起，空间被撕裂，惨白色的刀气威势惊天，杀伐无限，笔直的向炎澜天袭杀而去。

<br/>　　恐怖的刀气，那迷漫的锋芒，那凌厉的杀机，如同浩荡天威，庞大而又恐怖。

<br/>　　炎澜天不再留手，双手迅速抬起，掌中黒焰汹涌，立刻形成以一条数千丈长的黒焰魔龙。
<br/>　　昂！
<br/>　　魔龙栩栩如生，身上黑色的鳞甲寒光森森，张牙舞爪，一声怒吼，直接向刀气飞扑了过去。
<br/>　　这魔龙的气势，如同山呼海啸，威势似真龙临世，比之适才的魔莲强大太多了。

<br/>　　魔龙的速度，快到极致，庞大的身躯碾碎虚空，瞬间便临近了刀气，而后巨大的龙空张开，猛然一吸。
<br/>　　惊人的一幕出现了，唐古强大的刀气，居然被魔龙一口吞入了腹中。
<br/>　　而后，魔龙的体内轰鸣不断，而魔龙的速度却是丝毫不减，直扑唐古。
<br/>　　“什么？”
<br/>　　这一次，唐古直接吓得面色巨变，他怎么也没想到，炎澜天的实力，居然强大到了这种地步。
<br/>　　来不及多想，唐古立刻向一旁急速闪避。
<br/>　　就在此时，魔龙巨大的龙爪猛然探出，撕裂长空，向唐古身上狠狠的抓了过去。
<br/>　　龙爪庞大无比，覆盖一方空间，让得唐古根本避无可避。
<br/>　　面对此种局面，唐古只有两个选择，要不硬撼龙爪，要么退出战台。
<br/>　　唐古有心向硬撼龙爪，但感受到那龙爪恐怖的攻击威势，他不由自主的身形暴退，退出了战台。
<br/>　　“炎澜天胜！”
<br/>　　北方看台上，主持比试的褐袍老者朗声开口。
<br/>　　“就你这种废物，还扬言打败我，真是可笑！”炎澜天见此，顿时不屑一笑，随即他意念一动，魔龙化为黑色的火焰，融进了他的身体之中。
<br/>　　“你……！”唐古闻言大怒，但却是无法反驳，他的实力确实是远不如对方。
<br/>　　“唐古，回来吧！”
<br/>　　这时，真枯半圣的声音响起，唐古闻言，只能脸色黯然的返回了看台。
<br/>　　“道宗的顶尖天才还是太弱了！”
<br/>　　“是啊！看来这一届会武，依旧不会有人进入前十名！”
<br/>　　“道宗是一代不如一代了！”
<br/>　　很多观战之人，无不暗暗叹息，唐古的实力与炎澜天差距太大了。
<br/>　　这道宗三大最强天才之一，彻底惨败，不知道那吕罡和林皇两人，是不是也是如此不济。
<br/>　　若两人的实力，也是如此的话，那此届会武的前十名，基本上与道宗弟子无缘了。
<br/>　　一众道宗弟子，个个面色难看，唐古的落败，让他们感觉到巨大的挫败感。
<br/>　　连唐古都败的如此干脆，他们还如何去争？
<br/>　　吕罡和林皇两人，面色无比凝重，他们都清楚唐古的实力，此刻心中也是有些忐忑了起来。
<br/>　　两人感受到炎澜天的强大，在心中暗暗盘算了一下，还真不是对方的对手。
<br/>　　真枯半圣和一众道宗的长老，也是有些气馁，果然啊，道宗顶尖天才，与其它几大势力的顶尖天才，差距还是很大的。
<br/>　　这让他们也是感觉无奈，只能希望接下来的比试中，吕罡和林皇不要败的太惨了。
<br/>　　若是三大顶尖天才，全部被人击败，那道宗的脸面，真的是有些挂不住了。
<br/>　　“哎！”冯光叹了口气，无奈的摇了摇头，他虽然为人不是很正派，但也不想看到道宗弟子一败涂地的局面。
<br/>　　耳边听闻冯光的叹气声，感受到周围道宗弟子的落寞，苏莫心中暗叹，自己等会要不要出手呢？
<br/>　　本来这次会武，他是不想出手的，只想来见一见青璇，但现在道宗的处境似乎不妙！
<br/>　　略一沉吟，苏莫觉得自己还是要出手，帮助道宗挽回颜面事小，得到第一名的成绩，对于他来说还是很重要的。
<br/>　　因为第一名能得到太阴族的重视，还能得到太阴族虚神境大能的一次指点。
