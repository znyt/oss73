
四人边说边往xx俱乐部里走，范二哥和书画在前面，常文书跟王惊蛰并排，挤眉弄眼的跟他说道：“二哥找这地方挺有深意的，你造么？”<br/>　　<br/>　　“多深的意思啊？”<br/>　　<br/>　　常文书朝着右边四十五度努嘴说道：“往那边看，穿着礼服在一起闲聊的三个女人看见没有，眼熟不的？”<br/>　　<br/>　　王惊蛰顺着他大嘴唇子的方向望过去，发现大厅一处沙发上坐着三个打扮靓丽，面容精致的女子，眼熟倒是谈不上，但长得确实不错。<br/>　　<br/>　　常文书低声说道：“这都是常年混迹于此的，有女星有名媛，她们来这里的目的就一个，以身体为本钱来换取自己索需要的一切资源，你能给予她什么你就能干什么，哥们那夜一番长谈我发现你也是此中高手啊，有心思下手么？如果有，你和我或者二哥说一声，随时可以领走一个，嗯，就是领走了想干啥就干啥，然后也不犯法的那种，懂否？”<br/>　　<br/>　　“不懂！”王惊蛰瞪了他一眼，一本正经的说道：“家里有，外面彩旗我就不飘了，好意心领了”<br/>　　<br/>　　“装？”常文书诧异的吐出一个字。<br/>　　<br/>　　“这里面的庸脂俗粉，跟我家里的那个一比，根本就不是一个层次的……”<br/>　　<br/>　　“叮”电梯门开了，四人进来后，从旁边的另外一栋电梯里就走出两个女子。<br/>　　<br/>　　其中一个忽然间猛地扭过脑袋，望着已经关闭了的电梯门，惊愕的张着嘴脸上挂着疑惑和不解，那扇门里有个背影看着莫名的熟悉。<br/>　　<br/>　　“小臣，你看什么呢？”另外一个女子问道。<br/>　　<br/>　　“没什么，好像看错人了”叫小臣的女子摇了摇头，挽着同伴的胳膊说道：“以后你别带我来这地方了，你看那帮男人的眼睛，看你的时候都跟要喷火似的”<br/>　　<br/>　　这女子眯眯着眼睛笑道：“哎呀我不带你来带谁啊，大不了下次你再来的时候，带个灭火器不就得了？”<br/>　　<br/>　　小臣无语的翻了翻白眼，叹了口气说道：“我认识的男人里面，就没有不冒火的”<br/>　　<br/>　　“比如，那个王惊蛰也是？”同样抿嘴笑道。<br/>　　<br/>　　小臣又扭了下脑袋，看着已经升到楼顶的电梯，更加觉得自己似乎是看走眼了，这家俱乐部的最顶层是从来都不对外开放的，只针对于寥寥无几的内部高端会员。<br/>　　<br/>　　小臣回过头来，皱眉说道：“我跟他是两路人，以后能不能别提了……”<br/>　　<br/>　　俱乐部顶层的一个小套间里，四人落座后，没多久就上来酒菜，范二哥拿起毛巾擦着手说道：“先吃一点垫垫胃，待会各展雄风，喝多也没事，楼下就有房间，也可以提供暖床的服务”<br/>　　<br/>　　书画白了他一眼，娇嗔的说道：“请你尊重下，在座的仅有的一个女性，可以么？”<br/>　　<br/>　　范二哥歪着身子，拍了拍她的手，笑着和王惊蛰说道：“惊蛰啊你看我的家教如何？”<br/>　　<br/>　　王惊蛰看了书画一眼，矜持的说道：“二哥，我送你句话”<br/>　　<br/>　　“嗯？”<br/>　　<br/>　　“女子痴情时感人最深，女人无情时，也负人最狠，你还是小心着点吧！”<br/>　　<br/>　　范二哥顿时一愣，才哈哈笑道：“好，这句话回头我纹在背上，谨记！”<br/>　　<br/>　　书画顿时借坡下驴的端起酒杯，笑吟吟的说道：“来，惊蛰，我敬你一杯，他把这话纹在背上不算，我还得装裱上挂在家里，时刻警惕他……”<br/>　　<br/>　　话题展开，一杯酒喝完，气氛就悄然而上了，常文书和范二哥都有心和王惊蛰拉近关系，这提酒的频率基本就没怎么断过，王惊蛰也有心结下范二这个善缘，就敞开了量，也没太拒绝。<br/>　　<br/>　　一个多小时后，几人都已经喝的面红耳赤了，颇有酒过三巡菜过五味的架势。<br/>　　<br/>　　气氛最后被烘托起来的时候，范二哥就挪了下椅子离的王惊蛰近了点，嘴里喷着酒气侧过身子，感慨的说道：“老弟，我是有段日子没喝的这么畅快淋漓了，家里人常年都难得聚在一起，没有什么机会，在外面呢喝酒也是点到即止，为啥？我怕言多必失，有可能无意说错一句话就把自己的一片大好前程都给败了，就像前两年有个主持人在酒桌上唱着曲点评了下那位老人，从此以后他就没有在露头的机会了，所以像今天这么放开了喝，我差不多得有半年没经历过这种场合了，你知道为啥么？”<br/>　　<br/>　　王惊蛰手指摩挲着酒杯，淡淡的笑了笑。<br/>　　<br/>　　范二哥搂了下他的肩膀，意味深长的说道：“因为从前几天起，我就看出来了，我就算言多了不管失了什么，在你这里对我来说都没有什么影响，对不？”<br/>　　<br/>　　“你确定自己看的就这么准？”<br/>　　<br/>　　“我看的准，也是因为你点的准！”范二哥端起茶杯喝了一口漱漱了嘴，沉默半晌后说道：“我谢谢你那一句话的点拨，但我更想知道你是怎么看出来的”<br/>　　<br/>　　王惊蛰瞥了眼在对面和书画闲聊的常文书，心道这人还挺靠谱的，至少他没有把自己的来历告诉范二哥，嘴倒是挺严的，而范二也肯定跟他打听过了。<br/>　　<br/>　　王惊蛰从桌子上拿起烟点了一根，轻声说道：“我解释的太多你未必听得懂，但我要是不说，你可能又会心生惦念，范二哥我只能跟你说，碰见你的那天是申日遇寅木，你面相驿马星为旺为喜，表事业多有升华，额下三道纹路为三月迁移，你面相偏阴，名中犯水，若要去了扬州必然会犯冲，油菜花遍地开多为山地，不犯冲就算是好事一件了，但也不是说了去了上饶一带就万事大吉了，最后送你句话，就是如履薄冰依旧小心为妙吧”<br/>　　<br/>　　范二哥听闻后，眉头就深深的拧了起来，他拿起酒给两人面前的杯子满上，然后举起说道：“我若是再多问，你是不是也不会多说了？”<br/>　　<br/>　　“得道多助，失道寡助，你把这句话记清楚了，比我跟你前面说的那些要管用得多”王惊蛰忽然凑到他耳边，低声说道：“你命中多小人，多堤防吧”<br/>　　<br/>　　范二哥举起酒杯晃了晃，点头道：“受教了！”<br/>　　<br/>　　王惊蛰缩回身子，舔了舔嘴唇，说道：“二哥，过几天我有件事可能会求到你”<br/>　　<br/>　　范二一听当即就愣了下，短暂的呆愣过后，心头莫名一喜，他是真巴不得王惊蛰有事能求到自己身上来呢。<br/>　　<br/>　　人和人之间，点头之交就是一笑而过，从此两路人了，但要是求来求去的，这关系就能越走越近了，很明显范二就想要这一点呢。<br/>　　<br/>　　(=)
