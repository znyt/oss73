　　“这就是，净土佛尊，血邪魔帝？”
<br/>　　楚风眠的目光落在了在这大殿王座之上的这一位老者。
<br/>　　他本以为，这血邪魔帝，已经是坠入佛道，应该是如同一尊杀人无数的魔神一般，凶威浩荡。
<br/>　　血邪魔帝，以血邪为称呼，一生之中，不知道是杀过多少人，何止千万。
<br/>　　可眼前坐在这王座之上的老者，却是如此慈眉善目，让楚风眠都有些无法相信，这就是血邪魔帝。
<br/>　　可从这老者身上散发出来的恐怖魔意来，却是足以证明他的身份。
<br/>　　佛身，魔心。
<br/>　　这就是血邪魔帝，魔族之中最为恐怖的一位魔帝。
<br/>　　血邪魔帝，端坐在王座之上，经历数十万年的时间，却依然肉身不朽。
<br/>　　他的脸庞，看起来如同是活的一般，只是在这王座之上沉睡。
<br/>　　肉身不朽。
<br/>　　哪怕是岁月的流逝，都无法在这血邪魔帝的身上，产生任何一点点的变化。
<br/>　　“不朽真身，这血邪魔帝，也已经是练就了不朽真身，跟那青铜天魔一般。”
<br/>　　楚风眠之前因为曾经见到过青铜天魔，所以对这一幕，并非是太过惊讶。
<br/>　　可是这血邪魔帝身上的力量，却是比起青铜天魔，还要更加的恐怖，在他身上涌现而出的魔意，让这血邪魔帝看起来，简直是像活着的一样。
<br/>　　只要是稍微一动，都会惊醒他。
<br/>　　血邪魔帝，可是曾经魔族之中最为厉害的魔帝之一，实力几乎比肩十方天魔的存在。
<br/>　　而且比起古老的十方天魔，血邪魔帝距离如今的时代之中，并不遥远，只有数十万年的时间，算得上是上古时代的人物了。
<br/>　　青铜天魔，可是远古时代便就陨落的人物，所以楚风眠看到那青铜天魔的时候，才感受到，上面已经是没有多少的力量了。
<br/>　　但是这血邪魔帝的骸骨，却是截然不同，从上面蕴含的力量，简直是这血邪魔帝一旦是苏醒，就足以横扫在场的众人。
<br/>　　哪怕是这血邪魔帝的骸骨，是无比珍贵的宝物，现在也没有人敢对他起贪婪之心。
<br/>　　楚风眠一眼扫过，在这大雄宝殿之中，少帝的身影，赫然出现。
<br/>　　不过纵然是少帝，现在也不敢靠近这王座之上的血邪魔帝。
<br/>　　血邪魔帝纵然是已经是死了数十万年，只剩下一具骸骨，都蕴含着滔天强横的力量。
<br/>　　现在谁敢先去收取，都是死路一条。
<br/>　　少帝也不敢轻举妄动，只是在这大雄宝殿之中，距离这王座不远处，等待着。
<br/>　　血邪魔帝的力量虽然强横，但是他身上的力量，却是在一点点的衰弱。
<br/>　　在这血邪魔帝的胸口之上，一把断剑，正是刺入到了他的胸口之中。
<br/>　　血邪魔帝的肉身，数十万年都未曾腐朽，几乎已经是练就了不朽真身。
<br/>　　像是现在的楚风眠，肉身仅仅是达到了半步不朽真身的地步，都如此强横。
<br/>　　更何况是真正的不朽真身。
<br/>　　传闻之中，真正的不朽真身，乃是无敌的肉身，任何力量，都无法打破。
<br/>　　但是如今这断剑，却是打破了血邪魔帝的不朽真身，刺入到了他的胸口，刺入到了他的心脏之中。
<br/>　　太初剑！
<br/>　　这断剑，就在剑柄之处断裂，剑锋依然刺入在血邪魔帝的胸口之内。
<br/>　　这就是一只在这镇压这血邪魔帝的太初剑锋，樊老的本体。
<br/>　　看着如今血邪魔帝身上散发出的恐怖力量，楚风眠都有点感觉到，如果不是这太初剑锋，在镇压这血邪魔帝。
<br/>　　现在的血邪魔帝，甚至可能是会突然复活。
<br/>　　楚风眠看着眼前这坐在王座之上的血邪魔帝，都不由的想到了这个想法。
<br/>　　血邪魔帝已经是陨落数十万年，死而复生，简直笑谈，可楚风眠的脑海之中，居然是出现了这个想法。
<br/>　　眼前这血邪魔帝，看起来完全不像是一尊死人，而只是一尊被镇压，沉睡之中的魔帝。
<br/>　　“还有一个时辰的时间，这血邪魔帝的力量，就会被镇压在最弱的地步。”
<br/>　　樊老悄悄提醒道。
<br/>　　“恩。”
<br/>　　楚风眠点了点头，目光再度向着大雄宝殿之内看了过去。
<br/>　　除了少帝，在少帝的身旁，还有着几尊强横的魔族。
<br/>　　这几尊魔族的实力，几乎是都超越了蝎狮王，足以是跟龙摩罗那个级别的霸主，实力相近的强者。
<br/>　　这几尊魔族，都站在了少帝的身后，在等待着少帝的命令。
<br/>　　“这几尊魔族之中……”
<br/>　　楚风眠观察这，脑海之中突然察觉出了几分异样的感觉，猛然仔细的看了过去。
<br/>　　就在这几尊魔族之中，似乎是有着两位魔族的样子，让楚风眠突然注意到了。
<br/>　　“这两个人，不是魔族！”
<br/>　　楚风眠脑海之中，突然出现了几分熟悉的感觉，这两尊魔族，楚风眠绝对是没有见到过的。
<br/>　　但是在他们的身上，楚风眠居然是感觉到了几分熟悉的感觉，至少肯定是曾经，跟楚风眠见到过。
<br/>　　楚风眠仔细观察，脑海之中突然想到了两个身影。
<br/>　　“居然是他们两个？”
<br/>　　楚风眠仔细观察这，这两尊魔族，不是其他人，正是转轮和尚跟小耀王寒夜二人假扮而成的。
<br/>　　之前楚风眠跟转轮和尚，小耀王寒夜有过交手，所以他们身上的一举一动，都被楚风眠记住，看到这两组魔族的一刻，楚风眠一下子想了起来。
<br/>　　这转轮和尚跟小耀王寒夜，居然是跟楚风眠一样，杀了两个魔族，假扮成了那两个魔族，潜入到了这净土佛寺之中。
<br/>　　这让楚风眠倒是有些意外，这两尊魔族的实力，可都不亚于龙摩罗，他们二人居然有这样的实力。
<br/>　　尤其是小耀王寒夜，之前是被楚风眠吞噬了九成的精血，已经是失去了九成的力量，只怕是就连寻常的九劫古帝都不如了。
<br/>　　不过现在看起来，小耀王寒夜已经是恢复了实力，看来是这转轮和尚，想办法帮助他恢复了实力。
<br/>　　在这一片藏地之中，了解最深的人，就是转轮和尚，他出身佛门，更是佛门之中最为神秘尊贵的轮回宗的弟子。
<br/>　　转轮和尚知晓这藏地之中的无数秘密，无数宝藏隐藏的所在。
