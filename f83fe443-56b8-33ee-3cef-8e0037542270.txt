　　第428章
<br/>　　千古文人侠客梦，也有人说什么宁为百夫长，不做一书生。
<br/>　　但梦终究只是梦，书生也只是书生。在宣纸上挥毫泼墨，指点江山，便以为自己胸有百万精兵。但在战场之上，挥洒的却是淋漓鲜血。
<br/>　　陆洛，高估了自己。
<br/>　　凉州兵兵败如山倒，落荒而逃。陆洛也加入了逃跑的大军中，被手下扶着上了马，在亲卫的保护下狼狈向凉州城方向逃去。
<br/>　　而程大雷一方却是士气高涨，追着凉州兵厮杀，当将后背朝向敌人后，迎接他们的也就只有死亡。程大雷骑在黑牛背上，用手中的千里镜远远看着这一幕，瞧见了溃逃的陆洛。
<br/>　　五路大军与高飞虎合在一处，如狼似虎扑向敌人。
<br/>　　“好啦，收兵。”
<br/>　　程大雷在马上道了一声，铜锣敲响，追击敌人的队伍同时止步。秦蛮五人纵马来到程大雷身边。
<br/>　　“大当家，为何不让我们追下去了。再追下去，定能一举歼灭了他们。”秦蛮。
<br/>　　“歼灭他们又有什么意义，穷寇莫追，况且，我们的目的已达到。”程大雷挥挥手：“打扫战场吧。”
<br/>　　打赢这场战斗，诸人的心情都很愉悦。可要说最高兴的，还是高飞虎。
<br/>　　从江南到凉州，一路上费了好多辛苦，他也一直承受着手下的质疑。程大雷能否还将大家当兄弟看待，他究竟可以安排好大家……
<br/>　　直到此时，所有谣言不攻自破，自己在蛤蟆寨的人缘，也并没有想得那么糟糕么。
<br/>　　高飞虎终于在手心兄弟前扬眉吐气，能有这么多人来救自己，他也觉得脸上有光。瞧瞧，这都是我以前的兄弟，我出事谁会不管……
<br/>　　高飞虎高昂头颅，装作风轻云淡的和其他人打招呼。
<br/>　　“哈，老秦呐，好久不见了。”
<br/>　　“张三爷，你身手可是更好了。”
<br/>　　……
<br/>　　然后，高飞虎尴尬的发现，似乎没有人搭理自己，因为他们都正忙着抢夺凉州兵留下的盔甲、马匹、武器。似乎没有看见高飞虎挥手的动作。
<br/>　　“豹子，过来！”
<br/>　　“快些，把东西都收拾干净了。一点东西也别给他们留下。”
<br/>　　不说其他人，就连高飞豹都没有时间搭理自己大哥。高飞虎立在那里，心情有些郁闷。齐少建、郭不戳等人围绕在他身边，愣愣的看着周围。
<br/>　　蛤蟆城士兵的表现，令他们觉得有些怕。这些人穿着整齐统一的制服，就算在争夺战利品这件事上，也保持着一定的秩序。而在刚才战斗中他们展现的素质，更是令人不寒而栗。
<br/>　　他们从富庶的江南而来，心里委实有些看不起凉州这西北边关。对于高飞虎整日吹嘘的程大雷，更是有种下意识的反感。
<br/>　　对方就像一块经过千锤百炼，洗去所有杂质后的钢铁，经过烈火淬炼，已成为一柄杀气渗人的钢刀。
<br/>　　而己方，穿着乱七八糟的衣物，有那狼牙棒的，有拿铁锤的，有拣根木棒当武器的……怎么看，怎么都像是一群乌合之众。
<br/>　　这个时候，程大雷冲高飞虎挥了挥手。
<br/>　　高飞虎领着手下人忙不迭的来到程大雷身边，慌张从马上下来，他和黄三元一起抱拳行礼。
<br/>　　“大当家！”
<br/>　　程大雷点点头：“都过来啦？”
<br/>　　高飞虎憨憨笑了一声：“南边情况越来越不好，就带着手下人过来了。大当家，咱们这边情况如何？”
<br/>　　“还算过得去吧。”
<br/>　　高飞虎在江南也有一支人马，手上有上万兄弟，此时能抛家舍业过来投奔程大雷，足见对程大雷的信任。算起来，双方已经是三四年不见，程大雷目光向周围转了一圈，见高飞虎的手下衣衫褴褛，面带菜色，想也知道，他们这一路过来，定然是吃了不少的苦头。
<br/>　　齐少建、郭不戳等十几人立在高飞虎身后，偷眼打量着程大雷。
<br/>　　关于程大雷的名字，高飞虎是经常提起，众人耳朵都快要磨出茧子。但程大雷本人，他们却是今天第一次见到。
<br/>　　只见程大雷骑在墨玉乌月兽背上，身披照月蛤蟆甲，除了相貌丑陋些，的确称得上威风赫赫。他们当然不敢小看程大雷，一开始对程大雷的质疑，此刻已烟消云散。毕竟，能号令这样一支队伍的领袖，岂是泛泛之辈。
<br/>　　高飞虎愣了愣，这才意思到程大雷在看什么。他一脚踹在齐少建腿上，道：“都愣着干什么，这是我的老大，还不快向大当家见礼。”
<br/>　　几人慌忙反应过来，同时垂首，口中高呼着：“见过大当家。”
<br/>　　程大雷点点头：“不说这个了，等先回山寨再说。”
<br/>　　其时，大概是午夜子时，战斗持续了数个时辰。再将战利品都收拾上时，诸人开始返回蛤蟆城。
<br/>　　这个时候，秦蛮、关鱼等人才想着过来找高飞虎打声招呼。毕竟是曾经共同并肩作战的兄弟，他们也不可能完全将高飞虎忘了，只不过刚才忙着抢战利品，没时间顾忌高飞虎。
<br/>　　蛤蟆城按战利品和人头数论功行赏，负责这件事的是和珅。所以在打扫战场这件事上，谁也不肯落后，因为这决定以后各支队伍得到的资源多少。
<br/>　　高飞虎身边围了一大群人，都是当年从青牛山一起走出来的老弟兄。此时见面，自然也有说不完的话聊。前呼后拥，高飞虎整个人完全抖了起来。
<br/>　　当初，山寨的老弱妇孺随着高飞虎逃向江南，在那里安营立寨。此刻互相认识的聚在一起，说着这三年内发生的往事。
<br/>　　齐少建等人原先的担忧烟消云散，这一幕打消了他们的疑虑，蛤蟆寨和飞虎寨本就是一体，原先的担忧本就是多余的。
<br/>　　这个时候，程大雷和黄三元同行，向他询问飞虎寨最近这段日子的情况。
<br/>　　情况不是很好，否则的话，高飞虎也不会想来凉州。而这一路上，更是遇到太多困难，有些兄弟死在了半路上。
<br/>　　高飞虎从江南离开的时候，带着一万余人，和凉州兵打了这一仗，目前只剩下七八千人，其中还有大半的老弱妇孺。
