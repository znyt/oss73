　　纣绝阴的实力，在如今九域的年轻一代，还算是个人物。

<br/>　　就连寒雪阁的整体实力都提升了无数倍，更合可是七大宗门。
<br/>　　八成以上的资源，几乎都是被七大宗门所瓜分了，让现在七大宗门的整体实力已经是今非昔比了。
<br/>　　纣绝阴行事嚣张，想要杀了他的人多得是。
<br/>　　可却没有任何一人，敢真正杀了纣绝阴。
<br/>　　纣绝阴的背后，代表的可是六道魔门，在如今的九域，谁敢跟六道魔门作对？
<br/>　　战狠长老等人，也同样是没有想到楚风眠居然是说杀就杀，才没有反应过来。
<br/>　　“好，看来阁下今日是真的要跟我们六道魔门作对的！叫人！”
<br/>　　战狠长老眼神冰冷无比的看着楚风眠，眼神之中杀机涌现。
<br/>　　今日楚风眠杀了纣绝阴，他绝不能让纣绝阴活着离开归墟城，不然的话，战魔尊主怪罪下来，他可就完了。

<br/>　　楚风眠冰冷喝道，他的背后，天涯剑匣当即打开，一道道剑气从中飞了出来，顿时向着在场所有六道魔门弟子斩杀而去。

<br/>　　战狠长老咆哮一声，手心一动猛然一捏，滚滚魔意凝聚，在他的手心之中凝聚出了一把长刀。
<br/>　　刀意翻腾，面对这一道剑光的斩杀而来，战狠长老一刀当即斩杀而出。
<br/>　　这战狠长老虽然看不出楚风眠的真正实力，可是他可以清楚一点，那就是楚风眠绝对不是一位古帝。
<br/>　　古帝身上雄厚的天地之力，几乎是无法掩饰，既然不是古帝，他就丝毫不怕。
<br/>　　古帝之下，皆为蝼蚁。
<br/>　　一刀斩杀而去，掀起滔天魔意，向着楚风眠斩杀而去。
<br/>　　惊天动地，整个归墟城内，都足以感觉得到这魔意的翻腾。
<br/>　　“这就是一位古帝的力量？”
<br/>　　“六道魔门的长老，真是可怕。”
<br/>　　“据说六道魔门的六位尊主，比起这些长老还要强大的多，不知道是出手得是多么恐怖！”
<br/>　　“这小子也是找死，现在其他六大宗门，都不敢去招惹六道魔门，他一个小家伙，算是什么东西。”
<br/>　　几乎所有人，看到这战狠长老出手的一刻，都觉得楚风眠已经是必死无疑了。
<br/>　　可下一刻，刀锋跟剑光碰撞的一刻。
<br/>　　刹那之间，那战狠长老斩出的刀锋，当即破碎，在这交锋之下，就连抗衡都做不到。
<br/>　　楚风眠现在催动天涯剑匣，所打出的剑气，几乎是跟八劫古帝全力一击打出来的威力，相差无几。
<br/>　　这战狠长老仅仅只是一位七劫古帝罢了，又能如何抗衡？他的全力之下，都未必是这一道剑气的对手。
<br/>　　轰！
<br/>　　一招而败。
<br/>　　反震的力量落在了战狠长老手中凝聚的长刀上。
<br/>　　只见这一般长刀，都是在力量之下，被震的节节破碎，当即化为了一团粉末。
<br/>　　战狠长老更是也吐出了一大口的精血，倒退练练。
<br/>　　全场死寂。
<br/>　　无论是谁，都没有可能想到是这个结果，这可是六道魔门的一位长老，一位古帝啊。
<br/>　　就算是在如今的九域，古帝可都是绝对是强者，足以横扫一片的存在。
<br/>　　现在居然是被楚风眠一招打败？
<br/>　　而且不止是战狠长老，那些其他的六道魔门弟子，也同样是受到了剑气的轰击，一个个身躯连连爆炸开来。
<br/>　　在这通天楼之内的十几名六道魔门弟子，没有一人还活着。
<br/>　　这让众人心中都是冰凉无比。
<br/>　　楚风眠在对付这战狠长老的时候，甚至是还在分心去斩杀其他六道魔门弟子。
<br/>　　饶是如此，这战狠长老，都不是楚风眠的一招之敌，就连楚风眠一招都接不下。
<br/>　　尤其是楚风眠还是如此年轻，从楚风眠身上的气息判断，楚风眠绝对没有百岁。
<br/>　　不到百岁，就有如此实力。
<br/>　　“好厉害的剑气！”
<br/>　　感觉到通天楼之中的战斗，无数身影，都是赶了过来。
<br/>　　其中更是有着几位古帝，他们见到那战狠长老被剑气所击败的一刻，都是瞳孔放大。

<br/>　　九域不大，所以九域之中的强者，几乎都互相叫得出名号，可他们可不记得，有楚风眠这样一个人。
