
<br/>　　<br/>　　    崔玄策说李家的风水坏了，那就是真的坏了，做为南昆仑这一代弟子最闪耀的那颗星星，做为南昆仑掌门座下首席大弟子，崔玄策在风水一术上的造诣是让很多人都望而却步，拍马都追不上的，在风水一事上说他字如千金都不过分。<br/>　　<br/>　　    枯井溢水为泄财，老坟开裂为散运，这两个变故要是不加以妥善处理的话，陇西李氏恐怕要遭逢第二次大难了。<br/>　　<br/>　　    第一次，是唐灭宋起。<br/>　　<br/>　　    王惊蛰走到正厅门口的时候，就知道自己可能要迈不过李家的门槛了，后面突然冒出了好几股肃杀的气息，是李兰荣，茅清水和余婆婆还有几个李家老人的。<br/>　　<br/>　　    杀父之仇夺妻之恨，和毁人基业都是最遭恨的！<br/>　　<br/>　　    李兰荣踉跄着走了下来，愤怒且带着惊恐的说道：“坏我李氏风水？你好大的胆子······”<br/>　　<br/>　　    “搞搞清楚，是我坏的么？”王惊蛰缓缓的转过身子，嘲讽的说道：‘从我进你们李家大门开始，我有动过一个手指头么？我有动过一刀一剑，念过一句咒么？人太不自知了，你们坏了规矩坏了自家的风水，偏要赖到我的头上，这是哪来的道理？”<br/>　　<br/>　　    李兰荣顿时哑然！<br/>　　<br/>　　    茅清水也站了起来，冷冷的说道：“在我们李氏宅门里讲规矩，讲道理，你不觉得自己太蠢了点么？”<br/>　　<br/>　　    王惊蛰一愣，忽然觉得这句话听起来有点耳熟，好像不久之前就有人跟他说过，和女人讲道理你蠢不蠢？余婆婆小声跟一旁的南昆仑长老问道：“李氏的风水真坏了？”<br/>　　<br/>　　    南昆仑长老沉吟着说道：“确实坏了，长此以往下去李氏必然衰败，大唐国运二百八十九年是靠着李淳风和袁天罡二位国师一力操持的，后来两位国师都先后仙去这大唐的基业终究没有挺过三百年，李家后期到现在算是苟延残喘了，要是风水再坏恐怕陇西李氏就要成为过去了”<br/>　　<br/>　　    李淳风和袁天罡只推了大唐国运二百八十九年就不推了，是因为两人知道大唐盛世终究过不了王朝三百这个坎。<br/>　　<br/>　　    纵观华夏历史，各朝代年限，就拿最出名最鼎盛的几个大朝来讲，大唐国运二百八十九年，大清帝国二百七十六年，明朝二百七十六年，没过百年的朝代一抓一大把，而大秦只有区区十五年，能过三百年国运的屈指可数。<br/>　　<br/>　　    享三百年国运是个坎，难过！<br/>　　<br/>　　    余婆婆叹了口气，深深的望了一眼宠辱不惊的王惊蛰，然后再次低声说道：“李氏风水不能有损啊，徐杰长老你得看看了”<br/>　　<br/>　　    徐杰长老“嗯”了一声，招呼了崔玄策一声说道：“你与我一同去看看，破镜尚且能重圆，更何况是风水坏了，自然也有修补之道，你下昆仑山不过几年，时间还短，实践和理论终究要结合起来才有大用的”<br/>　　<br/>　　    崔玄策起身说道：“知道了，师叔”<br/>　　<br/>　　    “走吧·······”<br/>　　<br/>　　    李兰荣指着王惊蛰对李伯还有李家的人狠狠的叮嘱道：“看着他不要让他离开李家，一步都不许往出走”<br/>　　<br/>　　    王惊蛰淡淡一笑，索性走了回来随意的坐到一张椅子上，闭目养神起来。<br/>　　<br/>　　    闭上眼睛之后，王惊蛰幽幽的说了一句话：“你们若是续接上这坏了的风水，南昆仑也就不是南昆仑了”<br/>　　<br/>　　    还未走远的徐杰和崔玄策都同时脚步一顿，两人自然不认为王惊蛰是在那里随意的出口一言了，只是却有点听不明白。<br/>　　<br/>　　    徐杰皱眉问道：“你这话说的是什么意思？”<br/>　　<br/>　　    王惊蛰只是呵呵一笑也不解释，李淳风和袁天罡为陇西李氏布下的风水局，出自连山一书，此书天下现在恐怕只有王冬至和王仙芝能够读的通透了，王惊蛰也不过是只知皮毛而已。<br/>　　<br/>　　    你南昆仑虽然执掌天下风水一道，但连书你们总归不晓得，所以拿什么来续接上这已经坏了的风水局？<br/>　　<br/>　　    接上这风水局，自然也就不是南昆仑了，应该是昆仑天下才对。<br/>　　<br/>　　    崔玄策深深的看了王惊蛰一眼，见他不吭声就跟随徐杰走向了李宅后院，一直来到那口老井处。<br/>　　<br/>　　    此时，曾经枯了能有百年的老井，井口还在往外徐徐的冒着井水，已经浸满了整个院子，甚至还能看得见井水中还涌出了鱼虾。<br/>　　<br/>　　    余婆婆诧异的问道：“这井不是枯了么？”<br/>　　<br/>　　    “听说井下连着渭河，枯了只是因为下面被堵塞，渭河水过不来而已”李兰荣皱眉解释道。<br/>　　<br/>　　    徐杰和崔玄策看了眼枯井后，随即一句话都没说，两人同时离开了院子，然后绕着整个李氏宅院快速的摸索起来，到底是风水行家，他俩只看了一眼基本上行就知晓问题出处在哪了。<br/>　　<br/>　　    良久之后，崔玄策和徐杰差不多又先后出来了，同时问道：“不对，宅中不应该只有这一口井的，还得有一口才是”<br/>　　<br/>　　    李兰荣和几个李家老人同时一愣，摇头说道：“没有，只有这一口井”<br/>　　<br/>　　    “不可能的！”徐杰摇头说道：“这口井说是阵眼，但其实是一只龙眼，龙眼哪里会只有一只的道理，应该成双才对的”<br/>　　<br/>　　    但凡真正的风水大阵，基本都是建在龙脉上的，而陇西李氏的这个院子其实是建在了个龙头上，两口井就是龙眼，不过这一只龙眼在这里，那另外一只在哪？<br/>　　<br/>　　    枯井出水，就是龙眼啼泪，乃是大大的不吉征兆！<br/>　　<br/>　　    李兰荣顿时懵了，这李宅中的一草一木一块地哪里还会有人比他更熟悉，家中有几口井他又怎能不知道？<br/>　　<br/>　　    “这口井出了水，那如果真要是还有的话，另外一口是不也出了？”余婆婆忽然插嘴说道。<br/>　　<br/>　　    “怪就怪在这里了······”崔玄策看了眼正厅的方向，说道：“好玄妙的局，还真是不容易能看透呢”<br/>　　<br/>　　    徐杰拧着眉头，捉摸不定的说道：“一只龙眼？怎么可能就只有一只龙眼呢，这太不对劲了太反常了，玄策，昆仑中的风水典籍你差不多都读烂了吧，据你所知有出现过一只龙眼的风水阵么？”<br/>　　<br/>　　    “据我所知·······”崔玄策满脸皆惊的说道：“好像，还真有过一次这种状况”<br/>　　<br/>　　    “啊？”几人目光顿时全都落在了崔玄策的身上。
