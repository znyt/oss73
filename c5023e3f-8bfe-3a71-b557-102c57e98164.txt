　　连霆的气势无比强大，磅礴的雷霆威压，滚滚不休。
<br/>　　咻！
<br/>　　连霆出手了，一刀狠狠的斩下，一道犀利的雷电，从高空之上一劈而下，速度快到极致。
<br/>　　这一刀恐怖无比，远超白展朝数倍不止，犀利的雷电刀芒，充满了毁灭般的气息。
<br/>　　下方，苏莫见此，眼眸一凝，毫不犹豫，立刻便动用了不败战气元神。
<br/>　　此元神暴露而出，对于他来说，也不会有什么大的影响。
<br/>　　不败战气元神，隐藏在吞噬元神之中，苏莫意念一动，不败战气元神就离开了吞噬漩涡。
<br/>　　刹那之间，苏莫的身上不仅剑势冲霄，更是弥漫着惊天的战意。
<br/>　　这股战意浩大磅礴，犹如凝结成了实质，不仅对其他人有强大的压迫了，更是让的苏莫气势暴增。
<br/>　　咻！
<br/>　　苏莫同样出剑了，一剑逆斩而上，直冲天穹。
<br/>　　不过，苏莫并没有动用混沌之力，剑气依旧是五色玄力，只不过浑厚了数倍而已。
<br/>　　轰！
<br/>　　一声爆响，五色的剑气和雷霆刀芒相击在了一起，霎时之间，劲力滚滚，五色剑气轰然炸开。
<br/>　　犀利的雷霆之光闪耀，如同一支利箭，直击苏莫的面门。
<br/>　　很明显，这一击雷霆刀芒更甚一筹，比五色剑气更强。
<br/>　　苏莫眼疾手快，电光火石之间，手中太云神剑急忙一横，封挡住了雷霆刀芒。
<br/>　　铛！
<br/>　　一声震动耳膜的爆响，雷霆刀芒即便威力大减，依旧强大。
<br/>　　巨大的力量倾泄而出，让得苏莫身形一震，立刻蹬蹬蹬的后退了十几步。
<br/>　　“好强的战意，上官昊有两种元神！”
<br/>　　“两种元神也无用，不可能抗衡连霆！”
<br/>　　“这是什么元神？难道是传说中的不败战气？”
<br/>　　众观战的弟子，议论纷纷，心惊苏莫的实力。
<br/>　　连梁师兄都震惊不已，这上官昊居然强大到了如此的地步。
<br/>　　“天赋这么强！”梁师兄面色稍稍凝重了下来，他倒要看看，这个上官昊的极限在哪里！
<br/>　　“再接我一刀！”
<br/>　　一击没有彻底击败苏莫，连霆感觉很没有面子，他怒目圆睁，彻底爆发出了巅峰的战力。
<br/>　　滚滚源力涌动，连霆仿佛化身成为了一个巨大的雷球，无上的雷霆之威，猛然暴增。
<br/>　　巨大的雷霆威压，浩荡不休，笼罩整个千渊府，弥漫在整个千渊城。
<br/>　　雷之规则！
<br/>　　这是超越了奥义的存在，奥义参悟到九阶大圆满，便能领悟到规则之力。
<br/>　　规则，又被称为天地规则、或者是大道法则，乃是奥义的终极所在。
<br/>　　轰隆！
<br/>　　一声雷霆爆响，璀璨的雷刀再次一斩而下，如同九天神雷降临，直击苏莫。
<br/>　　“好强！”
<br/>　　“超过了一品高级真神，接近普通的二品低级真神的实力了！”
<br/>　　“上官昊有危险了！”
<br/>　　众人见此，均是心中一惊，不过，并没有人出手阻拦，因为他们都不会掺和此事。
<br/>　　场中，苏莫全身汗毛一炸，这一刀太强了，是他生平仅见。
<br/>　　一瞬间，苏莫体内五行玄力瞬间融合，血脉之力尽数催动。
<br/>　　因为，他若是再继续隐藏，很可能被这一刀重创。
<br/>　　霎时之间，苏莫气势大变，整个人身上混沌之威、剑势之威、战意之威，汇聚在一起，如同苏醒的战胜，不可一世。
<br/>　　轰！
<br/>　　一声炸响，苏莫脚下重重的一跺，整个人冲天而起，直上苍穹，迎着雷霆刀气冲了过去。
<br/>　　所有人都瞪眼了眼眸，包括不远处的梁师兄，和刚刚斩出一道的连霆。
<br/>　　因为，苏莫的气势，猛然暴增了数倍不止。
<br/>　　苏莫人剑合一，如同一支混沌利箭，撕裂空气，瞬间便与雷霆刀气撞击在了一起。
<br/>　　轰！
<br/>　　一声爆响，雷光万丈，纷纷洒洒。
<br/>　　强大的雷霆刀气，瞬间便被击碎，彻底溃散开来，漫天洒落。
<br/>　　而苏莫的身形，依旧以极快的速度，冲向高空，向连霆而去。
<br/>　　“什么？”
<br/>　　连霆见此，双眸一瞪，眸中尽是骇然之色。
<br/>　　上官昊居然如此强大，这怎么可能？
<br/>　　不过，连霆来不及多想，眼见上官昊急速而来，他的身形疯狂的暴退。
<br/>　　苏莫身上的气息，让连霆感到骇然，下意识的就选择了暂闭锋芒。
<br/>　　“败吧！”
<br/>　　苏莫轻喝一声，手中太云神剑混沌之光暴涨，急速挥斩了起来。
<br/>　　一瞬间，苏莫斩出了七七四十九刀，四十九道剑气，如同一阵风暴，向连霆碾压而去。
<br/>　　剑气的速度，远超连霆的速度，瞬间之间便追了上去。
<br/>　　“给我破！”
<br/>　　连霆怒喝，手中雷刀疯狂的斩击，想要击溃混沌剑气。
<br/>　　然而，这一刻苏莫的强大，超过了连霆的预料。
<br/>　　转瞬之间，连霆便被漫天剑气击中，身形急速抛飞了出去。
<br/>　　血光四溅，血水飞洒，染红了苍穹。
<br/>　　轰！
<br/>　　连霆破烂般的身躯，飞出了演武场，飞出了数千丈，砸落在了大地之上。
<br/>　　静！
<br/>　　整个演武场彻底寂静了下来，所有人都不可思议的望着这一幕。
<br/>　　连霆败了！
<br/>　　败的凄惨无比！
<br/>　　即便修为强大数畴，依旧不是上官昊的对手。
<br/>　　演武场上寂静了约莫两个呼吸的时间，便彻底的沸腾了起来。
<br/>　　“上官昊是混沌族人！”
<br/>　　“他修炼了混沌神帝的道统，大混沌五行神诀！”
<br/>　　“他居然隐藏的如此之深！”
<br/>　　“难怪他虚神境三变的修为，能来到神域！”
<br/>　　众人一片哗然，议论声四起，这个结果出乎了所有人的预料。
<br/>　　混沌神帝，作为神域八大至尊之一，甚至是八大至尊之首，其修炼的混沌大道，几乎是无人不知无人不晓。

<br/>　　苏莫闻言，脸色一沉，不禁停下了脚步。
<br/>　　这是殿主的声音！

<br/>　　微微点了点头，苏莫无奈的向殿主坐在的塔楼走去。
