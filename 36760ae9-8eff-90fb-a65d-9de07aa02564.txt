　　“你竟敢——如此对我！杨家不会放过你的！”
<br/>　　杨之凡扬天大吼了一声，双眸几乎要喷出火来。
<br/>　　周围其他众人看到这一幕，也是个个目瞪口呆。
<br/>　　杨之凡何许人也？那可是东江省杨家的少主！
<br/>　　以杨家如今的势力，即便是在整个华夏，也能跻身一流家族的行列。
<br/>　　对方竟然说废就废，而且手段如此强势霸道！
<br/>　　叶尘冷冷瞥了一眼地上的杨之凡，
<br/>　　“若非看在同学一场，我早就一掌灭了你，还不滚！”
<br/>　　“你——”
<br/>　　杨之凡心中恨意滔天，可是面对如此强势的叶尘，哪里还敢多说，被周围的手下搀扶着，仓皇逃离而去。
<br/>　　众人仓皇离去之后，唐威三人当即走上前来，全都满脸担忧之色，
<br/>　　“尘哥，你、你杀人了……这可如何是好？”
<br/>　　他们毕竟还是学生，哪里见过这种场面，自然怕的要命。
<br/>　　叶尘却一脸淡然，直接屈指一弹，一道三昧真火，弹射到何坤山的身上。
<br/>　　轰！
<br/>　　火焰升腾起来，几乎一眨眼的功夫，何坤山的尸体，便化作了一缕青烟，仿佛从来不曾在这个世界存在过一般。
<br/>　　“这……”
<br/>　　三人顿时再次目瞪口呆，不由得狠狠揉了揉眼睛，甚至都怀疑自己出现了幻觉。
<br/>　　“尘哥，你……你到底是什么人啊？”
<br/>　　四人离开武道社后，返回宿舍的路上，李虎实在按耐不住心中的好奇，忍不住凑到叶尘跟前，小心翼翼的开口问道。
<br/>　　唐威和陈东也纷纷朝叶尘望来，全都一脸期待的倾听着。
<br/>　　叶尘淡淡一笑，
<br/>　　“我就是我呀，是你们的室友、兄弟！”
<br/>　　听到叶尘这话，三人先是一阵失望，继而又纷纷露出一抹喜色。
<br/>　　是呀，不管叶尘是什么身份，永远都是他们的室友、兄弟，这就已经足够了。
<br/>　　“尘哥，你刚刚那样羞辱杨之凡，还把他当成重伤，只怕他不会善罢甘休！”
<br/>　　唐威忽然一脸凝重道。
<br/>　　李虎也跟着道：
<br/>　　“是呀！杨家在东江省，如今已经隐隐有成为第一大家族的趋势，而这杨之凡又是杨家的少主，只怕是不好惹！”
<br/>　　说到这里，李虎微微一顿，接着又道：
<br/>　　“不过，尘哥，你若是能请唐家帮忙，此事应该还有回旋的余地。”
<br/>　　叶尘微微一笑，
<br/>　　“这个你们不用担心，区区一个杨家而已，还奈何不了我！”
<br/>　　……
<br/>　　几个小时后，天海第一医院，
<br/>　　“是谁！是谁这么大胆，竟然敢把我杨国强的儿子，打成这个样子！”
<br/>　　杨家家主杨国强，望着病床上，双腿被废的杨之凡，不由得一阵怒吼，周围的医生护士，以及十多名黑衣保镖，顿时个个禁声。
<br/>　　要知道，最近这半年，东江杨家的实力暴涨，杨国强更是一跃成为了华夏商界的新巨头，频频在电视新闻上露脸，即便是在天海市，也已经是无人不识。
<br/>　　杨之凡看到自家老爸来了，顿时哭的稀里哗啦，一把拉住自家杨国强的手臂，开始控诉起来，
<br/>　　“老爸！你一定要，一定要为我报仇啊！”
<br/>　　杨国强两眼一瞪，
<br/>　　“儿子，你放心！敢欺负到我们杨家的头上，老爸一定会让他付出血的代价！”
<br/>　　“你先把事情给我仔仔细细讲一遍，到底哪个吃了雄心豹子胆的，竟然连我东江杨家都不放在眼里！”
<br/>　　杨之凡抹了一把脸上的泪痕，眼眸中射出一道充满怨毒的目光，一脸恨恨道：
<br/>　　“老爸，打伤我的人名叫叶尘，是我们生物学院的学生，这家伙的武功很高，恐怕至少也有化劲宗师以上的实力……”
<br/>　　听完杨之凡的描述之后，杨国强顿时眉头紧皱，露出一抹凝重之色，
<br/>　　“不到二十岁的化劲宗师？而且还姓叶？”
<br/>　　杨国强脑海之中，猛然跳出一道瘦削的身影，不过很快摇了摇头，便将那道念头直接抛到了脑后。
<br/>　　和杨之凡一样，杨国强也无法相信，名震全球的云州叶仙人，会来到一个小小的天海大学读书，这简直太荒诞了！
<br/>　　而且那些叶仙人的修为，可远远不止化劲宗师。
<br/>　　就在杨国强思绪飞转的时候，身后忽然一名唐装老者走上前来，笑呵呵道：
<br/>　　“不到二十岁的化劲宗师吗？看来自从那云州叶仙人一战成名之后，世人尚武成风，如今武道天才辈出啊！”
<br/>　　杨之凡看到这唐装老者，顿时两眼一亮，满脸都是激动之色，
<br/>　　“华老！您、您也来了！真是太好了！有您在，我看那小子还如何猖狂？”
<br/>　　原来，这唐装老者名叫华云生，乃是海外华人武道界，赫赫有名的大宗师，一身修为深不可测，据说已经接近神境。
<br/>　　此人之前一直隐居海外，最近不知道是何缘故，突然返回华夏，成为了杨家的座上宾。
<br/>　　华云生瞥了一眼病床上的杨之凡，淡淡道：
<br/>　　“如果那小子真如你说的那么厉害，老夫倒是想会一会此人！”
<br/>　　杨国强父子听到这话，顿时尽皆大喜，
<br/>　　“如此那就有劳华前辈了！”
<br/>　　杨国强向华云生躬身道谢之后，转而又望向床上的杨之凡，
<br/>　　“小凡，你在医院好好养伤，我就派人把那小子抓过来！”
<br/>　　不料，杨之凡挣扎着从病床上坐了起来，一脸兴奋之色道：
<br/>　　“不！老爸，如今有华老坐镇，这件事就让我自己来解决吧！我要亲眼看着那小子，跪在我的面前！！”
<br/>　　一想起之前叶尘给他的羞辱，杨之凡就气得发狂，如今有了华云生这个大高手做靠山，他如何还坐的住，自然要亲自将之前的羞辱讨还回来才算解气。

<br/>　　而与此同时，天海大学，生物学院，
<br/>　　此时正值下课时分，叶尘坐在教室的角落里，随意的翻看着一些书籍。
<br/>　　他最近看了一些生物学的理论知识，发现和修真之道，竟然有相互印证之处，引起了他不小的兴趣，一时间竟看的有些入迷。
<br/>　　忽然，
<br/>　　哗啦啦！
<br/>　　教室外传来了一阵躁动声，继而一群黑衣壮汉，从外面冲了进来。
<br/>　　一众学生何曾见过这等场面，顿时吓得纷纷惊呼起来。
<br/>　　紧接着，
<br/>　　一个轮椅被人从门外缓缓推了进来，而坐在轮椅上，赫然正是杨之凡，
<br/>　　“诸位同学，你们不必慌张！我和叶尘同学有些私人恩怨要解决，无关人等就请先暂时离开吧！”
<br/>　　杨之凡在人群中扫过之后，很快便锁定了角落里的叶尘，虽然他被手里的书本遮住了大半张脸，杨之凡还是一眼将他认了出来。
