        连星儿找到自己的座位坐下来后，从包里掏出了一根棒棒糖，拆了包装纸，塞嘴里含着。
<br/>
<br/>        此时，坐在后排的一对情侣，开始讨论了起来。
<br/>
<br/>        男的说：“这次的抽签对战，不料抽中的是强强对战，真的是便宜了那些不强的战队。”
<br/>
<br/>        “为什么？”女的问。
<br/>
<br/>        男的解释道：“今天的开场入围对战，是EA战队对战MS战队。EA和MS是所有黑客战队里两个最强大的两个战队。以往的冠军和季军总是在这两个战队之中徘徊。在这次的入围赛里，他俩其中一方赢了，就相当于他俩其中一方已经是总决赛的冠军了。而另一方，若是在入围赛就被淘汰了，那损失就大了，几乎是相当于他们一年的努力就全白费了，什么奖金都没有了。”
<br/>
<br/>        “我赌MS战队赢！因为有C神在里面！”
<br/>
<br/>        “我看不一定，EA战队也是来势汹汹！”
<br/>
<br/>        连星儿听着，也十分认同身后那女孩的说法，有她师父在，一定拿第一。
<br/>
<br/>        起初，耳边还是一阵聒噪声。
<br/>
<br/>        直到九点整，第一个战队入场时，才传来欢呼声。
<br/>
<br/>        当MS战队（黑客部）入场时，除了欢呼声以外，还有此起彼伏的呐喊声。
<br/>
<br/>        而MS战队（黑客部）的五个成员，全都穿着一身墨蓝色战队服，戴着统一的MS标志的墨蓝色牛仔鸭舌帽和黑色的MS标志口罩。
<br/>
<br/>        他们把自己全副武装得连亲妈都不认识，更别提连星儿能从中找出谁是她师父了。
<br/>
<br/>        直到他们一一在电脑桌前按名字落座，连星儿才知道，坐在最中间那个就是她的师父“余寒犹厉”。
<br/>
<br/>        明司寒和四个队友坐好后，他们的教练便开始跟他们沟通一些比赛时的注意事项。
<br/>
<br/>        本来，明司寒有在跟大家沟通，但是沟通着，沟通着，他的目光便不由自主地落向了对面的VIP观众席。
<br/>
<br/>        第一眼，他便看到了人群中的连星儿。
<br/>
<br/>        她坐得很端正，叼着的棒棒糖的棍子歪在了嘴角。
<br/>
<br/>        因为距离有点点远，他不知道她在看谁，但能确定，她是在看他们这边。
<br/>
<br/>        “C，你有听我说吗？”教练喊了一声。
<br/>
<br/>        明司寒回过神来，点了下头：“嗯，我明白的。”
<br/>
<br/>        “你是主攻手，一定要抓住机会，见缝插针。”教练叮嘱道。
<br/>
<br/>        明司寒点了点头。
<br/>
<br/>        比赛开始后，不管是对战的是个赛手，还是观众席上的观众，一个个都看着紧张兮兮。
<br/>
<br/>        连星儿看着大屏幕，发现完全看不懂他们到底在做什么。
<br/>
<br/>        只知道，黑色的底幕上，无数条英文快速滚动着。这个画面，就像电影里所演的那种骇客帝国。
<br/>
<br/>        虽然她看不懂，好在那边的一男一女解说员的解说功底不错。
<br/>
<br/>        连星儿倒是听懂了，而且，通过那两个解说员的解说，她大概可以了解到，MS战队更胜一筹，此外“C”的攻势很猛。
<br/>
<br/>        “EA战队的系统瘫痪，无法再修复运作，数据已被MS战队成功读取，恭喜MS战队首开旗胜！”男解说员对着话筒激昂地说道。
<br/>
<br/>        在场支持MS战队的所有观众顿时兴奋地从座位上跳了起来。
<br/>
<br/>        口哨声、欢呼声、掌声，不绝于耳。
<br/>
<br/>        看到中央对战坪里，MS战队的成员纷纷放下耳机从电脑桌前起身，连星儿身边的几个女生也跟着蠢蠢欲动起来。
<br/>
<br/>        “我们快去后台，给MS战队的成员们送花。”
<br/>
<br/>        “我还想要签名！”
<br/>
<br/>        几个女生激动不已地说道。
<br/>
<br/>        连星儿眼珠子一转，连忙起身，跟在了那几个女生的后面。
<br/>
<br/>        她跟着她们一起去了后台，其实就是一条退场通道，还有保安拉起了横条将她们给拦在了一张双开门的旁边。
<br/>
<br/>        当MS战队（黑客部）的成员走出来的时候，女生们便尖叫了起来。
<br/>
<br/>        有的喊“C”，有的喊“G”，还有的喊“O”等等。
<br/>
<br/>        MS战队（黑客部）的五个成员的名字代号分别是C、G、O、D、Q，替补是P。
<br/>
<br/>        所以，他们还有个外号叫“六个圈”。
<br/>
<br/>        连星儿站在横条后，也忍不住大声喊了一声：“余寒犹厉！”
<br/>
<br/>        而就在这时，明司寒突然停下了脚步，循声望去。
<br/>
<br/>        当女生们看到“C”朝她们这边看过来时，一个个激动不已地叫了起来：“C，看这里！看这里啊！”
<br/>
<br/>        连星儿见“C”朝自己这边看了过来，连忙低头从包包里翻出小本子和水性笔，向“C”挥动着手，十分期待地喊道：“可以给我签名吗？”
<br/>
<br/>        明司寒看着连星儿那一脸小迷妹的表情，忍俊不禁地朝她走了过去。
