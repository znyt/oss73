　　第405章
<br/>　　“战争终究不是一件好玩的事情，在即将到来的乱世，注定有大片大片的人死去。从福德勒开始，我们都要试着接受牺牲。其实我们什么都做不了，唯一能做的，也只是尽量保护身边的人，让他们活过这个乱世。”程大雷看着李婉儿的眼睛。
<br/>　　李婉儿默然，她不愧是明帝的女儿，身上流着皇族的血，总是以天下为己任，让她放开心思，其实并不容易做到。
<br/>　　“你会出兵么？”李婉儿问。
<br/>　　“不会。”
<br/>　　程大雷坚定的摇摇头，这是他一开始就想好的。他既不会站出来支持朝廷，也不会支持正义教，长安城已经成了一架绞肉机，虽然山寨弟兄人人思战，但程大雷并不想让他们去送死。
<br/>　　程大雷看到，听到自己的话后，她眼中的神采暗淡下来，最后默然离开房间。
<br/>　　对此，程大雷也无可奈何。
<br/>　　身为人子，李婉儿自然不希望自己父皇出事。可推己及人，身为人子的林少羽，也并不希望自己的父亲横尸街头，遗臭万年。这本就是一笔算不清楚的糊涂账，每个人都只能按照自己的出发点做事。
<br/>　　宋伯康曾指责李行哉：只有自己的悲喜，却不顾及其他人生死。
<br/>　　这话他没有当着程大雷说，如果他问程大雷，程大雷毫无犹豫的反问他：谁又不是只顾自己悲喜，谁又在意其他人生死？
<br/>　　人，本就是自私的。
<br/>　　程大雷不会出兵长安城，但他一直没有放弃收集来自长安城的消息。因为两地距离遥远，消息传到琴川关，平均有七天的时间差。
<br/>　　程大雷无时无刻不在思考，长安一战的后续变化，心中想的，还是如何保住琴川关的基业。
<br/>　　随着战争开启，又有源源不断的流民涌入琴川关，琴川关的人口数量突飞猛进，直到这一日，程大雷正在书房浏览着手下送来的情报，随着嘟地一声，程大雷忽然收到一条系统提示。
<br/>　　嘟，恭喜升级完成，四级山寨升级为初级城池，请为您的城池命名。
<br/>　　程大雷怔了怔，他一直没放弃升级山寨的工作，随着时间渐渐过去，大部分建筑已经完成，唯一缺的恰恰是人口。
<br/>　　及到今日，山寨终于凑够了升级所需的十万人口。
<br/>　　『请为您的城池命名？』
<br/>　　程大雷想了想，回答道：“蛤蟆城。”
<br/>　　城池：初级蛤蟆寨
<br/>　　城主：程大雷
<br/>　　人口：100000
<br/>　　城池建筑：城主府、点将台、兵营、工匠坊、仓库、市场、军策府、养马场。矿场……
<br/>　　城池技能：无
<br/>　　咦！
<br/>　　程大雷睁大眼睛，我的山寨技能呢，我的固防、齐心、马术精通等等技能的。你给我一个『无』算是几个意思。
<br/>　　程大雷查看了一遍，发现不仅琴川关的技能没了，包括落玉寨、蛤蟆寨的技能也变成了『无』。
<br/>　　程大雷气得跳脚，道：“你给我出来，出来……”
<br/>　　『正义的少年喔……』
<br/>　　“我正你妈个头。”程大雷气得大骂：“我的山寨技能哪里去了，你给我从实招来。”
<br/>　　『请注意你的措辞，侮辱系统之灵，是会受到惩罚的』
<br/>　　程大雷余怒未消，但想想这系统大大也并不好惹，还是稍安勿躁，把事情琢磨明白再说。
<br/>　　“那山寨技能，都是我辛苦收集来的，你这样给我收走，可是不应该吧？”程大雷的语气明显变得缓和起来。
<br/>　　『山寨已升级为城池，发展模式是不相同的，原先的山寨技能已不适用城池』
<br/>　　“有什么不同？”
<br/>　　『山寨以劫掠为生』
<br/>　　“那么城池呢？”
<br/>　　『以攻占他人城池为发展目标』
<br/>　　“这还不是一样。”程大雷心情颇为郁闷：“那落玉寨和蛤蟆寨的技能呢，你给我取消也是不应该吧。”
<br/>　　『蛤蟆寨、落玉寨、琴川关共同组成初级城池蛤蟆城，他们的技能自然也会被收走』
<br/>　　程大雷无语，不过也不得不承认，三家山寨加起来才凑够十万人口。
<br/>　　『不过你不必担心，你升级到初级城池，自然会奖励你专属城池的技能』
<br/>　　“奖励在哪里，我怎么没有看到？”
<br/>　　『还没来得及发放奖励，你就把我叫出来了』
<br/>　　“呃……”程大雷被噎得够呛：“那有什么奖励？”
<br/>　　『你可以在原由的技能中挑选三个，继承到新的城池中』
<br/>　　程大雷思考了片刻，立刻有了答案。诸神的祝福是一定要的，这项技能可以每三天开启一项特殊BUFF，江湖传言也是要的，程大雷可以迅速收到长安城的情报，就是江湖传言发挥了作用。还有最后一项，程大雷选择建筑精通。如今落玉寨、琴川关、蛤蟆寨三位一体，建筑精通可以在任何一地发挥作用。
<br/>　　程大雷迅速做出选择，新的城池继承了这三项技能。
<br/>　　『随机奖励城池技能1，获得城池技能：兵贵神速』
<br/>　　兵贵神速：行军速度提升30%。
<br/>　　收到这条系统提示，程大雷的心情才好受些，总算是有些收获的。这项技能可以提升三成的行军速度，在瞬息万变的战场上，这甚至已可以决定一场战场的胜败。
<br/>　　“还有什么奖励，一块发下来？”程大雷。
<br/>　　嘟，你可任意提升绝世以下，任意一人的等级。
<br/>　　嘟，奖励战旗十面，附带技能：鼓舞。
<br/>　　嘟，奖励阵法1：梅花阵。
<br/>　　咦，最后一条系统提示令程大雷发出一声轻呼。阵法？系统开启这方面功能了么。
<br/>　　阵法在古代战场中传得神乎其神，最有名的一场战役，是诸葛丞相摆出八卦阵困住十万敌军。
<br/>　　随之，程大雷脑海中出现关于梅花阵的介绍。梅花阵是十人一组，战斗时互相配合，有攻有守，同时十人队又可组成百人队的大梅花，百人又可组成千人队。现在的程大雷，已经不是当初那个愣头青，对于战斗他并非一无所知。
<br/>　　只是粗略扫了一眼，便可感受到这梅花阵的威力。
<br/>　　得到这梅花阵，算是稍稍安抚了程大雷的心情。至少将山寨升为城池，并不是毫无作用，更重要的是，有一个便有第二个，未来出现八卦阵也不是不可能。
<br/>
