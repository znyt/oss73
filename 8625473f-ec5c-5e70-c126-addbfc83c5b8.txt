<br/>　　另外四个学宫，自不必多说。
<br/>　　他与这四个学宫的交集本来就极少。
<br/>　　而天藏学宫这边，他同样没认出来。
<br/>　　但在他的感知中，这些带队长老，起码都是绝世9994道修为以上的强者。

<br/>　　至于弟子，同样如此。

<br/>　　而另外一边，则是十八府天骄、北隐宗武者，还有近百一流势力。
<br/>　　同样的，都有着自家长老带队，且各两个。
<br/>　　十八府和北隐宗这边的长老，亦皆是绝世之境以上强者，合共38人。

<br/>　　事实上，以他实力，动辄遇到武道大能以上，遇到绝世之境强者罢了。
<br/>　　而放眼整个庞大无比的中域，武道大能，其实已是相当强的武者。

<br/>　　一个武道大能，真正放到一流势力中，已然是最顶尖的战力之一了。
<br/>　　在场这些一流势力，带队的圣皇境巅峰强者，应该是势力之内有数的最强长老。
<br/>　　而带队的武道大能层次，则应该是这些势力的宗主和老宗主之流。
<br/>　　萧逸自然也认不全这些人，但少数几个确实声名斐然的武道大能，他还是能认出的，皆是一流势力中的宗主人物。
<br/>　　在场的武者，大致就是这些。
<br/>　　而着重要说的，便是数量。
<br/>　　对，数量。
<br/>　　五大学宫那边，每一支队伍的人数，都在二十以上。
<br/>　　除却两位带队长老，真正入古帝之墓的天骄，绝对达到二十人以上。
<br/>　　而各大势力这边。
<br/>　　除了十八府和北隐宗的人数比较多外，其余的近百一流势力，不过是寥寥几人。
<br/>　　五大学宫那边，自不必多说，五支队伍自成一体。
<br/>　　而各大势力这边，则明显可以看到，各势力间泾渭分明地聚集在一起。
<br/>　　有的，更靠近天枪府一边；有的，更靠近叶圣府一边…
<br/>　　明显可以看到，这近百势力，都平均地分布为四个整体。
<br/>　　而这四个整体的中心，则分别是天枪府、叶圣府、箭星府，还有东离剑宫。

<br/>　　自然的，对于那些没有名额的各大势力来说，需要向这四大势力靠拢。
<br/>　　他们的天骄，即便进不了主局，但能进入古帝之墓，本身就是一份机缘，且或多或少能有收获。
<br/>　　若是运气好的话，这番收获，不一定小。

<br/>　　但他们能跟着叶流、冉琦、凌鸿进入。
<br/>　　当然，这肯定需要代价。
<br/>　　是否愿意带领这些天骄入古帝之墓，取决于这些拥有名额者。

<br/>　　其它一流势力也一样，或是给予了天枪府、箭星府、叶圣府一定的好处，又或是对东离剑宫付出了一定的代价。
