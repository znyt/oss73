　　听闻慕容雷利之言，公孙嘉微微笑了笑，道：“慕容兄不用介怀，此事已经过去了！”
<br/>　　对于苏莫，公孙嘉并无多少怀恨之心，所以对慕容雷利之言，倒是没有附和。
<br/>　　“公孙兄，我会给你一个交代的！”慕容雷利郑重的说道，虽然公孙嘉貌似没记恨在心，但是，他必然要对付那个秦陨。
<br/>　　不然，难消心头之恨。
<br/>　　“那多谢了！”公孙嘉点了点头，既然对方好意，他也没有言辞拒绝。
<br/>　　随后，两人便闲聊了起来。
<br/>　　时间过的飞快，众人在等待之中，万道剑冢汇聚的天才越来越多。
<br/>　　不过，这些天才的修为，最高之人，也就四品低级，并未修为太过强大之人。
<br/>　　虽说万道剑冢，千岁之内的武者，都可以进入。
<br/>　　但是，一般的千岁之内的武者，也就三、四品的修为。
<br/>　　当然，有些顶级妖孽，甚至是万古天骄，千岁之内可以达到七八品乃至九品的地步。
<br/>　　但是，这些人基本上都接近千岁，以前早已来过万道剑冢，或者说万道剑冢对这些人已经没有用处了。
<br/>　　再者，来此的都是剑修，其它武修一般不会来此。
<br/>　　所以，来万道剑冢之人，低则二品，高则四品，很少有更强之人来此。
<br/>　　时间不断的流逝，又过了几日，万道剑冢聚集的天才，达到了近三千人之众。
<br/>　　其中，万古天骄不下二十人，个个都是耀眼无比。
<br/>　　而此时此刻，距离万道剑冢开启，已经近在眼前了。
<br/>　　苏莫盘坐在深坑边缘，暗暗的松了口气，他之前还担心，有神王级别的强者到来，看来担心是多余的了。
<br/>　　别说神王级别的强者，就算是五品修为之人，都没有一个。
<br/>　　“如此最好！”苏莫心中暗道，没有高品级的强者，他的安全性将大大的提高。
<br/>　　咻！咻！
<br/>　　就在此刻，两道犀利的破空声响彻天际，东西两个方向，各有一道身影，不约而同的急速而来。
<br/>　　东方的身影是一名女子，身穿橘黄色长裙，一头乌黑的长发随风舞动。
<br/>　　这名女子身材高挑，肌肤百能，容颜秀美，气质极为出众，仿佛是九天仙子。
<br/>　　西方的身影，是一名黑衣男子，身材高瘦，气息极为强大。
<br/>　　但诡异的是，此人的真实面容，让人无法看清。
<br/>　　仿佛此人的面容在不断的改变，每一个眨眼，都是不同的相貌。
<br/>　　这种感觉极为诡异，好像此人有千千万万种形态，根本没有具体的模样。
<br/>　　“天罚云沐、千幻邪君！”
<br/>　　众人看到这急速而来的两人，顿时心中微震，纷纷瞩目。
<br/>　　整个神域，虽然万古天骄有千位之多，但是，万古天骄之间，天赋也有强弱之分。
<br/>　　而这到来的两人，绝对是万古天骄之中，顶尖的存在。
<br/>　　那位女子，名为云沐，不仅拥有太古武魂榜排名第十七的天罚神瞳元神，还拥有太古异体榜排名第八的天衍之体。
<br/>　　而那位男子，同样不凡，名为林千幻，人称千幻邪君。
<br/>　　拥有太古武魂榜排名第十二的千幻元神，也拥有太古异体榜排名第十三的虚无玄体。
<br/>　　两人的这两种元神、两种血脉体质，无一不是顶级的存在。
<br/>　　可以说，一般的武者，拥有任何一种，便可成为顶级妖孽。
<br/>　　而这两人，每人都拥有一种超强元神和超强体质，自然便是万古天骄之中的佼佼者。
<br/>　　嗖！嗖！
<br/>　　呼吸之间，云沐和千幻邪君两人，几乎是同时来到了万道剑冢的上空。
<br/>　　两人对视一眼，个个面色孤傲，眸光有些争锋相对。
<br/>　　“云大小姐真是和在下心有灵犀啊，居然同时来到这里！”千幻邪君嘴角勾起一丝玩味的笑容，揶揄的说道。
<br/>　　“林千幻，你不要找不自在！”云沐面色冷然，眸中没有丝毫的表情。
<br/>　　千幻邪君闻言，撇了撇嘴，没有再多言，他的天赋虽然不比对方差，但是，由于对方年长二十岁，修为已达四品低级，比他高一些。
<br/>　　众人见此，暗暗惊讶，这千幻邪君可是个狠人，杀戮极重，但是，现如今面对云沐，都不禁有些认怂了。
<br/>　　嗖！
<br/>　　就在此刻，下方人群之中，有一名青年飞天而起，急速的远去。
<br/>　　此人，仿佛是想要逃离万道剑冢所在之地，速度着实不慢。
<br/>　　“甘通，你逃得了吗？”
<br/>　　云沐蓦然转头，看向那逃离之人，她的身上突然升起了一股神圣的光芒，面色也变得庄重与肃穆。
<br/>　　这一刻，云沐好似主宰一切的神灵，掌控一切，俯瞰天地。
<br/>　　“我云沐，代表天道，赐予你死亡！”
<br/>　　一道淡漠而萧杀的话语，从云沐的口中响起，柔美而冷厉。
<br/>　　其话音一落，目光爆射长空，形成了一道雷霆，向逃走的青年袭杀而去。
<br/>　　这雷霆极为古怪，根本不是正常的雷电，而是一种呈透明之状的虚空闪电。
<br/>　　虚空闪电的速度，快到极致，威力无与伦比。
<br/>　　那名逃走的青年，修为高达三品中级，但是，在这道虚空闪电之下，几乎是没有任何反抗之力。
<br/>　　啊！
<br/>　　一声凄厉的惨叫声响起，虚空闪电重重的劈在了青年的身上，霎时之间，一切都彻底毁灭了。
<br/>　　青年的元神和肉身，直接被摧毁、湮灭，荡然无存了。
<br/>　　嘶！
<br/>　　所有看到这一幕的人，无不心惊，很多人甚至倒吸了一口凉气。
<br/>　　天罚云沐，拥有天罚神瞳元神，掌控天罚之力，果然非同小可。
<br/>　　只是，这云沐居然来此就杀人，这是什么意思？震慑他们吗？
<br/>　　“云沐，你这是什么意思？”终极神王殿的韩生，阴沉着脸，直接开口问道。
<br/>　　“此人曾残杀我云家两位族人！”云沐冷淡的说道。
<br/>　　对于这种云家的仇人，她绝不留情。
<br/>　　众人闻言恍然，原来是仇人，难怪如此。
<br/>　　少倾，云沐身形缓缓落下，没有理会任何人，在深坑边缘盘膝而坐。
<br/>　　“实力确实厉害！”千幻邪君轻笑一声，随即也缓缓的落了下来。
<br/>　　众人大量云沐和千幻邪君，心中暗暗警惕，这两人都是危险人物。
<br/>　　千幻邪君自然不必说了，能被称为邪君，便可知其为人。
<br/>　　而云沐虽然长的美丽，但是，高傲不可一世，谁若是得罪了他，那也离死不远了。
<br/>　　“天罚之力！”苏莫瞥了一眼云沐，心中低声自喃，这种力量果然玄妙，而且不是一般的强大。
<br/>　　他也暗暗警惕了起来，这两人一个是四品低级的修为，一个是三品高级的修为，都是有威胁的人物。
<br/>　　少倾，苏莫也不再关注这两人，继续一边凝练修为，一边等待了起来。
<br/>　　他的修为，距离突破，已经近在咫尺了。
<br/>　　此刻，距离万道剑冢开启，也已经不远了。
<br/>　　终于，又等了一日之后，本来平静的万道剑冢之内，浓密的雾气突然剧烈的翻滚了起来。
<br/>　　“万道剑冢要开启了！”
<br/>　　“很快就能进入了！”
<br/>　　众人纷纷来到了万道剑冢的边缘，等待其内的雾气消退。
<br/>　　这雾气，是一种恐怖阵法形成之物，能腐蚀一切，不可碰触。
<br/>　　“哈哈哈，天池的朋友们，不知道欢不欢迎我拓跋流明！”
<br/>　　就在此刻，一道洪亮的大笑声突然响起，震动四野。
<br/>　　众人循声望去，便见到天际的尽头，一道壮硕高大的身影，虚空踏步而来，强大的气势铺天盖地。
