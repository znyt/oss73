        连星儿倒是不痛不痒，十分淡定地回答道：“他是我师父！我喜欢、我高兴！你要是不待见，你跟我解除婚约就好了啊！”
<br/>
<br/>        “你‘养’个洛羽不够？还要‘养’一个网络世界里认识的男人。以后，是不是，只要是你喜欢的异性，你都打算去‘养’？嗯？”明司寒有些气急败坏地质问。
<br/>
<br/>        连星儿却一脸认真地点了点头，反问：“有何不可？”
<br/>
<br/>        她喜欢洛羽，是洛羽的粉丝，有能力帮他拉到一些好资源就帮他拉，有何不可？
<br/>
<br/>        她喜欢师父，崇拜她师父，有资金帮师父组建团队，又有何不可？
<br/>
<br/>        连星儿不觉得自己做错了什么。
<br/>
<br/>        而此刻，明司寒终于明白了一件事情。
<br/>
<br/>        那就是，他的价值观，跟她连星儿的价值观，完全不在同一个频道里。
<br/>
<br/>        明司寒忽然间沉默了，没有接下文。
<br/>
<br/>        连星儿也不再跟他搭话，两人在一片安静中度过，气氛严峻到就连坐在驾驶座上开车的司机，开车时想打喷嚏都不敢打出声来。
<br/>
<br/>        北城，东片区，明家四合院。
<br/>
<br/>        下车后，明司寒领着连星儿走上台阶，跨过门槛，进了一处大气的四合院。
<br/>
<br/>        这里虽然没有临海城傅家大庄园那么大，但也拥有上千平方米的占地面积。
<br/>
<br/>        刚进大宅门，连星儿走到一半，就不肯走了。
<br/>
<br/>        明司寒转过身去，走到她跟前，二话不说地拉起她的手，牵着她继续往前走。
<br/>
<br/>        两人拉拉扯扯、别别扭扭地进了大宅门内的垂花门，路经庭院，往正房大厅那边走。
<br/>
<br/>        途中遇到明家的佣人，几个佣人都会礼貌性地对明司寒颔首问候：“小少爷，好。”
<br/>
<br/>        连星儿想甩开明司寒的手，却被她越牵越紧。
<br/>
<br/>        她的手很软，就像握着软软的棉花团一样，很舒服。
<br/>
<br/>        明司寒这一牵，就不想放手了。
<br/>
<br/>        正房大厅里，明护国坐在高堂上喝茶，看到自己这小孙儿牵着小孙媳妇进了门，感情很要好的样子，顿时笑不拢嘴。
<br/>
<br/>        “爷爷，我把星儿带来了。”明司寒将连星儿拉到了爷爷明护国的跟前欠身颔首。
<br/>
<br/>        连星儿再怎么不喜欢，但出于礼貌以及对长辈的尊敬，她还是微微颔首，唤了声：“明爷爷。”
<br/>
<br/>        “叫我明爷爷多见外啊！直接叫爷爷就好。”明护国笑容满面地说道。
<br/>
<br/>        连星儿睐了身旁的明司寒一眼，心不甘情不愿地改了口：“爷爷。”
<br/>
<br/>        “诶！乖！”明护国笑着点了点头。
<br/>
<br/>        “爷爷，明司寒还说，这次带我来，是要当面跟您提，和我解除婚约的事情。”连星儿挺直了腰板，不顾明司寒的脸色难看，接着说道，“他跟我说，他与贺家的千金贺思思两情相悦。”
<br/>
<br/>        一听这话，明护国顿时一脸严肃地瞪着明司寒：“小寒，爷爷警告你。你这辈子，娶的女孩，只能是连星儿。其他女孩，想都别想！”
<br/>
<br/>        明司寒连连点头，唯命是从地附和：“是是是，孙儿知道了。这辈子，只娶连星儿为妻。”
<br/>
<br/>        “？！”下一秒，连星儿一脸懵。
<br/>
<br/>        回过神来后，连星儿连忙跟明护国解释道：“不对！爷爷，明司寒和贺家的千金贺思思才是两情相悦，您不能棒打鸳鸯啊！”
<br/>
<br/>        “星儿，你放心好了。爷爷会替你做主的！既然你选了小寒，我就会让小寒对你负责到底。他若是敢做背叛你的事情，爷爷替你打断他的腿！”明护国气势汹汹地说。
<br/>
<br/>        连星儿眨巴着眼睛，有些欲哭无泪了。
<br/>
<br/>        事情不是这样子发展的吧！
<br/>
<br/>        怎么反倒是讲不清了！
<br/>
<br/>        “爷爷，我觉得我可以成全他们俩的！真的，没问题的！”连星儿忙着开腔，想把话题的重点重新体现出来。
<br/>
<br/>        明护国似乎是在装糊涂，一直在跟明司寒强调：“小寒，你看看，人家星儿多好多体贴。哪像思思那丫头那么闹腾。你要好好珍惜人家星儿，知道了吗？”
<br/>
<br/>        “是，爷爷。”明司寒暗自窃喜地颔首，一副彬彬有礼的样子。
<br/>
<br/>        顿时，连星儿感觉自己是跳进黄河也洗不清了。
<br/>
<br/>        “星儿，你别吃醋了。爷爷明白了。你放心，爷爷替你担保。小寒这小子，绝对不会辜负你！”明护国一本正经地说道。
<br/>
<br/>        连星儿此刻是哑口无言，不停地向明司寒使眼色。
<br/>
<br/>        明司寒却视而不见。
<br/>
<br/>        给明护国问完安后，明护国给了连星儿见面礼金，但是连星儿没接，明司寒倒是毫不客气地替她接了。
<br/>
<br/>        连星儿意识到自己被明司寒给“坑”了后，心里极其不爽。
