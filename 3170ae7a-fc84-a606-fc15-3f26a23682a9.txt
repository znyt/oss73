        &lt;h3&gt;第四卷第十五章  陈伟冯入坑&lt;/h3&gt;
<br/>
<br/>        呼延临庆带着夏颜的吩咐去找了陈伟冯。
<br/>
<br/>        陈伟冯有些吃惊呼延临庆来找他，呼延临庆平时跟着夏颜的身后，把他们这些副职都不当回事，今天竟然来找他，太阳是从西边出来了，这是不是证明夏颜已经准备提拔他了？
<br/>
<br/>        “呼总，今天怎么有时间过来？你不是在洛城吗？项目做得怎么样了？”陈伟冯问呼延临庆。
<br/>
<br/>        呼延临庆看起来有些憔悴。
<br/>
<br/>        “项目在进行着，目前有点难关过不去，他们开发软件对接的时候，总是有点小卡壳。”呼延临庆叹了口气。
<br/>
<br/>        “遇到难题了？”
<br/>
<br/>        “唉，当初做方案的人不在，他写的方案和现在做的总感觉有些出入，怕交不了差。”
<br/>
<br/>        “这个项目拖了很久了啊，我记得当初我也参加评标了。”陈伟冯给他叔叔陈子霖打了个低分，将陈子霖的方案一棍子打死了。
<br/>
<br/>        “我就是为这事情来的，当初要不是你啊，我还得不到这个项目，说实话，虽然时间有点长，还是很赚钱的，为了感谢你，我特意来看看你。”呼延临庆胖胖的脸上堆着笑。
<br/>
<br/>        “不，不，不，你是夏总的朋友，我们得为夏总着想，你明白我的意思吧？”
<br/>
<br/>        “明白、明白，夏总当你是他亲弟弟，我也当你是亲弟弟。这不，她调走了，我这里以后还得你多照应着点，哥哥这里有点心意，你一定要收下，阿姨当初走的时候，我被洛城的事给拖住了，也没有去探望一下，实在是心里愧疚，这一脱开身我就来了。”
<br/>
<br/>        “心意到了就好，我现在什么都不缺。”陈伟冯推搡着呼延临庆送给他的包裹。
<br/>
<br/>        “知道你现在不缺，不过后期你再进一步的时候，上上下下不得打点啊，我们谁跟谁啊，再说，你现在在这个位置上，也算是颜颜的人了，总该为大家着想吧，总不能老是一个人吃饱全家不饿吧？”
<br/>
<br/>        “这个我真的不能拿。”陈伟冯看着眼前的包裹，有些迟疑。
<br/>
<br/>        “兄弟啊，我现在是为你着想，你看看你现在是一穷二白的，什么都没有，你还想不想上去啊，目前位置空缺着，你总得想想办法吧？话又说回来，我和你姑姑是同学，我能害你啊？当初做项目的时候，你要是多给你叔叔打两分，就轮不到我来做这个项目了。你说你都将你叔叔得罪了，你总得为自己想一想吧，上去了，你脸上也有光彩啊。”
<br/>
<br/>        “这个.......”
<br/>
<br/>        “别这个那个的了，我是为你着想，你还年轻，总得要一番作为，就算你夏姐姐要帮你，你也要有拿得出手的东西啊，就凭你那点死工资，都不够你花，原来还有阿姨帮着你，现在阿姨走了，你不得靠你自己啊，你还得养孩子不是？我和你夏姐姐这么多年了，你还不放心我啊？”
<br/>
<br/>        “我........”陈伟冯动心了。
<br/>
<br/>        “行了，我走了，后期听说还有智慧城市的项目，到时候记得你哥哥我啊。”呼延临庆放心下包裹就走了。
<br/>
<br/>        “颜颜，我按照你的吩咐都做好了，他确实推脱了一阵。”
<br/>
<br/>        “做好了就行，呼延，洛城的项目你得抓紧，不敢再拖下去了，另外这一段时间你暂时不要和我来往了，等这一段风声过后我们再联系，当然，你是我最好的朋友，我不可能不管你，你得做好最坏的打算，将你现在公司里的资金转到研发的那个公司上去吧。”
<br/>
<br/>        “颜颜，我这心里慌得很。”
<br/>
<br/>        “慌什么慌，再怎么着你还有我呢，只要我还在这里，你就是安全的。”
<br/>
<br/>        “有你这句话我就放心了，你放心，有什么事情我都会顶着的，不能让你有一丝丝伤害。”
<br/>
<br/>        “行了，我有啥事情啊，你照顾好你自己别给我添乱就行了，我这里还有事情，回头联系。”夏颜挂了呼延临庆的电话，长长的舒了一口气。
<br/>
<br/>        洛城的项目就算有事情也动不了她的根基，陈伟冯还在前面挡着呢。
<br/>
<br/>        刘清影找机会见到了吕严华和胡安娜。
<br/>
<br/>        如出一辙的说了一番话：“项目的进度是有些慢，但是我们都是在努力着，你们当初招标的时候也是经过了正规的程序，要有责任也是我们工作进度太慢，我们也是想要给你们一个满意的答案，实在是拖累你们了，你们无非就是失察，别的方面我们从来就没有过交集，你们的亲戚朋友也很多，借点钱买辆车啊房子啊什么的也是可以的。你放心，我们不会拖累你们的。”
<br/>
<br/>        吕严华和胡安娜心明如镜。
<br/>
<br/>        在马总的干涉下，洛城的项目继续在缓慢的进展着，吕严华和胡安娜因为工作失误，导致项目缩水，耽误工期，如刘影所说：失察！双双从现在的位置上给调离了。
<br/>
<br/>        最憋屈的就是吕严华了，在这个位置上连两年都没有干到，完全就是背了一口黑锅。
<br/>
<br/>        不过三十年河东三十年河西，只要有靠山到哪里都是干。
<br/>
<br/>        洛城的项目还是在攻克难关中，他们在商议要不要绕开原来的方案重新引用新的系统，但是最后被新来的老总给否了。
<br/>
<br/>        项目进入了胶着状态，夏颜平稳的过渡到了现在的位置上，无风无浪！
<br/>
<br/>        呼延实在的没辙了，这些项目耗费了他很大的精力，他只得又打电话求助夏颜。
<br/>
<br/>        “我现在不在其位不谋其职，你不是和陈伟冯关系好吗？让他帮你解决不就行了？你现在脑子怎么就转不开了呢？”夏颜提点呼延临庆。
<br/>
<br/>        陈伟冯并没有如愿做到他想要的位置上。
<br/>
<br/>        夏颜走后，那个位置空了一段时间，宋长生对夏颜说：“你也知道，陈伟冯和洛城的项目有瓜葛，如果你再将他提上去，对你不利，不如放一段时间，等洛城的项目完工了，再给他安排一份好的位置。”
<br/>
<br/>        夏颜原本就是想用陈伟冯来搪差事的，洛城安稳落地，夏颜现在风生水起，陈伟冯也就失去了作用，况且她还一直对陈伟冯心生警惕，自然就没有提拔陈伟冯，有宋长生在耳边吹吹风，夏颜将孙毅提上去了。
<br/>
<br/>        陈伟冯虽然没有得到想要的，呼延临庆也算是弥补他了，呼延临庆隔三差五的就会来找他，送些东西喝喝酒，顺便安慰安慰陈伟冯，一来二去，俩个人成了朋友了。
<br/>
<br/>        孙毅被提上去了，并没有为难陈伟冯，只要是夏颜打招呼的项目，孙毅都交给了陈伟冯负责，孙毅从来都不插手，陈伟冯实际上成了夏颜的替代品。
