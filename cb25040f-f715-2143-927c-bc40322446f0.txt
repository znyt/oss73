　　　　昂...
<br/>　　江行泽不断的喝着蛇血，他发出了一种愉悦的呻吟声，全身的毛孔居然丝丝缕缕的冒出了烟雾。
<br/>　　他自己早就见怪不怪，这是他体内的蜂毒被吸血蛇的血给压制住了，暂时安分起来了。
<br/>　　浑身的痛楚也烟消云散。
<br/>　　半响，江行泽的眼神变的怨毒之极，脑海中出现了一条蛇的模样，恨的深入骨髓，不是那条银白蛇，自己也不会变成如此不成不鬼的样子。
<br/>　　每天只能跟这条吸血蛇在一起，没有它的蛇血，自己必死无疑。
<br/>　　他甚至觉得自己已经不是一个人类了，而是蛇人，体内流淌着蛇血，性情也变得越发凶残暴戾。
<br/>　　最关键他每日喝的还是吸血蛇的血，这种蛇，出了名的凶煞，喝它的血，将一个人变的妖邪起来，也不足为奇。
<br/>　　另外，不得不说，成为绝世蛇种的吸血蛇，它的血是多么的天材地宝，能将一个必死的人，硬生生的留在人世间。
<br/>　　蜂毒被压制，江行泽也不在选择继续喝血，他也喝饱了。
<br/>　　转身看向吸血蛇，它早就将门卫彻底的吞入了腹中，可堂堂一个人成年人被吞进去，它的腹部居然能不见一丝变化。
<br/>　　再看整条蛇的体型更是足够吓人，不过半个多月时间，此蛇已经暴涨到了二十米，它就静静的盘卷的，都能将人吓得屁滚尿流。
<br/>　　江行泽面目可憎的脸色一抹邪笑，朝着空中伸出一只手，吸血蛇的蛇头骤然摇曳而来。
<br/>　　“真乖，人肉吃的美味吧”
<br/>　　他温暖的摸着吸血蛇的巨大头颅，眼中发光，开口道：“认我为主吧，我保证每天都能让你吃到人肉，已你现在的阶段，也唯有吃人，才能让你变的更强”
<br/>　　“我师父只拿你当药，你就是他的药引，实验品，而我就不同，我身上留着你的血，我们现在可是血脉相连”
<br/>　　他徐徐诱之，继续道：“你只有跟着我，你就海阔天空了，以后也不用关在这牢笼中”
<br/>　　吸血蛇妖治的蛇眸冒着幽光，发出可怕的声音：“我占时还摆脱不了他对我的控制”
<br/>　　江行泽摇了摇头，笑了起来：“我师父不过是用蛇魂来控制你，只要以后你不吃他给的蛇魂，你就不用听他命令”
<br/>　　吸血蛇妖治的蛇眸散发异样的光，看着江行泽。
<br/>　　“吃下它，认我为主”
<br/>　　江行泽的手中出现了一颗殷红的小药丸，递到了吸血蛇的巨大血口下。
<br/>　　要是蛇博士在的话，一定会一眼认出，这江行泽手中的药丸正是他给的蛇魂，只是这蛇魂原本是漆黑如墨的小黑丸，可在他的手中却变了色。
<br/>　　江行泽耍了一点小心机，他用自己的血浸泡蛇魂，让里面的药性蕴含自己的印记。
<br/>　　原本现在他每日都喝着吸血蛇的血，体内既是人血，也是蛇血，如今，在拿这颗用血浸泡的蛇魂给吸血蛇吃下。
<br/>　　那么吸血蛇肯定会彻底的臣服与自己，认自己为主，终生不会背叛。
<br/>　　“你们人类最卑鄙了，一个个就想奴役我，你们都该死”
<br/>　　吸血蛇眼眸冒出了凶光，火红鳞片如同有火焰在跳动，这座牢笼的温度骤然上升。
<br/>　　没错，是温度上升，都知道蛇类的身上是阴冷的，但吸血蛇一生气，温度居然上升了，违背常理。
<br/>　　要知道传闻中的变异吸血蛇何等的凶煞，才刚出世就成为了整个白镇捕蛇人的噩梦，但现在它实力暴涨，血脉提升，反而却沦落到被人类控制跟奴役。
<br/>　　它岂能甘心，但被蛇魂药性控制，只能委曲求全。
<br/>　　“你只有认我为主，你才能肆无忌惮，你才能自由，不用被人当成药用”
<br/>　　“来，吃下药丸，认我为主，我带你出去”
<br/>　　江行泽本不想这样做，但他的师父要将他送出国，显然是准备抛弃他。
<br/>　　他不仁就别怪自己不义。
<br/>　　唯有控制吸血蛇，他不但不用死，还能得到一头强大的宠物，到时候还可以找那条该死的蛇报仇。
<br/>　　所以他只能这样做，也必须这样做。
<br/>　　吸血蛇的蛇眸凶光越来越盛，它恨不得一口吞了江行泽，他比他师父更可恶。
<br/>　　但这等觉醒生物，灵智不会逊色于人，非常的聪明。
<br/>　　懂得利与弊，如果吸血蛇将江行泽给杀了，那么以后它就只能被永远关押在这座牢笼中了。
<br/>　　“吃吧，你只有跟着我，你才有好日子过”
<br/>　　江行泽压制住自己的激动，将殷红的药丸朝着吸血蛇的血盆大口中放，同时他也在观察这条蛇的反应。
<br/>　　吸血蛇只是冒着愈演愈烈的凶光，静静诡谲的看着江行泽将手中的药丸放到它的血口中，也没反抗什么。
<br/>　　成功将药丸给吸血蛇吃下之后，江行泽整个人都振奋起来，控制了这条蛇，他就宛如拥有了整个世界，兴奋与激动如同决了堤的洪水，浩浩荡荡，哗哗啦啦地从他的心理倾泻了出来。
<br/>　　“叫主人”
<br/>　　江行泽旋即冲着吸血蛇大吼了一声，心像煮开的沸腾一样，激动的要溢出来。
<br/>　　吸血蛇的蛇眸赤红一片，充满嗜血，没有反应，只是死死盯着江行泽，猛然，它二十米的蛇躯扶摇直上，像是要发狂，场面煞是吓人。
<br/>　　噗通...
<br/>　　江行泽被吓得一屁股坐在了地上，怔怔而惶恐不安的看着直立成擎天柱般凶煞摄厉的吸血蛇，莫非是失败了？
<br/>　　不可能，他本来跟吸血蛇就等于血脉相连了，无比的亲近，在加上师父给他的蛇魂，怎么可能还控制不了它。
<br/>　　昂...
<br/>　　一声惊天动地的咆哮声冲天而起，汹涌呼啸，整个牢笼都无法无天的震动起来，仿佛要崩塌，世界末日降临。
<br/>　　真的失败了？还是自己激怒了这条凶煞的蛇。
<br/>　　它想干嘛，想杀了自己？还是想吃了自己？
<br/>　　江行泽害怕了，惶恐了，他毕竟不是他师父蛇博士，对蛇有着几乎天生的统治力。
<br/>　　任何毒蛇在他师父的手上都乖巧如宠物，哪怕是凶煞暴戾著称的吸血蛇也不例外。
<br/>　　如果真的要说有一次例外，那么就是那条将他害的人不人鬼不鬼的银白蛇了。
<br/>　　它当时在实验室，明明吃了蛇魂，也朝着蛇博士臣服了，但最后却大闹一场的逃走了。
<br/>　　这是蛇博士一生之中唯一的失败，也正是他唯一的失败，造成江行泽变成如此样子。
<br/>　　所以，暗地之中，江行泽甚至有点恨他的师父，他为什么会没能控制住那条该死的蛇。
<br/>　　要是控制住了，自己会变成如今不成不鬼的样子吗。
